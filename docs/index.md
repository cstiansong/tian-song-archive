# 松

田松，男，土家族，1997年11月生，贵州铜仁人。
毕业于西安交通大学（副部级），获文学学士学位。
现任读书小天地（读书交流群）管委会群主（正群级）。
注：读书小天地、读书交流群为一个机构、两块牌子，合署办公，共用一套领导班子（正群级1名，副群级4名），田松同志主持全面工作。
