# 浪淘沙之纵横交大 by 名字

# 目录

# 第一章 走出大山

2017年的夏季，决定人生命运的高考刚刚过去，填好志愿的我在家里并没来得及歇多久，就收到了西京交大的录取通知书。录取通知书都是往学校里寄的，无非是为了方便老师统计学生都去了哪里，当然这只限于用来做喜报为来年招生打广告的优秀学生。

我所在的钢仁中学就是这样一所闻名于全国的超级中学。钢仁中学旧址位于钢仁市中南门附近，早些年民国时候那是国立贵州中学，凡是带着‘国立’二字，诸位就该明白这钢仁中学的来头有多值得一叙了吧。民国到现在，虽然称不上百年名校，但也足够历史悠久，更值得一说的是这‘国立’二字的名声确实没有给败了。新中国成立之后，原来民国时期的名称都不许再用，国立贵州中学也就改称钢仁中学。至今为止，这里已出了不少状元，录取我的西京交大虽比不上清北华五等校，但在中国也绝对是排得上号的，也就是说我就是能登上喜报的那一批学生。想到这里，我挺直了我本就挺拔的胸膛，昂首阔步地朝校门走去。

钢仁中学西临人民路，大门也开在这条路上。大门整体成暗灰色，显得大气奢华，门上烫金的钢仁中学四个大字，更是将这所名校的气质体现的淋漓尽致，据说就连本省211大学贵州大学的校门都没有钢仁中学气派，不过这也没什么奇怪的，钢仁中学的学生根本瞧不上贵州大学，每年只有排名倒数的学生才会被发配到贵州大学上学。

每年到发放录取通知书的时候，学校门口都会站着这么几波人，美其名曰“志愿者”。这些个伙计全是老师喜欢的类型，分明已毕业两月有余，还是一水的穿着校服搁门口站岗，打听来取通知书的学生都上了哪里。咱们实话讲，这些个学生也不是说学的多好或长得多好，就是会来事能讨好老师，毕业了还指望着能靠过去的威风打压同学，买他们账自然不多，但也还有那么些个。我过去那是属于不学好的了，高二的时候还考班级的倒数第一名，自然入不得老师的法眼，更是被这些人欺负的对象。说真格的，我上学那时也羡慕过他们，可我当时为人老实，长得也不够俊俏，也就得不到老师高看，因此从没有成为‘他们’的机会。但今天不一样，我考上的是西京交大，我有意要煞煞他们的威风。

离校门还有段距离的时候，我就看着他们几个发现我了，那几个人本来正各自散开着打听其他同学的去向，现在却围在了一起朝我逼近了过来。为首的一个是兰洋洋，他后面紧跟着武松、辉渣渣等人。“嚯，这不是陶砂吗？”兰洋洋率先开口了，我也先不回话只是直直朝他们走去，待到快和他们脸碰脸了，方才回问，“什么事？”

“什么事，这个点在校门口见着你还能有什么事？”武松轻蔑地盯着我“不就是想看看你小子离贵州大学差几分吗？”说完他们几个人就哈哈大笑起来，显然是笃定了我高考暴毙，失意而来，甚至还想把对别的高分同学的嫉妒也发泄到我身上。

这次他们算是撞枪口上了，我心中一种莫名的兴奋油然而生，我早就将在校门口被‘志愿者’询问的事情在脑海中反复演练过好几十回了，虽然逮到一般的‘志愿者’炫耀炫耀也够令我高兴的了，但恰好遇到的是这几个。我心想，怎么着，老天爷让我长脸的机会这么快就来了？

我直视着兰洋洋，脑子里飞速盘算着该怎么说才能把这些人的脸最大限度的踩在地上，直说985高校吗，不行，985太多了，他们估计要用中央民族大学来嘲笑我，不过嘲笑就嘲笑吧，你们现在有脸笑嘻嘻，等一会儿知道什么是C9的时候不把你们的脸打的啪啪响。

这时候兰洋洋可能是受不住我自信的凝视，他按捺不住了“你这人神气什么啊，一年的倒数第一不以为耻，还有脸从正门进来拿录取通知书？”没过瘾似的，兰洋直接放出了杀招“知道我上的哪儿吗？南东大学！985、211、建筑老八校。”老八校三字特别拉长了语调，仿佛要提醒我南东大学是与清华、哈工大、同济并列的，兰洋洋想一下子就把我的自信心磨灭成碎渣。

这段话说完，我心里算是有底了，合着你们几个这么神气，也就上个南东大学。本来我还怕我这西京交大不够硬派，现在看我怎么让你们几个好看。见我的表情没有变化，武松一巴掌拍到了我肩上，“说话啊，你小子，上哪里了说出来让我们大家快活快活。”武松说话的声音高亢，有意把校门口众人的注意力都吸引到我们身上，看来他是想让在校门口的都来看我笑话。

我扭头朝四周望了望，果然校门口的人都注意到了这边发生的事情，纷纷聚了过来。得了，我本来还想给你们留点脸面，现在这局面可是你们自找的了，人不要脸，神仙难救啊。今儿个，我要不在校门口把威风立住，贵州我算是混不下去了，还不知道你们几个要在年级群里怎么说我呢？

念及此处，我一手把武松搭在我肩上的胳膊甩开：“哥几个，咱们什么时候这么熟了？我今天来就想拿个录取通知书，没打算跟你们交朋友。“武松听了这话直用眼翻我，还没待他开口，辉渣渣骤然出手，推了我一个趔趄，没等我站稳，就拍着胸口的钢仁中学学生会徽章向我咆哮：”看见这是什么了吗？跟你交朋友？要不是老师派发的任务，你看你配和我们说话吗？怎么了，你不服，就你倒数第一的成绩你凭什么不服啊。“我把身形稳住，心想：好家伙，你们几个这也太横了，‘志愿者‘就这样的水平吗。旁边围拢过来的学生，也似有打抱不平之意。

兰洋洋眼尖，开口说道：“辉渣渣你别急，陶砂这人不配合学生工作也不是一天两天了，要不然怎么他考倒数第一呢。”这话显然是在跟同学辩解，为他们粗鲁的行为找个合理的理由。不过也好，用这种理由正好让我大有可为。我打定了主意，开口便说：“怎么，学习好的就能欺负学习差的吗？”兰洋洋冷笑一声：“怎么能是欺负呢，是你不配合工作。”说着兰洋洋伸出手在我身上乱掏：“手机呢？赶紧拿出来，我们的时间也是宝贵的，没工夫跟你耗。”

这些个人还真是不知好歹，这种局面下，必须得下狠手了。我左手一把推开兰洋洋，右手掏出手机直怼到他脸上，这两个动作一气呵成。接着我就问道；“看清楚了吗？大声给大家念念。”兰洋洋一时没反应上来，其他几个人也完全没想到，我一个‘差生‘居然敢当着这么多人的面冲他们甩脸子。一时间，所有的人都不敢轻举妄动，通通都等着看我葫芦里卖的什么药。

兰洋洋把脸稍微朝后缩了缩，把目光聚焦在屏幕上，边看边一脸不屑的说：“也就上个贵州大学，神气个什……什……什么，西京交大！”瞬间兰洋洋的脸就失去了颜色，旁边的武松，辉渣渣等人也都大惊失色。兰洋洋无意识的抬起了手想把我的手机拿到眼前仔细看看，以防刚才都是眼花，我可不给他这个机会，直接就把手机收了回来。

“你们说的对，学习差的就是不配跟学习好的交朋友。”说着的同时我在微信通讯录中找到了兰洋洋，这是之前班级生活强制要求加的，接着我当着兰洋洋的面把他拉进了黑名单：“滚远吧，哥几个，我该去取录取通知书了。“说罢，我拨开了兰洋洋等人，留下呆若木鸡的他们，如侠客一般走进了钢仁中学的大门。

# 第二章 北上西京

取完录取通知书也就该去上学了，离我在钢仁中学门口收拾兰洋洋一行人已经过去了3天。我这几天在家也算是听够了老爹跟村里人的炫耀，老娘对我远行求学的嘱托，我寻思着时间也该出发了。这天我起了个大早，跟爹妈还有两个妹妹打过招呼，接着就踏上了前往西京交大的路途。

西京市地处我国中心，也是个四通八达之地，从古至今有13个王朝在那里建都，其中不乏周秦汉唐这样的强盛之世，这也是西京得名的由来。然而从我这个小村庄到西京市却不容易，我不得不先坐上前往钢仁市的乡镇大巴，到了钢仁市再做打算。大巴在山上这么七转八转的，直到了中午，才好不容易捱到了钢仁市。钢仁市火车站地处市区东北，而汽车站却在南边，二者离的距离并不算近。害怕前往西京市的车票售罄，我顾不得吃午饭就匆匆乘坐公交前往了钢仁火车站。

钢仁站并不算大，是一幢上下三层通体白色的建筑，但对于从没坐过火车的我来说够震撼的了。售票厅的窗口只开了两个，但由于是中午，厅里买票和候车的人同样也不多。因此也就5分钟，我就买好了票，只是钢仁到西京并没有直达的火车，需要在重庆中转。不过这样也好，中转之后到西京就是第二天早上8点了，刚好可以省去一天的住宿费用。

距发车还有段时间，饿劲也在这时犯了上来，我决定先去找点吃的垫吧垫吧。火车站地处郊区，其实并没有什么选择，我随意走进了路边的一家米粉店。还真别说，在什么山卖什么药这话就是没说错，开在清水大道上的米粉店牛肉粉的汤就是跟清水一样。我正漫无目的地吸溜着寡淡无味的牛肉粉时，一个人在我对面坐下了。我心里纳闷，这小破馆子根本没人吃，那么多空位怎么偏偏坐我这儿呢。“是你吗，陶砂？”，我听着声抬头一看，这才发现是坤哥。

坤哥姓何，名字看起来和清代的巨贪和珅有些相像，大家有时也用和珅这么戏称他。然而和坤哥熟识的人都知道，坤哥为人何止是不贪小便宜，而且十分正直讲义气，在村里的名声那是一顶一的好。在过去，我也是常和坤哥一起玩儿的主。有那么一年秋天，我从家里翻出来了一小瓶火油，这可算是得了宝了。2000年那会儿，麦秸还是用作燃料的，家家户户在秋收之后都要把麦秸搭成麦垛储存下来。平时生火做饭或是冬天取暖就让小孩去麦垛扯上一些，小孩也乐得做这些事，夜里点火和放炮算是同样的乐趣了。因此这一小瓶火油足够我把十里八乡的小孩都招揽到旗下了。但是俗话说嘴上没毛，办事不牢，火玩着玩着确实就没谱了，一个不小心就点着了人家家里没搭完的半个麦垛。眼见着火势越来越大，一起玩儿的小孩哭的哭，跑的跑，只有坤哥打定主意带我到山里去避一避。坤哥带我在一个火车桥洞下待了一个晚上，整个夜里，火车哐当哐当的从我俩头顶通过，我俩不禁去想火车这种物件究竟会通往哪里。

第二天白天，我俩被我二伯找到了，幸得点燃的麦垛并不太大，加上坤哥极力帮我分担责任，我才没有被打成三个月不能下床。后来我外出去钢仁求学，一两个月才能回村一次，就和坤哥断了联系。这时看到坤哥，我心里不由得高兴：“坤哥，你也来坐火车？”

“是的啊，我考上了一所西京的二本。”坤哥看着我说。

坤哥为人聪明，方正的面庞配上挺拔的五官，再加上经常健身造就的健美的身材，实在容易让人喜欢。只是坤哥家境并不十分好，让他并不能全身心投入学习，时常要做些活计补贴家用。

我决定先不把自己考上西京交大的事告诉坤哥，只说也去西京上学，待后来找个机会与坤哥好好聚聚再说。虽然我知道坤哥只会真心恭喜我，但我目前还过不去心里炫耀的坎。三言两语间，我知道了坤哥没跟我买上一趟车，他是转道长沙再去西京的。十几年过去了，那夜共同畅想远方的我和坤哥，终于搭上了各自的火车。

一路无话，早上7点54分，火车准时抵达了西京南站。西京南站是原来长安火车站改建的，专发重庆成都等地的往来车辆，因此离西京市区还有30多公里的距离。这里又没有公共交通可乘，我只好坐了一辆黑车前往西京交大。

从交大南门进去，直对着的是思源学生活动中心。这栋建筑四四方方，是90年代交大建校百年时所建，上面还有知名校友苍劲有力的题字。再往北走是新建的A\B\C\D\E五座主楼，其中主E处于中心有23层108米之高，是学校教务行政管理系统的集合所在，其余四座教学用楼环绕在主E周围。建筑群的东西南三侧各有一个地标，东边是盛宣怀，西边是求学鼎，南边则是孙中山。更往北是钱学森图书馆和曾经的教学楼，专业课大多在那里授课，现在不便多叙。

新生注册完成后，我被分到了崇实书院东723宿舍。我进宿舍的时候，其他三人已经到了，正聚在一起熟络的聊天交流。看到我进来，他们也没有暂时停下，同我打招呼的意思，我只好主动开腔：“你们好哇，我是陶砂。“

三个人这才转过头来看我，然而回答我的不是同样的你好或自我介绍，是一句来自A床位的冷冷地询问：“你就是那个加了20分来的少民？“听了这话，我心里一惊，怎么我才刚来底就被漏了呢？

# 第三章 初露端倪

西京交大的宿舍是上床下桌布置的4人间，从门口进来后左右各两个床位，依次编号为A\B\C\D，其中A和D靠门，B和C靠阳台或外窗。同我问话的就是靠门边A床位的同学。一般说来A床位靠近门边，不仅出入方便，而且远离阳台冬不冷，夏不晒，晚上还安静，是宿舍老大的专属位置，难怪他敢一马当先揭我底牌。

俗话说，知己知彼方可百战百胜，目前我初来乍到，完全摸不透这些城里人底有多厚，水有多深。更何况这三人似乎早已熟络，又和我同处一室，安榻之处，不得不防。再一看他们留给我的位置，果然是紧挨厕所的D床，我心想：好啊，你们还真是一点都不装，这D床我就先睡下了，但你们等着，早晚我要占了你A床的位置。

我心下盘算清楚，开口回话道：“多亏了祖国扶贫的政策，我才有机会来到这里学习，今后还要多跟大哥们学习城里人的生活方式。”这番话算是给了他们面子，又把我自己与国家的大政策绑在了一起，足以让他们对加分一事再无话可说。

果然他们三人见讨了个没趣，也就不再理我，又旁若无我的聊起了天来。

我也不愿多生事端，自顾自地开始整理床铺，大堡村地处偏远，也没什么特产，所以我从家里带来的都是书。书读多了，我就能充满智慧，就能成为大作家，我始终相信这点。来这里之前，我已经好好了解了西京交大的历史，从这所名校走出过各行各业伟大的人物，却唯独少了作家，我陶砂想成为第一人。

转眼到了熄灯时间，虽然崇实书院的一大好处就是无门禁，不断电，但为了明日开始的军训，我还是按着时间关灯休息了。

军训一事，历时长久，早些年是为了应对国际的恶劣形势而设立的必修课。在我国立国之初，先是美帝后是苏修，外部压力一直很大，因此有必要让更多人接受军事教育。那时的军训正规严格，往往要训练三个月之久，训练项目也包含了射击、掷榴弹、战术动作等实战内容。而后来随着改革开放，外部压力减小和军事科技的进步，军训越来越流于形式，不仅时间削减到了只有两周，训练内容也仅剩队列操练。这样的军训不知为何还没有消失，于我而言，我分不清左右，有时又会顺拐，中学时甚至还被训哭过，自然是更讨厌军训的了。

因此开训没两天，我就假装崴了脚，当上了病号。其他人操练时，我就坐在一旁看书，这样的军训，也算得上是件美事。军训这两周在我看书的时间里过的飞快，一本书看完就到了总结大会的前夜。这天晚上解散后我收拾好书本和马扎，正打算去食堂吃饭，就接到了坤哥的电话。

“喂，陶砂啊，你考上了西京交大怎么不跟我说啊，还把不把我当哥们了。”

听到坤哥这么说，我心下有些奇怪坤哥是怎么知道这事的，不过坤哥知道了也好，我本身也没打算瞒他。来西京两周我一直还没习惯城市生活，因此也没来得及招呼坤哥，现在坤哥打来电话，我自也是十分开心：“哪能啊，坤哥，这不是那天你赶火车太急，咱俩都没说上几句话，把这事给忘了吗。”

“那你今天总没事了吧，出来我请你吃个陕西特色，咱俩好好聚聚。”坤哥在电话那头说道：“你可别想推脱，我知道你们军训结束了。”

坤哥不来找我，我也是要主动邀请坤哥的，这事自然没有拒绝的理由，因此我欣然应允，约定30分钟后东南门外见。

时间虽然不算充沛，但毕竟出校门吃饭也不好再穿迷彩服了，因此我回宿舍简单的换了身衣服，就匆匆赶往了东南门。

坤哥早就在东南门外头等我了，见了面我俩也不再客套，就直奔今天的主题——吃饭。坤哥要带我去吃的是魏家凉皮，店铺离东南门不远，出了门过天桥朝北走没两步就到了。说起凉皮可是西京的特色食物，讲究个酸、辣和筋道。凉皮虽然用的是米皮，可嚼起来却要有扯面的口感。碗里先调好醋和蒜水，醋要用岐山的陈醋，香中带酸，酸里透香，谁也不能盖住了谁，蒜要用当日的新蒜，这样调出来的味道才是可口的，而且吃完还能没有蒜味。接着将蒸好切细的米皮尾端，浸到鲜红油亮的油泼辣子里去蘸上一蘸再加到碗里。这才算是正宗的西京凉皮，魏家凉皮为了快餐生意省却了许多步骤，醋都用成了米醋，辣子也是料包，不过整体说来味道还算稳定，各家店都差不太多。

我俩点餐，取餐，落座，接着自然是叙旧，不多时就由过去说到了现在。坤哥突然一停顿：“你之前一直专心读书学习，对网络还不太熟悉吧。”

我点点头，不知道为什么坤哥突然说起这事。

“我早些年就开始玩儿贴吧了，因此对网络比你接触的早，还没跟你说吧，我上的是西京交大思源学院。”坤哥说着又吸了一口冰峰：“这个学校跟你们共用一个BBS，叫做兵马俑BBS，这和贴吧差不太多只不过是学校内部的，我就是在这里发现你上西京交大的。”

“啊？这跟我有什么关系？”听坤哥这么说我难免疑问重重。

“你自己看看吧。”坤哥打开手机找着什么，接着递给了我。

我赫然看到置顶的一篇帖子“钢仁中学倒一生，加20分考进西京交大”

“你在西京交大已经火了。”坤哥对着我说。

这时我回想起军训时候的事，难怪有那么多人偷偷盯着我，从我身边过的时候还小声说着什么，我原以为是之意我腿瘸，谁想到竟是质疑我的学习能力。这我可忍不了，我倒要看看是谁揭我伤疤，只见帖子下面紧跟着发帖人的昵称“马儿嘚嘚”。
