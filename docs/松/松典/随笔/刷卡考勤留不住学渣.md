# 刷卡考勤留不住学渣

Date: April 18, 2022
album: 象牙塔

我不知道其他学校有没有这样的制度，但沙坡村职业技术学院就有。每次上课，我们走进教室都需要用校园卡在系统上刷卡。时间太早，刷卡无效；时间太晚，刷卡不是迟到，就是无效。累计三次缺勤，该门课程强制不合格。

对爱学习肯努力的人来说，刷卡考勤可有可无，而且会因为全勤，期末成绩得高分。但，对学渣来说，刷卡考勤是留不住他们的。我们吐槽，我们抨击，我们看不惯，归根结底就是形式主义太泛滥。只是，形式主义像决堤的洪水泛滥，我们还是有自己的办法，那就是刷卡走人。 课前跑到教室，刷完卡，回到寝室睡觉，又或者干其他事情，大有人在。

大学前期，我有逃课到图书馆阅读课外书。印象最深的是基础心理学。我到教室刷完卡，从四楼跑下，直奔图书馆。也是我运气不好，走到楼梯口，巧遇授课老师迎面走来，不得不跟她招呼。那时，我还青涩幼稚，心里想什么别人看看就知道，压根不需要思考。我有些儿脸红，甚至表现出羞涩。随后，这位老师就在课堂说起我来，说我有点不好意思。

随着时间的流逝，同学当中包括我自己，没有谁抱怨刷卡。人性就是如果有机会改变，总要去尝试。如果没有机会，就只能随波逐流或者说逆来顺受。在老师眼皮子下翘课，这种事情，大概是愚蠢的。抱怨亦是愚蠢的。

没有撞见，偶尔有两次，其实情有可原，或者说无关紧要。但翘课被老师逮个正着，那还真是件难办的事情。毕竟，老师也是人，是人都有人性共有的东西，比如说偏见与价值取向。有偏见，有价值取向，期末成绩就很有可能因为傲慢无礼而被弄得老低老低。当然，大多数课程，我没有翘，即便自己不愿意听课，还都在教室翻看着课外书。

寒来暑往，很快进入高年级。那段时间，我忙着读书，忙着创作，每天是以小时为单位精确计算的，心头洋溢着日新月异的激情。上午做什么，下午做什么，夜晚做什么，我安排得妥妥当当。我飞速奔走在教学图书馆食堂寝室之间，时间还是不够用。于是，有段时间，我跑到教室刷完卡就回到图书馆忙碌自己的事情。这天下午，天空阴沉，乌云压得老低。刷完卡，我从中楼走出来，走在樱花东道上。也是我悖时，竟偶遇任课老师，任课老师很生气。

“你简直就是无组织无纪律，任意妄为不懂规矩！”我只能灰溜溜跟着老师回到教室。路上，他告诉我说：“一次两次就不和你计较，三番五次，还这样张扬还这样明目张胆，太不把我放在眼里了！你自己想像，你这样，我还让你通过，其他老师怎么想？！”我觉得老师也不容易，何必要给他们增添麻烦。自那以后，我再没有逃课翘课。

夏天远去，秋天已到，我基本是最早到教室，最晚离开教室的，亦是经常泡在图书馆的人。

“前面你就像小孩，压根不懂事。后来，变化明显，成长得很快很快。”是啊，有些许经历以后，再晚熟的人还是慢慢成熟起来。如果重来，我大概不会那样。矛盾的是，如果不会那样，我又能成熟吗？
