# 好讨厌高数

Date: April 17, 2022
album: 象牙塔

在全国，文科生要学高等数学，这是比较少见的。这比较少见的事情，偏偏发生在我身上。那时，我非常痛恨高数，非常痛恨高数老师，非常痛恨安排高数的院系领导。我无法理解他们的意图，无法认可所谓的通识教育。

中学时代末尾，我心想，考进大学，然后选个文科专业，我再也不用学习可怕的数学。然而，现实狠狠给我打了耳光，打得我晕头转向。虽然极其不服气，虽然极其不舒服，但是，我没有办法，我只能接受现实。

每次，我早早走进教室；每次，我坐在前排中间；每次，我认认真真听课；每次，我认认真真抄作业……遗憾的是，我依然差那么点点。于是，那个学期末尾，我的高等数学正式挂科。每门课挂科都有次补考机会，尽管我努力复习，还是以相同的分数再次挂科。说实话，我对高数的恨已到达无以复加的地步，恨不得将教材扔进垃圾桶。

憋屈的是，我还不能扔。重修需要用教材，扔掉，还要掏钱买。暑假，高数有重修，我听课啊复习啊。结果，再次以相同的分数挂科。我觉得，我的心被高数折磨得像老农民丢到路边的包心菜，由里到外都已经烂透。

函数、极限、矩阵、微积分、参数方程、排列组合等等，现在，我只能记住这些名词，并且，还像那时坚定地认为，文科高数没有鸟用。也许，文科高数对有些人，确实有鸟用，但对我来说，真他娘的，没有啥鸟用。即便没有啥鸟用，我还是要去学习。不学习没有搞过，我就不能获得毕业证，这就是现实，现实就是现实，就是这样残酷。

那年秋天，我继续重修。我的重修可以说是，屡败屡战，屡战屡败。我跑去交作业，老师告诉我说，你不用交作业。随后，渐渐坐到教室后排。他的讲课没有任何变化，没有任何变化包括他的笑话他的动作等等。

前面那节课，我认真翻看教材，认真阅读教材后面的故事。后面那节课，我专心阅读课外书，阅读武侠小说。那学期很快很快就结束，转眼来到期末。考前有次答疑，我跑去办公室问老师，他说那怎么行呢？！我心灰意冷，从数学与统计学院走出来，看着腾飞塔跟前的不挂女神，我还是来年继续重修吧。没准，我可以用态度感动老师放行。

期末考试来临，我深刻理解到，死猪不怕开水烫。我就像是死猪，我压根不怕！我依然早早走进教室，坐在特定位置，仿佛走进刑场的囚犯生无可恋。我认认真真答题，时间飞快流走，走出考场远离梦魇。

成绩出炉那天，我抱着侥幸心理点进教务系统，谢天谢地！我奇迹般通过。平时满分，考试及格，我知道，这是老师慈悲为我放行。为此，我特意修书，特意准备书籍，专门跑到办公室去表示绵薄感谢。

后来，我得知跟我重修的人，由于平时没有去，单纯参加考试，这位老师公事公办，继续煎熬他们。

再后来，跟老师吃饭，期间，我有说起这件事情，老师告诉我说：“这就是情商，情商是捷径，可以解决很多问题。”我点头附和，心想，人都是有偏见的，都是趋利避害的。情商高，确实可以左右逢源，甚至办成大事。

好讨厌高数，也许是许多文科生的心声。但是，成熟的人几遍好讨厌，也要去把事情做完做好。
