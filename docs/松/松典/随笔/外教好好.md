# 外教好好

Date: April 20, 2022
album: 象牙塔

秋初，天空高远、辽阔、湛蓝，为酬谢朋友曾把他的坐骑让与我作伴十多个月，我们在东南们附近吃串串。期间，他曾告诉我说，那是平静的小镇，人们各干各的事情互不打扰，人与人之间相处很和谐。

在那瞬间，我想到外国人。进入大学，我陆陆续续接触许多外国人，有来自土耳其的，有来自巴基斯坦的，有来自吉尔吉斯斯坦的……印象最深的，还是来自美国的。某种程度，我要感谢学校，感谢学校在小学期邀请来外教。

我们的外教有两位，每位给我们讲课六天。前面那位外教是为中年妇女，她把她的女儿带到校园，带进我们的课堂。透过外教，我能够感受到不同民族之间教育的差异。她们注重孩子的天性，注重孩子的兴趣，平等而坦诚地对待着每位学生。反观我们的教育经历，只有极少数教师可以达到这样的境界。大概就是这种文化造成创造力差异。

成熟的社会尊重每位社会成员的天性，尊重每位社会成员的选择。在那样的社会，人与人之间不需要太多尔虞我诈，不需要太多勾心斗角，不需要讲究太多规则与潜规则，只需要按照自己的理解度过渴望的人生。

自由、平等以及包容的社会往往是天才诞生的土壤。没有这样的土壤，杰出人才永远培养不出来。

小学期末尾，我们有拍照合影留纪念。临走，我准备了英文书籍，当礼物送给外教。来年，外教又来我们学校，我们曾在微信上聊天。有时候，世界很大；有时候，世界很小。在校园，我们偶遇，她身边是她的女儿。

再后来，疫情突然席卷全球，中外之间的交流中断。那年秋天，外教发消息告诉我说，因为疫情，今年不能到中国，我们很遗憾。我回复她说，海内存知己天涯若比邻，祝您生活愉快。当然，我们之间的交流是用英语进行的。我不懂表达的内容借助翻译软件。哪怕就是美国这样强势的国家，它的民众依然值得交往，因为他们坦诚平等尊重个性。

国家与国家之间，民族与民族之间，也许存在利益的冲突，但这并不意味民众就怎样怎样。

后来，即将远离校园，我拜访外文系的老师。这位老师告诉我说：“外教不能来中国，小学期的英语就由我们外文系跟新东方合作来完成。”我听着老师说话，心想有些新生不能接触到外教，还挺遗憾的。

回头细想，这只是我以为他们遗憾。也许，他们自己不会觉得呢。
