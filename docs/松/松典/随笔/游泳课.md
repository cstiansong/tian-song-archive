# 游泳课

Date: May 11, 2022
album: 象牙塔

游泳课

有天整理书架，我瞧见泳镜、泳帽以及泳裤。凝神细视，我的思绪飘到遥远的北方。

那时，我知道，游泳课结束，大概不会再用它们，但是，我舍不得丢掉。所以，我把它们堆放在衣柜某个角落。每次整理衣柜，我会瞧见它们，瞧见它们时，我会想起许多事。由于课程冲突，我不能再选原来的健体，不得不改选，改选到游泳课。事后，前往图书馆的路上，我听到有女生说：“选游泳课的人太多，我都选不上。”

我们的体育老师很好，每次课前，他总是要给我们摆事实讲道理。他曾询问我们：“有没有同学想要成为权威专家？”我看着他觉得他阅历丰富有思想。“大师泰斗需要天赋，但权威专家只要努力就够。”

我心想也是，哪怕就是小虾米，天天嚷嚷着成为龙，天天以龙的标准训练自己，最后未必是龙，但肯定是龙虾！我这只小虾米脱光衣裳脱掉裤子，全身光溜溜的，只剩四角内裤。镜子跟前，我骨瘦如柴，浑身上下没有多余的肉，完全就是皮包骨，裸体重量五十千克。可以说，很瘦，很轻，所以演示下水救人，我扮演的是溺水者。

有些人包括我自己为了引起关注，经常发表那些能够否定社会普遍认同的结论，进而突显自己的个性。游泳课上，我总是在跟周边的人那样说话。如果他们不在身边，如果我没有事情做，我就会注视着远方出神。

许多话，我想要说出口，但是，说出口，我就是罪人。作为罪人，我就是付出相应的代价。所以，我没有说，我没有说许多时候，我总是在偷偷观察他们的身体。我羡慕他们，羡慕他们结实的肌肉，健美的躯体……每次游泳课结束，我们都要到澡堂洗澡。这是我大学仅有进澡堂洗澡的经历，其余时间，我总是到开水房打热水，然后到卫生间冲洗。北方的浴室只分男女，每间浴室同时可供多人洗澡。我看着那些裸体，看着那些裸体黝黑的地方，雄性动物洋溢着野性的美。

在我亲眼目睹裸体在澡堂冲洗时，我在想我有那样健硕的躯体，有那样雄伟的阳物……我肯定会很自信，我会自信到走在路上抬头挺胸，我会自信到遇见温柔美丽的女孩跑上去询问联系方式……作为小个子瘦弱的男生，那时候，我在身体上极度自卑。人在这方面极度自卑，就会在某个方面极度自傲。所以，我是个极度自卑自傲的人。

有次，我潜游到游泳池底部，我觉得世界静悄悄的，除了窒息，只剩安静。我享受着那种安静，那种安静让我欢愉，让我全神贯注，让我心无旁骛。在那以后，我猛地意识到，无论做什么事情，专注可以让效率翻倍。

许许多多的随想，许许多多的经历。无论欢乐，或者忧伤，最后都会跟随岁月的流逝老去。直到多年，它们成为遥远记忆里某道亮丽的风景，足够自己可以追忆似水的年华。老实说，游泳课给我的记忆挺多。

在挺多的记忆里，最深刻的是，我习惯在游泳池里像鸭子飘来飘去，无拘无束的感觉让我觉得自由与幸福。有次，我仰躺在水面上，像睡在软绵绵的云上。少年时代那些关于游泳的记忆从心底缓缓浮出脑海的水平面。

很小很小的时候，我就会野泳。在那群伙伴里，我的野泳最差劲，差劲到我曾差点被洪水淹死。因为有了那样深刻的经历，我对水有种恐惧。走在河边，走在湖边，走到水边……我总是能够感受到幽深的水在引诱我。

我觉得水里面驻扎着什么东西，总是在引诱着我跳下去。每当涌起这种诡异的想法，我就不得不转移视线。我把视线转移到远处的女孩身上。我观察着她们的身体，她们的身体对我的诱惑远远大于我对文学的痴迷。

那个学期，我注意到有个穿着桃色泳衣的女孩，她鼓胀的乳房像山峰挺拔，圆润的臀部像西瓜凸起……属实胸大腰细屁股圆润。她的姿势、动作、神情，以及声音，让我神魂颠倒，让我忍不住多看几眼，仿佛多看几眼就属于我，就可以任我摆布。然而，当她顺着目光看过来，我已潜游到水底，不想让她注意到我，注意到我对她有些下流的想法。

游泳课结束，我没有再去游泳池，有关这个女孩的记忆随着时间的流逝被我丢到游泳池里。然而，后来我在前往东南田径场拉引体的路上，再次遇见这个女孩。遇见她的时候，我想起她穿泳衣的模样，腹部涌起滚烫的热流。即便这样，我不得不感慨，女人的脸还是很重要的。毕竟，不是在特定环境，我们最先看到的总是对方的脸。

那个学期结束，我每天坚持健身。随着时间的流逝，我的身体有明显变化，不再像曾经瘦骨嶙峋，而是洋溢着青春的活力。这种活力让我成熟，让我阳光，让我精力旺盛，让我觉得自己可以抱着女孩转圈。
