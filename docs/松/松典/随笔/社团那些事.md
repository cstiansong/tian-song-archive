# 社团那些事

Date: April 19, 2022
album: 象牙塔

有人曾亲口告诉我说：“没有参加社团的大学是不完整的。”我心想，你怎么不说，没有挂科的大学是不完整的？有关社团那些事，有关忙碌社团那些事的那些娃。哪怕有些人因此骂我，我都有必要写写，有必要说说。

中学时代，我曾参加学生会。前几次，我都参与进去了。后来，我觉得社团那些人其实蛮搞笑的，泛滥的形式主义流露出太多官僚主义色彩。与此同时，我想把时间用以学习，我觉得只有学习可以改变我的命运。社团只是人生路上的风景，并没有太多实际的价值，索性不再参加例会。进入大学，我曾高调宣称自己拒绝参与社团，我的理由是没用。

是啊，社团确实没用。如果有用，那就是可以结识朋友，甚至可以找到女朋友。当时，我没有意识到这些，没有意识到可以结识朋友，甚至可以找到女朋友。后来，我想了想，我还是没有必要到社团去鬼混。那些地方，太浪费时间。什么辩论社，辩论起来就像表演，我忍不住就想嘲笑；什么国学社，国学起来就像文盲，我忍不住就像捧腹……

我知道，有些社团还是很好的，比如说爱心社。虽然我没有参与爱心社，却有参与爱心社组织的活动，曾偷偷把自己不愿意翻看的书籍捐献。中间，我有参与社团。这个社团是读书社，读书社是西迁精神读书会。那是某个周末，某位学姐把我拉进读书群，随后参与这个社团的例会。但那以后，我走偏了，我心里想着的全是成名成家。

阻挡我成名成家的事情，包括学业在内已经变得不重要，就别说社团这种活动。社团这种活动，也许可以认识朋友，但是，长期浸泡图书馆亦可以认识朋友，比如说，我就有许多馆友，这些馆友不仅特别，而且优秀出色。

有些傻孩子有了社团那些事以后，学业方面不放在心上，最后自然而然没能进修深造。当年刚进大学那会儿，我曾询问学长关于社团那些事，这位学长告诉我说：“绝大多数参加社团的人，他们的成绩都不是很好。所以，时间越往后走，退出社团的人会越来越多。”我心想也是，资质都差不多，就看谁有的目标更明确，谁的规划更清晰，谁愿意付出更多努力。

这位学长补充说：“社团就是给学校给老师免费提供劳动力，比如说有些讲座没有听众，相关老师就安排参加社团那些人去。”我不得不认同这位学长。什么朋辈课程，什么暑期实践活动，我就没有参与，还是顺利毕业了。

但是，社团有社团存在的意义，哪怕没有大用，我们不能因为社团没有大用，就否定社团存在的意义。

远离校园，现在回想起有些事情，包括我对社团的态度，总觉得自己太幼稚。觉得没有意思的东西，不去接触不去表态不去冷嘲热讽，不要袒露内心太多真实的想法，这才是读书人应该持有的人生态度。

如果对后来者有什么建议，那就是，要清楚自己想要的，到底是什么东西。
