# 致谢在最后

Date: June 10, 2022
album: 象牙塔

亲爱的朋友，谢谢您阅读到这里，只是，我该跟您说再见了。我很坦诚地告诉您，下面这些的文字，并不是写给您的。所以，您可以放下书本，做您热爱并且相信的事情。但我还要继续，我要借此机会致以谢意。

谢谢军训教官，谢谢高数老师，谢谢体育老师，谢谢心理学老师，谢谢宋希斌老师，谢谢卫朝霞老师，谢谢美丽的女孩，谢谢外教和她的女儿，谢谢追太猛的姑娘，谢谢熊猫姑娘，谢谢花尾鱼老师，谢谢哲学老哥，谢谢班主任，谢谢陪我去灞桥看过风雪的那个她，谢谢张弓长老师，谢谢郑元锁老师，谢谢一只兔子，谢谢少年班那些天才，谢谢骗子，谢谢单身汉，谢谢小包工头，谢谢上海女孩儿，谢谢借我大奔的朋友，谢谢捕头大哥，谢谢平行线，谢谢占虎哥……

本部分是上文省略号的内容：谢谢葛建雄，谢谢独轮车主，谢谢博士姐姐，谢谢李尚宗老师，谢谢新东方，谢谢交大附中，谢谢法学生们，谢谢泰发祥，谢谢好兄弟，谢谢刘维政老师，谢谢王朝霖老师，谢谢郝明松老师，谢谢物理学博士，谢谢姚明今老师，谢谢燕连福老师，谢谢王树国校长，谢谢同在屋檐下的室友，谢谢我们中文系全体老师。

谢谢学长学弟，谢谢学姐学妹，谢谢同班同学……谢谢崇实书院，谢谢钱学森图书馆，谢谢西安交通大学……谢谢生命让我们相逢。也许，我们不会再遇见，但我们曾经相逢；也许，离别时，我不够好，但离别后，我会努力。

谢谢那些出现在我生活后来失去联系的朋友，亦感谢没有出现在这个故事合集，却依然留在我身边的人们。谢谢你们曾愿意忍受我年少时的情绪以及孤傲。真的很感谢大家，感谢生命让我们相逢。

最后，真诚感谢帮我出书的朋友，没有你们提出的建议，没有你们给予的帮助，我想，这部作品可以完成，却会遥遥无期地拖下去。最后的最后，真诚感谢参与本书的朋朋友，感谢你们让这部作品顺利降临人世间。
