# 初露端倪

Date: April 17, 2022
album: 象牙塔

我这个人性格不好，有段时间，现在依旧。我非常擅长说话，说起话来滔滔不绝；我非常渴望说话，说起来话喋喋不休。我非常喜欢热闹非常愿意凑热闹。可以说，哪里热闹，我就想往哪里去，人越多就越兴奋。

于是，刚进大学，我总是在各种课程群里挥斥方遒，仿佛自己说得越多，就越能证明自己牛逼。

不管是真实的课堂，还是虚拟的群聊，我时常指指点点，各种品头论足。那时，我看不惯有些人，我总忍不住要说几句，属实不说话，我就心里像是猫爪子挠痒痒，太难受。整天就是，这儿看不惯，我要说；哪儿看不惯，我还是要说。这个人这里看不惯，我要说，说起来不懂看眼色；那个人那里看不惯，我还是要说，说起来，直截了当毫不客气。有时，我还要为某些现象打抱不平仗义执言。远处的有，近处的有，压根就没有洁身自好的意识。

天长地久，被我看不惯的人，老爱跳出来，跟我争高低。那时，我心高气高，谁都不服，总是要据理力争，甚至争得脸红脖子粗，最后断交，老死不相往来。就这样，我经常跟人起冲突，经常跟人发生矛盾。

我看不惯他们精致利己，看不惯满嘴谎言，看不惯他们投机取巧，看不到他们不与人为善。老实说，我跟有些人素无瓜葛，但他们老来烦我，搞到最后，我是压根没有风度，没有涵养可言。如果岁月的年轮可以转到曾经，我想，我依然看不惯有些人。但是，我会爱惜自己的羽毛，妥善经营我的事业，专注着推进自我建设，而不是浪费时间。

善恶到头终有报，不是不报，而是时候未到。我看不惯的那些人，那些冥顽不灵执迷不悟的人，最后都没有很好的结果，而我将以世俗的成功回敬那些不是人的东西。反正，人活着要有气，要不服气。只要胸怀大志，就会有不服气，就要不服气就不会轻易放弃。也许，将会因为不服气而颠沛流离，但是，潜能就是这样激发出来的。

有老师告诉我说：“你是个与世无争的人。按理说，你的大学不应该这样磕磕绊绊。说到底，就是你没有叙事意识，不懂得叙事技巧。”当时，我不懂，只是印象深刻，牢牢记在心底。后来，我忽然想起，觉得那个因素确实存在。

归根结底，还是我个性太强烈，桀骜不驯，谁都瞧不起，有种无厘头高高在上的优越感。

就是这种优越感让人感受到高傲，感受到少年的棱角。所以，那些人看不惯我，看不惯我轻狂，看不惯我高傲，看不惯我不懂规矩。他们看不惯我的同时，我亦看不惯他们。心动，则天地动，看不惯就这样诞生。

由于长期沉浸在文学世界，我的脑海总是燃烧着熊熊火焰，让我活在某种幻想深处。这种状态下，我爱较真不服气，看不惯富有激情。每每遇见不顺利的事情，我就忍不住去抨击，像孙悟空，极具反抗意识。

我啥都没有，却用“说话”这根金箍棒，在某个小群体大闹天宫。等到内外因综合作用，真实的自我膨胀到极致，我像是武侠小说里面的人走火入魔剑走偏锋。我啊，任性随意，挥舞着金箍棒，不停不断地宣泄激情。意气风发与书生意气这两种力量相互交织，我却找不到表达的途径。在那以后，只要开口说话，周围人总要被扎到。

荒唐而任性的青春，在大学初期渐渐显露端倪，进而为后面极端情绪化埋下伏笔。
