# 王老师

Date: June 11, 2022
album: 象牙塔

时间像流水在消逝，有些事情像暮春时山巅上的白雪不复返。也许，在将来某个时间点，还会经历相似的事情，但原先那些逐渐被岁月的风沙掩埋，可能会在角落深处被遗忘。即便这样，我还是用文字记录了下来。

王老师是外国语学院的老师，我并不是她的学生。即便不是她的学生，还是因为缘分，成为她的学生。刚进大学那会儿，公共英语基础课有晚自习。我就是在那晚自习认识她的，遗憾的是，那时候，她并不认识我。

初夏，鸟儿展翅往天空高出飞翔，我凝神注视着它们，看着它们使劲往天上飞。有那么瞬间，我觉得不是它们在往高出飞，而是我往下沉。把目光从远方收了回来，我径直走向梧桐苑。从梧桐苑出来，我买了盒西瓜。走在梧桐西道上，我边走边吃，偶遇王老师。我小跑上去招呼老师好！“老师，您吃西瓜吗？”她说：“我家里有……

“你暑假回家吗？”我解释说：“我暑假不回家，我要在图书馆读书。”

随后岁月，偶尔在校园的路上遇见。那年盛夏末尾的夜晚，我从图书馆出来走到东南田径场散步，从东南田径场散完步回来，康桥苑门口，我们再次遇见了。王老师在这儿等学生交作业，我跟她说了好多好多话。

我告诉她我想要当作家我想要走什么样的道路……我加了她微信，推荐我的文字文章……事后，她曾告诉我：“我觉得你有天赋，你的文章深处有种独特的东西。”学生时代，关于文学梦的事情，有老师觉得我有天赋，有老师觉得我没有天赋。我没有因为肯定而认可，没有因为否定而批判，我认认真真地对待着他们的想法。

有次，我从梧桐苑出来，准备去钱学森图书馆。在路上，我遇见了王老师，王老师正跟有位老师交流。这位老师跟我介绍说：“这是哲学系的系主任。”彼时，我是个情商极低的人。“我不认识，我要去图书馆啦。”

随后某个傍晚，偶遇王老师，我坐在她对面吃饭，她告诉我说：“你说话要注意场合，尤其是你说话的方式！”关于那件事，我有在日记里反思，确实不应该说不认识，确实不应该说要去到图书馆。亦是这时，老师告诉我：“很多人有智商，但他们不勤奋。你不仅 有天赋，还勤奋自律，有很强的执行能力，所以，我相信你可以实现你的梦想。”

也许是我的真诚，也许是我的个性，也许是我的善良……我约了王老师，王老师约我到附近吃饭。期间，我们说了好多好多话。“我应该让你闭嘴，你看这些菜都凉了！”我送她回家的时候，她的脚弄伤了。

许久没有穿高跟鞋，许久没有穿白裙，老师有点儿不适应，不小心让高跟鞋弄伤了脚后跟，我有些儿心疼。送她到楼下，我目送着她上楼，她回过身看了看我，我招招手说了再见。这天这时候，苍白而昏暗的天空阴沉沉地笼罩着梧桐树梢，忧郁的天气却没有影响到我的心情。此刻，我走在路上，觉得很有收获，很感谢生命让我遇见善良的老师。

梦想是什么？梦想就是让你像苦行僧天天努力，却不觉得辛苦的感觉，这种感觉无时无刻不在证明你的的存在。我告诉自己任何时候都不要放弃梦想都不要放弃事业。我相信，东野圭吾在《在学生街的日子》说的那样：“如果一直找下去，也找不到，那也是种人生”。所以，我并不担心自己将来找不到工作，找不到工作就没有饭吃。

后来，我把我栽种的玫瑰送给老师。从崇实书院搬到彩虹桥，从彩虹桥搬到老师家里，我的手酸痛酸痛的，但我觉得这是我幸运。从老师家里出来，她从冰箱里取出半个西瓜几个桃，我回到图书馆就分享给馆友了。
