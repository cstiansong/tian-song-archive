# 上海女孩儿

Date: May 10, 2022
album: 象牙塔

男孩像天边那个太阳，女孩像夜晚那轮圆月。太阳没有月亮，它不会完整；月亮没有太阳，亦不会完整。

生命中，我们总是会遇见许许多多的男孩，总是会遇见许许多多的女孩。我们无法给予清晰的定位，但，他们给我们的人生增添某种光彩。也许，她曾是自己的月亮；也许，他曾是自己的太阳……遗憾，我们不得不忘记彼此。

上海女孩儿，她有种独特的气质，就是那种独特的气质诱导我写下这篇文章。没有人知道故事的主人公，而故事的主人亦不会明白，现在我会把她写到自己的文章，仿佛要在温柔的语言深处找回昔日的恋人。

我曾思索我们之间的微妙关系，曾认真思考她为什么有那种独特的气质可以吸引到我。说起来说出来，倒是很简单。因为我们的气质像太阳和月亮。太阳不会喜欢太阳，太阳会喜欢月亮，因为月亮阴柔、美丽。我知道，我兼具有阳刚和阴柔的气质，但大多数时候，阳刚占据了我的全部。我随意冲动热情坚强……她娴静稳重清冷柔弱……关键是她来自上海，来自上海那种富裕的家庭，那种富裕的家庭养育出了她那样优美典雅纯真，甚至高贵的女孩儿。

我在小山村出生，在小山村长大，我的童年踩在地上，玩着泥巴长大……这种生长环境让我对城市有种天然的憧憬，这种憧憬深处蕴藏着浓烈的虚荣。我渴望鲜花、掌声以及荣耀的人生。上海集聚着人世间的繁华，可以说是美丽的梦吸引着我。我想上海女孩儿就是那种美景的文化符号。所以，我倾慕她的美丽，倾慕她的柔软，钦慕她的身份……

有些人可能会觉得我下流，甚至龌龊，但是，我曾深刻地感受到这个女孩身上有着独特的香气，这种香气充满着诱惑，就像肥硕的乳房摆在我跟前，忍不住想摸。她身上散发着的诱惑让我变成蜜蜂，变成蜜蜂跟着她飞；让我变成蝴蝶，变成蝴蝶围绕着她起舞。这种感觉让我觉得自己像是在吸毒，完全就是飘飘欲仙心醉神迷。

岁月可以让我苍老，却不能改变我的灵魂；时间可以让我近视，却不能改变我的眼神。

我想，偶尔遇见上海女孩，我的眼神深处肯定荡漾着某种温柔。她这样的女孩从小到大都是掌上明珠，不像我的童年忧伤，不像我的少年曲折，不像我的青年荒唐，她压根就是小说里面才会有的女孩儿。

这个女孩，只有正直、善良以及成功的男人才配拥有。不仅因为她温柔，不仅因为她善良，不仅因为她美丽，不仅因为她懂得理解，而是因为她需要那种男人，需要那种男人真心实意地为她付出，她才会有幸福的人生。

我习惯于独来独往，常常喜欢在校园的路上走来走去。走路散步的时候，我的脑袋不会停止运转，常常会回忆从前。如果没有回忆从前。我就想入非非，思考着将来我要怎样怎样，思考着男男女女之间的情长情短。

我痛恨平庸，痛恨我在这个年纪，即便得到女孩的点头，亦没有能力守护对方的信任与托付。

莫言曾说：“作家的气是很重要的。当你处于胸怀大志急于表现又表现不出来的时候，那时候可能把你的很多潜能都调动起来。”正是那种明知道将来会大获成功，但又拿不出什么可以证明自己，这种状态让我觉醒天才。许多天才，刚刚登场时，乍看没什么了不起，越看越了不起，越看越觉得是天才。在天才觉醒以前，他们要忍受爱情的淬炼。

所以，男人的梦想往往源自女人。在许多男孩的青春岁月，仿佛没有爱情的忧伤，他们便不能蜕变成为成熟稳重的男人。事实上，失恋、贫穷，以及屈辱确实可以无限激发男孩的潜能。有什么样的思想，有什么样的志趣，其实是命运的安排，我们没有办法选择。我来自山村，与世无争，却在命运的安排下不断蜕变……不能归咎于我个人。

没有表白，只是曾坦言，你很好，让我觉得安详……虽然我曾靠近她，曾关心她，曾帮助她，她有回应我，有帮助我，有告诉我，你其实很优秀。但是，我们谁也没有戳破那层薄纱，给人留下诗意的神秘。

转眼间要各奔东西，也许是牵挂的力量让我们相逢。在某个傍晚，夕阳西下，树影被夕阳拉得老长老长，我们在校园的路上遇见。“既然遇见了，一起走吧？”她点点头，直直送她到楼下。期间，她告诉我说：“你不回图书馆吗？”

“我不在乎这点时间，倒是你，今生今世都有可能不会再遇见。”

她看着我，我看着她。我瞧见她眼睛深处荡漾着汪汪春水，让我的心都融化了。

如果在以前，我肯定会告诉上海女孩儿，我喜欢你，我愿意为你付出所有。然而，那个时候，我已成熟。也许还不是很成熟，但是我已成熟到，对于看不到希望看不到前途的男人，他没有能力给予对方想要的幸福。
