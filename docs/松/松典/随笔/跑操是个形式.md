# 跑操是个形式

Date: April 17, 2022
album: 象牙塔

那是夏天，夕阳落满校园内外，人影在晚霞深处被拉得老长老长。从康桥吃完饭，我们绕到田径场散步，边走边说，往图书馆方向走去。不知是谁引出话题，这个话题是关于跑操的。而且，话题出来，群情激奋。

正当我想要抨击几句，想要各种吐槽，有朋友抢先说：“他妈的，跑操有啥用？！”是啊，跑操有啥用，跑操跟体育课直接挂钩。一学期下来，跑操次数没有达到规定次数，体育成绩再出色都要挂科。我现在还记得当时，我说：“我觉得，提出跑操那些人就是傻子，搞这种形式主义！”是啊，这种跑操是没有用处的，费力不讨好。

如果我没有记错，早上有个把小时，傍晚有个把小时，每周最多只能跑四次。跑操的孩子，从寝室出来，路过康桥，从东边走进田径场，直直走到田径场西边出口。西边出口，那儿有三四位老师，手里操着印章，等待着学生递上跑操小本本。那年跑操，我没有傍晚去，全在早上。有时，忽然醒来，迷迷糊糊的，没洗脸没吃早餐，搂起裤子，先去盖章。盖完章，回到寝室继续睡觉。按理说，那个时间段，不是很早，早起早睡完全可以。遗憾的是，我总想睡觉。

我总想睡觉，像是在弥补中学末尾欠缺的瞌睡。老实说，我这种还算好的。只是，在上午刚起床来那会儿，有点儿想要多睡会儿。有些人，或者说多数人，如果没有课，他们要睡到大中午，睡到大下午。晚上出去活动，白天继续睡觉。于是，在这种的大背景下，有些人临近期末，才不情不愿地去跑操盖章。加上体育成绩，多数不用挂科。

当年，我们的体育课是乒乓球，授课老师是体育中心主任。每次上课前，他都要给我们讲课，像思修老师各种道德说教。那时，我还是孩子，傻傻相信着这位老师。我觉得，他说得太有道理，恨不得用本子记下语录。

两个学期下来，我确实有进步，而且，我认识到许多朋友。那些朋友，现在有些还有联系，有些已经相忘于江湖。关于跑操，我们那组有个人很有意思。他来自少年班，分流进入物理学系，小专业是什么，我不知道。这个人曾跟我说，他妈来自哪所大学，他爸来自哪所大学。毕业后，他要想要出国，想要移民，不愿意跟我们这些下等国民在一起。

那时，我津津有味地倾听着他吹牛。我想，我的眼睛深处肯定是羡慕。这个人打乒乓球，技术确实有，但是，他有点儿懒惰，有点儿油腻。虽然我没有学过，但是，由于性格爱较真，最后他连我都不如。

就是这么个人，跑操数次不够，我曾瞧见他在老师身边软磨硬泡。结果，老师冷冷告诉他：“这是学校的规定。”后来，他理所当然地挂科，理由是跑操次数不够不合格强制挂科。每次遇见他，我都会想起这件事，想笑。

操着小本本到东南门附近跑操，到我们这级结束。后来，体育中心推出跑操APP。我们拿着手机在校园跑操，可能是个性的缘故，我没有到过田径场，而是在校园在梧桐树下在梧桐东道在梧桐西道。我带着耳机拿着手机跑啊跑，跑完后，回到图书馆翻翻书，再回寝室。起初，有系统问题跑操作废，有手机没电关系跑操作废……后来就不存在这些。

我常常在路上跑操，有瞧见骑自行车的，有听说给人代跑的……反正，总有人有办法逃脱。正是应了老话“上有政策，下有对策”。制度再完善，只有是人，总有办法钻空子。比如说，有次，跟女孩子吃饭，她向我吐槽跑操好烦。“如果你愿意，让我帮你跑吧。”于是，我曾给人跑操，跑了二十多次。小心思，其实都懂。

再后来，这个女孩有了男朋友，逐渐消失在我的生活，而我偶尔会触景生情想起她。这件事让我感受到，不是所有付出都有回报的。当然，你只要努力，哪怕没有回报，你都会收获到问心无愧这样的礼物。

长大后回看曾经，跑操确实是形式，但形式的东西是有用的，因为形式就是内容。
