# 艺术家

Date: April 13, 2022
album: 美丽的山村

云痕像几笔勾勒出来的素描。透过玻璃窗，我注视着远方，忽然觉得远方有种忧郁的诗意。

现在的处境实属走独木桥，正走到一半。前面有段距离，后面已走完段路。除了坚持下去，我别无选择。有时，有些儿灰心丧气，有些儿想要回到从前，但是，我没有办法，没有其他选择，并且，我已经不可能回到曾经。

想到这里，我想起，学生时代，某个傍晚，夕阳漫天，我像往常到田径场散步，偶遇即将退休的老教授。这位老教授告诉我说：“前段时间，谁谁谁获得什么国际奖项，那个奖项不说是世界第一，但在他那个领域，他是世界第一！”

“这个人有个地方让我特别钦佩，他在他那个领域做基础研究，孜孜不倦地做了二十多年。他为了写一篇论文要研读一千多篇文献。那个年代，不像现在的教室、实验室都有空调。夏天做实验热得浑身是汗水，他跑到空地用凉水冲自己。孩子，你前面说的是对的，要写东西，首先得积累东西，厚积薄发晚成才能大器！”我的心灵深深受到震撼。

在那瞬间，我感受到某种崇高的东西。后来，我真正理解了它，它是信念。信念就是，即使全世界都不相信你，你都还想把你渴望的事情做成。没有谁可以保证自己一定成功，但成功者往往因为坚守得以创造奇迹。

后来，我重读卡尔维诺，再次接触到这个故事。我相信，这个故事不是卡尔维诺的原创。因为我知道，这个故事，在人们的流传当中已无法追根溯源，又经过我的添枝加叶，像神话没有强有力的证据表明是谁的作品。

很久很久以前，有个国王忽然想要艺术家给自己画出天底下最完美的女人 。于是，他让大臣去把当时最负盛名的艺术家请到宫殿。袒露心迹诉说愿望，国王询问艺术家是否可以满足？艺术家点点头，语气和缓，他告诉国王说：“尊敬的国王，我可以完成这个任务，只是，我有两个条件。”国王心想自己享受人世的尊荣无所不有，肯定可以满足他。

艺术家告诉国王说：“我需要僻静的画室，让我可以心无旁骛地工作。”国王点点头，他满足艺术家这个请求很容易。艺术家继续说：“我还需要整整五年时间。在此期间，不希望被人打扰，包括您，尊敬的殿下。”

从宫殿出来，艺术家住进画室。夜以继日，不分白天黑夜，他总是忙忙碌碌的，没有谁知道他忙什么。春去秋来，寒来暑往，五年期限已到。这天，国王想念起自己曾牵挂的女人，想念起自己曾让艺术家画天底下最完美的女人。于是，很快处理完政务，他在卫兵的保护下，来到位于郊区的画室。这间画室没有皇宫气派，却散发出某种气质，像有神仙居住。

艺术家出门迎接国王，国王走进画室。他坐在高高的椅子上，显示出帝王的威严。他问艺术家：“我要你画出最完美的女人，你画出来了吗？画出来了，就呈现上来吧。”艺术家不慌不忙，慢条斯理地回答国王说：“我还需要五年时间，还需要这间画室。并且，依然不希望有人来打扰我。”国王看到艺术家坚毅的神情自信的目光，再次应允。

光阴荏苒，岁月如梭，期限又到。艺术家居住的地方竟似奇幻仙境。这里，有鸟儿啁啾，有蝴蝶起舞，有万物在生长，有风吹树叶的旋律。国王率领大臣走进画室，只见画室空空荡荡，墙壁干干净净。这位国王属于学者型政治家，他博览群书、学识渊博，有着读书人的气度与修养，他忍住心底丝丝不愉快，问艺术家：“我要你画出天底下最完美的女人，现在总可以告诉我了吧？”艺术家回答国王：“殿下，您想要的，就在此间画室，只需片刻即可。”

国王神色凝重，目光流露出疑惑。艺术家转身用手指着不远处白净的墙壁，他大踏步走上前去。众人注视着艺术家，艺术家的动作娴熟，神情飘逸超然物外，舞动着手里的画笔很快就在白净的墙壁上画出天底下最完美的女人。

天底下最完美的女人，此刻就在白净的墙壁上，它虽然只是壁画，却像妲己勾走了纣王的灵魂。
