# 军训那些事儿

Date: April 16, 2022
album: 象牙塔

刚进大学那会，特别是军训那段时间，我总要在校园迷路。迷路后，我总是要问路。那个阶段，就跟后来刚进社会时极其相似，有点儿迷茫，有点儿苍白，有点儿懵懂。而且，每天穿着迷彩服，穿着军训套装，混迹在人群。

头几天，我按部就班地该干啥就干啥，偶尔忙里偷闲，翻看着几页书籍，或者发发朋友圈抒发情感。如果这些事情都不能做，我就跟周围搭讪说话，仿佛身边有个人，不管男生女生，不管外向内向，我都有话说。那时，我说起话来滔滔不绝源源不断喋喋不休，而且，常常指点江山激扬文字挥斥方遒，又不懂看人眼色，自顾自说下去。

反正，闲着又没有事情，就跟周围人闲侃，没准会有些许收获，有些朋友就是在那时结识的。

我这个人性格不好，个性极其强烈，有些儿任性妄为，有些儿随意冲动，有些儿棱角分明，这种性格天生憧憬自由憧憬无拘无束，有时候固执己见不服管教。所以，军训，对我来说，其实是种折磨，因为它要求我们懂得服从，懂得配合，懂得执行。同样道理，后来有人劝说我当兵，我没有犹豫，直接拒绝，理由是我有着强烈的自我。

多数教官都是附近军训的学长，我们这位教官很年轻，他的风格不像普通军人那样刚烈严格，在北方人里属于文雅随和的。所以，走齐步，走正步，老是顺拐，可以说是屡教不改，搞得教官都不知道该说我啥，搞得他很无语，然后在其他人休息时让我到旁边单练。中学时代，军训期间，我分不清左右手，老是慢半拍，曾被教官训练得哭了。

中学时代，军训期间，我分不清左右手，老是慢半拍，曾被教官训练得哭了。现在，我的心理素质强大，强大到教官说我几句，就忍不住要笑出声音来。尤其是，教官站在我跟前盯着我，不知道出于什么原因，我就是想要笑。

凌晨早起，我们绕着曲江拉练。那天下午，我有发朋友圈：“军训就是累狗、热狗、晒狗、淋狗、溜狗、训狗的过程。大太阳天是热狗、晒狗。五点起床，曲江拉练，是溜狗。老天落小雨淋狗，让我们变成湿狗。哎呀，走走总比站军姿，走起步被教官训要好。我希望时间能快些流。毕竟，我一点也不喜欢军训。”长大后，觉得这，可笑。

我们那级军训，还是很轻松。十来天，仿佛老天爷帮助，竟有三四天下雨，天气都不像往几年酷热。一遇下雨，教官就会领着我们去教室。每次去教室，我要么埋头读书，要么跟身边人窃窃私语，要么跟着学姐学校歌。

有关这位学姐，当时，我们这个连，有位教官仿佛对她有意思。我清晰记得，我们教官劝说那位教官，跑去要微信，然后约出来吃饭看电影什么的。然而，那位教官忸忸怩怩的。我知道，他们肯定没有后文。

再后来，我偶尔遇见这位学姐，这位学姐应该是换了两三次男朋友。爱情对女孩子来说不公平，因为有些女孩子恋爱以后，容光焕发珠圆玉润，惹得路人都要停留；有些女孩子恋爱以后，胭脂俗粉油光满面，只教人不想再看。

这位学姐属于后者，她不认识我，我不认识她。她跟他们，只是我人生路上的过客。
