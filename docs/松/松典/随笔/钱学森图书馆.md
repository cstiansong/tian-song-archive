# 钱学森图书馆

Date: June 1, 2022
album: 象牙塔

树叶哗啦啦落满校园，我如往常走钱学森进图书馆。从寝室到食堂吃早餐，从食堂到图书馆读书，中午再到梧桐苑吃饭，饭后回到图书馆……我想起某些前辈奔走图书馆的情形，幻想若干年后有没有孩子把我当前辈？

某天傍晚，我听完课从主楼回馆，无意间瞥见姑娘。这位姑娘端庄秀丽，清新脱俗的气质，有种南方姑娘别致的淡雅。言语眉间，神似曾痴迷的女孩。只是，再刻骨铭心的姑娘都淡化成青春特有的忧伤。毕竟，人活着不是生离，就是死别。未来很长段时间，我们会觉得压力越来越大。因为我们不仅要选择终身事业，还要选择终身伴侣。

我伤春悲秋地回到北三楼东边，钱馆北三楼的红皮沙发的红皮已经磨损光了。此情此情让我突然想起那些过去的、现在的以及将来的事情。年轻时，心爱的女孩只要认认真真看看我，只要认认真真鼓励我……我就会感动得要记住很久很久。就是因为这样的心，就是因为那样的爱，让我走偏，让我走弯路，让我盖的高楼出现了偏差。如果重来，我会残酷地把那样的心那样的爱扼杀。那时，我就应该知道，我的人生像盖高楼，不能允许超出合理范围的误差。

岁月磨损了我们的棱角，并不是让我们滚到泥巴地里去，而是让我们滚到人生的深处人生的远处。这时候，我们已经明白，爱情不是必需品。这就是成熟，成熟就是懂得沙粒与珍珠的区别，懂得云彩与天空的差异。

我看着残破的红皮沙发，思绪随着我的记忆飘到曾经的某个场景。在某个场景深处，那天中午，我到梧桐二楼吃饭。我吃的时候对坐的在吃，我走的时候对坐还在吃。我转身离开，留下对坐惊讶的眼神，而我继续修行。从梧桐苑出来，走在校园的路上，我心里想的是下午的事情晚上的事情。因为那个时候的我经受了情伤，我还能做的是证明自己。

从北三楼东边到北三楼中间，从北三楼中间到北四楼东边，从北四楼东边到北四楼西边，从北四楼西边到北四楼中间，从北四楼中间到南三楼中间，从南三楼中间到南大厅。在这样的过程当中，我从青涩懵懂的少年成为成熟稳重的青年。起初，我心里总是想要逃避，总是想着可以多走点路，后来觉得走在路上费时间，大多时候就只留南边。

我不是在钱学森图书馆，就是在去钱学森图书馆的路上；我不是在去钱学森图书馆的路上，就是在离开钱学森图书馆的路上。许许多多时间都被我扔进钱学森图书馆，所以，我在钱学森图书馆留下太多痕迹。那些痕迹大概会随着时间的流逝逐渐消亡，但活在我脑海深处的东西永远不会被遗忘，因为它们已经被我用文字永远留存下来。

我觉得大学的本质，就是提供相对自由与宽松的环境，让我们学习想学的东西，在自我追寻的路上试错。钱学森图书馆就给我提供了理想的空间，在这空间深处，我认认真真读书扎扎实实创作，坚韧坚实地成长。我努力继承是为了创新，但我的创新已经不是不是破釜沉舟的冒险，不是置之死地而后生，而是直觉加持理性分析，还有严苛的训练。

许多事情很快很就成为遥远的过去，而我偶尔在缅怀它们的时候总会勉励自己奋勇前进。在钱学森图书馆那些岁月，我觉得自己的内心空虚又寂寞。矛盾的是，我努力压制着这些情感，像训练特种兵以高标准严格要求自己，累死累活地进行着训练。虽然痛苦，却留下宝贵的回忆。所以，感谢钱学森图书馆，感谢没有宵禁的钱学森图书馆。
