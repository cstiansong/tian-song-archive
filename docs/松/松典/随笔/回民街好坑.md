# 回民街好坑

Date: April 21, 2022
album: 象牙塔

回民街好坑

外地人，或者说对西安比较陌生的人，他们总会以为自己到了某些地方，哪怕被宰客都会觉得不是事。这种想法可以理解，毕竟，人活几十年，某些地方就那么几次。在西安，有个地方就属于这种情况。它就在钟楼附近，它有个响亮的名字——回民街。我不知道回民街的来历，亦不想知道它的来自，只是，觉得这个地方不想再去。

尽管我不想再去，前前后后去了四次。这四次，我都是跟朋友去的。我们在拥挤的人群深处穿梭，瞧瞧这儿瞧瞧那儿，吃吃这个吃吃那个。很实诚地说，同样的食物放在回民街，它的价格大概要超出其他地方大半。

后来，我曾思考这个问题。它有房价租金的原因，有宣传营销的原因，还有消费群体的原因。根据坊间传闻，本地人都不怎么到回民街，到回民街那些人来自全国各地。既然可以到西安旅游，他们当然不会吝啬这点钱。

人活着就是这样慢慢识破谎言，慢慢看懂真相。识破谎言后，看懂真相后，又不去点破，甚至说自己没看懂，这就是真正的懂。是啊，没有事情，谁会去哪些地方？说起来，我在那儿有吃到许多美食；说起来，我在那儿看到许多东西。但是，对于我这种人来说，书房才是心灵栖居的乐园。我喜欢安静喜欢恬淡，不喜欢宣泄不喜欢吵闹。

我对吃穿住行等，没有强烈的欲望，不是很感兴趣。所以，条件允许，我还是愿意天天吃食堂，方便舒服又可以节省时间。跟有些人每学期知道食堂吃几顿，我这样的人没有条件像他们那样，但是，我并不觉得有何不可。偶尔招待朋友，我才会去外面那些饭馆餐厅什么的。这样的习惯持续到现在，亦会持续到将来，并延续很久。

实际上，大学那些岁月，我每次到外面吃饭，总会不舒服。虽然我的嘴巴很舒服，但是，我的肠胃忍受不了那么强烈的味道。某些人眼里的美味佳肴，反倒不如我偏爱的小鸡蘑菇，我偏爱的油浇虎皮，我偏爱的紫菜蛋花汤……年纪越长，我觉得自己的饮食偏向清淡，因为清淡可以提供必要的营养，却不会影响到我的学习创作以及日常活动。

那年，有位朋友从天津来西安旅游，我们又到回民街，又到回民街吃各种美食。那时，这位朋友要了铁板鱿鱼，刚吃进嘴里就吐了出来，他解释说没熟还是生的。我想这种事情肯定存在，这里做生意可以说是时间就是金钱。

商家为了追求利益，搞出不合格的东西，在所难免。在我们这个社会，压根不存在什么契约精神。不是店大欺客，就是客大欺店。像回民街这种地方就属于是典型的店大欺客，一个外地人被欺负，除了自认倒霉还有选择吗？

大学中后期，我再没有去回民街。对类似于这样的地方，没有兴趣。现在回想起那个夜晚，那个夜晚我跟朋友住在回民街某间旅馆。傍晚时分，在院子里，有好几位女孩子。我随意任性地跟着她们说话，透过她们的描述去想象世界。不知道怎么回事，我喜欢跟漂亮的女孩子说话，很享受女孩子投来羡慕的目光。也许，这是男孩的天性；也许，这是性格使然。

期间，有个女孩子告诉我说：“回民街好坑！”我点头附和说：“确实好坑！”

天底下，坑人的人从来不说自己坑人。哪怕就是你购买的东西远远高出市场价，你手里毕竟有东西，这怎么能说坑？说起来倒是，比如说我这位朋友吃铁板鱿鱼，虽然是生的，但他当时看出是生的，完全可以要求换熟的啊。
