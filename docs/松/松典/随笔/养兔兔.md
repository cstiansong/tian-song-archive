# 养兔兔

Date: April 2, 2022
album: 美丽的山村

我的人生虽然短暂，却多次养兔兔。现在，兔兔们已不在人间，有关它们的记忆却留在我的脑海，逼迫着我将那些记忆书写出来。我仿佛被某种神秘的力量牵引，有种控制不住自己想要把这篇文章结束的冲动。

小学末尾有天，天气很好，我跟伙伴赶场，在街上看到农户贩卖兔兔。我忍不住买了一只，运送回家。后来，我觉得灰兔兔形单影只，有些儿可怜。特意跑到街上买来黑兔兔。二十多天以后，有长辈告诉我说，这两只兔兔都是母的。我又跑到农户家买来白兔兔，这样断断续续买来三只兔兔。那时，我们家院坝没有修成，兔兔关在房子旁边的猪圈。

每天烧火做饭，剩余时间，我背着背篼到坡上到地里打兔儿草。人们都说，兔兔吃不得露水草，所以，即便早上打来的兔儿草，我都会用心晾晒。我的日子舒适惬意，时光静谧美好，没有人尘世的喧嚣，很快来到寒冬腊月。

寒冬腊月，到小姨家玩七天。回到家，灰兔兔跑没，玩起失踪。有人说，灰兔兔跑进山变成野兔；有人说，灰兔兔跑出猪圈被寻找猎物的雄鹰抓走；有人说，灰兔兔可能是被哪家悖时砍脑壳的狗偷吃……众说纷纭。反正，灰兔兔神秘失踪。随后一段时间，有时，我会惋惜；有时，我会懊悔。我天天寻找灰兔兔，日盼夜盼，灰兔兔都没有回家。

后来，我升入初中，我忙着学习，忙着考试，忙着成长。我家没有人愿意养剩余的黑兔兔，没有人养剩余的白兔兔。腊月末尾，长辈果断宰杀了它们。理由是，兔兔繁殖能力超强，后面有太多兔兔会很麻烦。

跟兔兔再有联系的时候，我从普通学生跻身学霸行列。在乡镇那所学校的重点班，我名列前茅。

那时，我妥妥是班主任的狗腿子，妥妥是同学朋友眼里的乖学生，我非常非常懂事听话，我的性格温驯得像猫咪，我的行为普通得像机器，我的说话就像诗歌这样温柔。我胆小如鼠，从来不惹事，不愿意出风头。当时我们班有个女孩，可能是对我小有情愫，又或者真心把我当朋友。有天傍晚，夕阳西下，晚霞透过窗户照射进来。

她跑到我跟前，温声细语般告诉我说：“我家有只兔子，我妈他们嫌养兔麻烦，我送给你养好吗？”凝神细望，我若有所思。良久，我缓缓点头。随后月考结束当天下午，女孩把兔兔带到教室。我打开麻袋，有只兔兔窝在里面。这只兔兔已经成熟，有点儿肥胖。回寝收拾完，我背着炸药包似的书包回家，手里提着它，搞到最后酸痛好长时间。

运回家，我把它养在院坝西北角的砖棚。初中，由于是学霸有周末补习，每周回家一次，在家待一天。每次回家，我都用心呵护这只兔子。半年以后，重点班重组，送我兔兔的女孩，她成绩不理想分到其他班。再后来，我跟这位女孩走散。现在，我已经想不起她叫什么名字。至于她送给我的兔兔，又过了些许时间，有位堂兄说打平伙。他出一只鸭，我出兔兔，我认真想了想，还是这样比较合适。老实说，那个时候，我觉得自己有点儿对不住朋友。但是，我又没有办法。

光阴似箭，岁月如梭，转眼间来到大学。我压根没有想到，大学期间，我会在家待这么长时间，以至于某天夜晚，我在淘宝购买两只兔兔。一只兔兔是小黄，一只兔兔是小黑，它们是情侣。闲来无事，我就会把它们带到附近放风。

这样的日子断断续续，很快就要返回学校。我把它们养大，它们已经长大，所以，把它们放到猪圈亦不会有什么大问题。回到学校第二个月，我妈电话告诉我说下兔崽……后来，兔兔越来越多，隔段时间就会有兔兔被宰。

学期结束，寒假回家，父母觉得太麻烦，建议我清零。无数灰尘在空气中闪烁着光芒，像浮游生物游动着寻找什么东西。于是，晴空万里这天。我亲自动手解决了它们。有那么瞬间，我看着鲜血汩汩流淌，觉得生命脆弱，不由自主地思索生命的意义。也许，就是我杀伐太重，随后很长时间，运气总是站到对立面，没有照顾我。

从那以后，我决心不再做这类事情。我尊重生命，我尊重生命的选择，我尊重生命的高贵。想吃鱼，就到鱼摊购买；想吃鸡，就到菜市场购买；想吃猪肉，就到猪肉店购买……不管是什么肉，我都不想做屠夫。

万物有灵，有些事情，我已不想触碰。有些人以为我这是自欺欺人。我觉得也是，毕竟我要吃肉。
