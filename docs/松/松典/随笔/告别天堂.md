# 告别天堂

Date: June 5, 2022
album: 象牙塔

月光浅浅地照耀着校园，桐叶闪烁着微弱的光芒，枝叶间不时有声响冒出，显示出夜晚的静谧。

我坐在银杉树下眺望着钱学森图书馆，从刚进大学那会儿回忆到现在。期间，我收获了欢乐，收获了忧伤。我觉得，我没有蹉跎岁月，我只是没有让自己觉得完美。毕竟，我没能得到女孩的认可，没能找到合适的工作。

不知道为什么，这晚的月亮很特别，像是诉说着离别的歌曲，让我不要忧伤。我把目光往上移动，移到星空。我发现，漫天的星斗像眼睛不停地闪动。今夜暗香浮动，皎洁的月亮在澄澈的天空里喷吐着清辉，仿佛无数白色的花朵从星空落下远处的霓虹灯放射出艳丽的光芒，将周围的景物映照得很有情调，这种情调很适合谈恋爱。

此刻，晚风阵阵，有些温暖，有些凉意。我坐在广场边缘的台阶上，凝视着远方，思索着未来。我知道，我心底充斥着复杂的感觉无法准确表达，它像紧张不是紧张，像焦虑不像焦虑，像幸福不像幸福。

年少时刻骨铭心的事情随着时间流逝，老早在岁月深处被黄沙掩埋，掩埋后被侵蚀，侵蚀后随风而逝。扭扭捏捏的文字在微微泛黄的日记里沉睡，直到被后来的我打开，那些画面在脑海里渐渐苏醒，承载记忆的文字被唤醒。往事像扑面而来的风，把我带回深刻的场景，仿佛昨天刚刚经历，如影随形般在眼前铺散开来。

按照时间先后顺序，我从开学典礼回忆到此时此刻。对于漫长的人生来说，这些岁月很短暂，但期间有许许多多事情值得我书写成文章。在此期间，我品尝到爱而不得的悲伤，品尝到友情破裂的失落，品尝到创造奇迹的激动，亦品尝到痛苦和绝望的寂寞。有了些许阅历，我仿佛看透般懂得，不管这些经历是忧伤的，还是欢愉的，它们都将在岁月的流逝里渐渐老去、渐渐死亡，甚至战胜时间。直到许多年以后，它们成为我遥远记忆里风景，足够我追忆逝水年华。

夜已深，我起身再看了看钱学森图书馆。在钱学森图书馆的时光里，我像日本动漫里刻苦修行的孩子，风雨无阻地修炼自己，企图在某个方面证明自己……我轻轻地说了声谢谢，又说了声再见。

乘坐公交，来到客车站。乘坐客车，来到西咸机场。飞机起飞，我看着窗外的风景，看着窗外风景时，我心想：梦想扬帆起航常常伴随着阵阵强烈的刺痛，这种刺痛不在于我们忍受不了，而是告别。

告别天堂，告别老师，告别朋友……对我来说，告别是痛苦的，因为有些人不会再见面。
