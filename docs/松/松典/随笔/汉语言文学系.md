# 汉语言文学系

Date: April 30, 2022
album: 象牙塔

通识教育结束，文科生专业分流，分流到具体专业。刚进大学那会儿，我在思索什么样的专业合适自己。这时，我没有明确自己想要成为讲故事的人。但是，由于长年累月的阅读，我对未来有了隐隐的轮廓。

根据蛛丝马迹，我判断出，即便倒数，我还是可以进入汉语言文学系。有了这样的底线，我对学业比较冷漠，多有种无所谓。我的目标明确，就是要进入汉语言文学。不学那些没用的，我都可以进去，我干嘛还要浪费时间？

于是，课堂，我读书；课后，我读书。偶尔有作业，我就抄作业省时间，反正，我横竖不亏。

无所谓的态度，无所畏惧的洒脱，让我成为文试倒数第三。得知综合排名时，我们在小学期，在小学期计算机课堂，我兴奋地告诉身边的女孩子：“我进中文系啦！而且，我还是倒数第三！这意味着什么？”

“意味着，我没有学，还有两个人没有干赢我。”我想，说这话时，我肯定很嘚瑟，甚至有种骄傲。在我说完这瞬间，不远处有位同学说：“你可能没搞清楚，后面两位是留学生。”这瞬间，我无言以对，有点尴尬。

小学期结束，我没有回家，留在图书馆读书。由于读书太多，我生出许许多多奇怪的想法。比如说，那个时候，我觉得老天爷给我们的时间不多，人这辈子其实很短暂，应该把生命投入到有意义的事情上。虽然不知道将来做什么，但是，我相信着开卷有益，相信着读书总是有用处。也许，我就是从这时想要走自己的路，想要以自己的方式过完大学。

后来，我以潇洒的方式如愿以偿。进入汉语言文学系，我很快发现，那种模式不是我想要的。我的理由倒是很简单，分数对我没有意思，排名对我没有价值。我想要的是博览群书，想要的是学识渊博，想要的是才华横溢，想要的是著作等身。我想要的不简单，属实是浩繁浩大的工程，唯有时间可以换来卓有成效的建设。

想清楚，然后实践，在实践中修正自己。随后很长段时间，我没有听课，没有参与课堂，默默翻阅自己的书籍。有次，我们老师问我：“你书都带来了，怎么不打开？”我解释说：“我觉得读书重要，教材可有可无。”

任何老师遇见我这种学生，没有修为，估计得是要气死，因为那时的我桀骜不驯冥顽不灵。

这位老师说：“许多年后，你会明白，其实自己也就那样。”听到这儿，我心想，我知道许多年后，我也就那样。正是因为那样，我更应该按照自己的方式走下去。从那以后，我彻彻底底堕落了。除了读书，其他事情，我都觉得无所谓。

对老师冷嘲热讽，难以跟周围人愉快相处。我的个性太强烈，强烈到浑身是刺，刺到许多人。

前面那两个学期，我是刺头，刺伤老师同学；我是问题学生，三天两头就冒出问题。反正，有些老师很反感，有些同学很抵触，我自己又孤傲孤僻，让人头痛。后来，经过某个拐点，我们那些老师觉得我变化明显，变得成熟变得优秀。

答辩结束当天傍晚，有老师说：“祝贺你毕业，从此再也不能祸害我们这些老师。”

成熟后，我很感谢汉语言文学，感谢汉语言文学系的老师，感谢汉语言文学系的同学。感谢他们的宽容，感谢他们的善良，感谢他们让我成长，并且，懂得文学的崇高。尽管这个专业是全交大最薄弱的学科，但是，我将用信念去坚守，去让人世间许多人明白，英雄不问出处，草原亦可孕育大树。大树近处给人遮风挡雨，远处是亮丽的风景。

汉语言文学系不能让人博览群书，不能让人才华横溢，但它可以让人有足够的空间修炼文学。
