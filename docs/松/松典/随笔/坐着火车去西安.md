# 坐着火车去西安

Date: April 16, 2022
album: 象牙塔

高三结束那个暑假，我每天忙碌着给学生补习。时间仿佛插上翅膀，很快很快就飞走。

按照日子，我拖着行李箱，到大湾河路边乘坐乡村班车到木黄镇，又从木黄镇坐大巴到铜仁。铜仁火车站，我在售票厅排队，服务人员告诉我：“没有直达西安的火车票，只能中转。有中转到怀化直达西安的，有中转到重庆直达西安的。”

姐姐的话没有说完，我抢答说：“麻烦您给我选中转重庆到西安的。”她在电脑上查找信息，告诉我说：“今晚七点四十发车，从这里到重庆没有坐票，还剩站票。你要买票吗？”我点点头，随后在候车厅等候。

很久以前的某次周末，我们骑自行车出去游玩。在大山深处，偶遇绿皮火车。那时，许是受到书籍的影响，我憧憬着有天乘坐火车前往大学读书。混迹在人群中，不懂的东西就询问周围人。我站在车厢末尾，洗手间附近，倚靠着墙壁。透过玻璃窗，我瞧见万家灯火，直觉告诉我说未来肯定会曲折艰辛，甚至会经受许多挫折与磨难痛。

长夜已尽，黎明即将划破夜空，重庆北站已到。走出站台，走进重庆北站，重庆北站大到让我找不到方向。附近有位中年大叔，我走上前去问他，他告诉我说：“我跟着我，我带你去，但你要给十块钱！”我没有吝啬这点钱。

走进车厢，按照车票，找到自己的位置。我对面的是一位青年大哥，这位青年大哥准备去山西太原。初生牛犊不怕虎，像我这种随意的人，很快就跟他说起话来。期间，我跟他说起，我从铜仁站到重庆。他告诉我说，正常情况，每节车厢都有个空位，这个空位没有售票。让我印象深刻的是，他告诉我说，成绩不重要，重要的是能力！

那时，我告诉他说，我到大学以后，要读好多好多书，要写好多好多文章。穿越秦岭特长隧道时，我深刻感受到，人生就是这样，要忍受漫长的黑暗，才能懂得光明的珍贵。如果没有经过努力，得到的东西终将失去。

在那瞬间，我忽然想起，有人曾说：“未经省察的人生，不值得一过。”懵懵懂懂，我满脸青涩，对未来压根没有明确的目标规划，只是在大量尝试当中隐隐预感到未来的轮廓。西安南站到了，我走出火车站，行李被司机装进后备箱，我被安排坐在副驾。这位司机极其不厚道，他中途把我转移给另外的出租车，另外的出租车司机有加价。

从彩虹桥走进校园，我拖着行李箱，走到梧桐东道上，拐弯往前走。繁茂的梧桐树下，我瞧见白色的不挂女神，瞧见直插云霄的腾飞塔。也许是童心大发，我推着行李箱往前飞奔，像是野狗撒欢，惹得周围行人投以惊奇的目光。

大学末尾，我跟朋友乘坐火车去延安。这时，我对火车已经没有感觉。窗外，风景还是风景；窗内，我已经不是青涩的孩子。回想大学，我觉得自己有许多遗憾。这些遗憾让我伤感，但它们亦是珍贵的回忆。
