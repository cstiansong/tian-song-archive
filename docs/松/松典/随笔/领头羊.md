# 领头羊

Date: March 21, 2022
album: 美丽的山村

深邃的夜晚使我想起许久以前的岁月，在那些岁月尽头，总有现在缺失了的某些东西存在。对于已经缺失了的东西，我的态度比较坦然。毕竟，人生永远不会完美。对于所爱，老天爷总不会同时给予我们。

我每天夜晚都要做梦。每次做梦，我都会陷入长长的无休无止的梦境。在那些梦境深处，我觉得自己经历的所有事情，就像是真真切切发生在现实生活里的。这天清晨，我穿起衣服搂起裤子，把昨夜的记忆留在床上。

我梦见，我回到村里，回到村里的路上。漫山遍野都是大群大群白色的绵羊，我站在路边，看着羊群就像是在看风景。不知怎么回事，我身边忽然有位老人家，这位老人家像学生时代某位老师告诉我说：“你的家庭结构没有问题，你的教育背景没有问题，你的思想观念没有问题……现在遭遇的事情，只能说你的时机还没有成熟。现在，你可以做的事情就是耐心等待，耐心等待的同时不断充实自己！”我觉得，日有所思夜有所梦，不是没有道理的。

风儿从山麓直直吹到山巅，我有看到远处的落叶随着清风飘扬。我问这位老人家：“您可以给我提供建议吗？老人家没有看我，他的目光落在远处，若有所思像极了进行哲学思考的智者。“高筑墙，广积粮，缓称王。这是明太祖朱元璋平定天下的战略，这样的战略同样适用于你读书创作的文学事业。”我认真倾听，心想怎么跟我想的太像了？

“高筑墙是与周围与时代保持距离，节省时间精力用于推进自我建设；广积粮是大浪淘沙沉者为金优胜劣汰胜者为王，把作品像粮食囤积起来厚积薄发精益求精；缓称王是认真读书扎实创作深入人群研究人群，让才华足够横溢让光芒足够耀眼。流水不争先，争的是滔滔不绝源源不断，争的是为有源头活水来。时机成熟，瓜熟蒂落水到渠成。”

这位老人家说得太对，让我无法反驳。我站在他身边，看着远处的羊群。有的羊儿埋头吃草，有的羊儿仰望天空，有的羊儿痴痴出神……我心想，假如把所有错误都犯了，剩余的结果当然正确。这个道理虽然简单，只是，绝大多数的人，尤其是草根，没有试错的资本，或者说试错的成本较高输不起……自然只剩极少数人可以成功。

老人家指着远处山头上，有头正在远离羊群的羊儿说：“你看那头羊儿，它属于领头羊。什么是领头羊？领头羊就是大量尝试、大量犯错，经过直觉与理性分析选择某个方向树立起目标。从此，任凭风吹浪打，不管不顾，只管往前。”

这位老人家就像年轻时候的自己总有说不完的话，我知道他没有说完，只见他自顾自继续说：“近处远处的羊群看着你，看着你朝着某个方向前进。有羊儿会嘲笑你，有羊儿会支持你，有羊儿会……但是，你没有在乎他们，你闷头闷脑地前进。有些羊儿会奇怪？你为什么朝着那个方向坚定不移？这个时候就会有羊儿陆陆续续跟着你。岁月流逝，跟着你的羊群越来越庞大，你感受到责任，感受到压力……但是，你会拥有许多人没有的荣誉、资产以及社会地位。”

也许，这就是梦，梦就像小说，做梦的人就是创作者，梦里的所有就像小说里的所有，虽然以不同角度虽然有不同口吻，但全是自己在说。所以，我认可老人家，就是认可我自己；我否定老人家，就是否定我自己。

此刻，风吹枯草，吹起我的头发，温暖的阳光照在我的身上。我忽然觉得，自己就是领头羊，我不愿意随波逐流，不愿意丧失自我，不愿意走在别人以为顺利的人生路上。我可以适当妥协，甚至作出较大范围的让步，但是，我不会向现实低头。我更不会因为眼前的利益放弃未来。在这瞬间，我的生命发生神奇的改观，缕缕阳光照了进来。

缕缕阳光照了进来，羊群消失了，老人家消失了，小山村消失了，所有的所有都消失了。这个梦醒了，我穿起衣服搂起裤子，把昨夜的记忆留在床上。窗边，我凝视着远处的小山头出神。如果有那种命运，请保佑我。
