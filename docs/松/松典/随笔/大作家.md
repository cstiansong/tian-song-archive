# 大作家

Date: April 15, 2022
album: 美丽的山村

答辩结束当天，我告诉老师说：“也许，我所有事情都会失败，但有件事情，我觉得自己可以。那就是作家！”三位老师眼睛放光地看着我，有位老师缓缓说道：“你这么年轻，不要给自己设限。”我露出微笑，没有回答。

我不否认，有些时候，在有些方面，目的性太强目标太明确，不是什么有益的事情。只因，我的眼睛专注地盯着自我建设的目标，无形当中失去人生另外的可能。当然，我选择的路，如果错了，让老天爷惩罚我吧。

后来，有老师告诉我：“你选择的路，极少数人才会走！你要成为作家，比你成为影视明星，还要艰难。”我微笑着回应这位老师：“像我这个年龄，有人曾问沈从文，为什么要来北京？他的答案是，理想还是要有。”诚如老师所说：“不是写了点东西，就可以自诩为作家，你别看有些人写的小说拍成电影，就以为他们是作家。作家是有标准的！”

那么，我是怎样理解这个词的？什么是作家？拥有些许作品，并且产生些许影响的人；什么是出色的作家？拥有些许出色的作品，并且产生广泛影响的人；什么是伟大的作家？拥有伟大的作品，并且对全人类产生巨大影响的人。他们的作品可能不是畅销书，却在任何时代都有他们的读者群，而他们本身具备特立独行的品质，俗称“异类”。

马尔科姆·格拉德维尔有部作品，叫《异类》。他在书里指出：“人们眼中的天才卓越非凡，并非天资超人，而是付出持续不断的努力。一万个小时的锤炼是任何人从平凡变成世界级大师的必要条件。”如果不是生来就具有绝世的才华，竭尽所能地训练，自然是仅有的选择。这个过程，很难很煎熬。只是，我曾在逆流而上的小船上划行，最后小船无法进行下去，索性跳进文学这片海洋。也许，我会苦苦挣扎最后溺水身亡；也许，我会乘风破浪游泳上岸。

从笛福走进我的童年，莎士比亚卡尔维诺川端康成塞万提斯肖洛霍夫……先后赶来。我渐渐知道，书房是我的乐园，大作家是我的宿命。这种宿命让我想要出与众不同非同凡响的小说。我希望每部重要的作品都带着强烈的自我意识，可以在现当代文学的天堂驰骋。我渴望着将来某天可以成功，为此，甘愿牺牲现在的自由与幸福。我以为，年轻时，最大的智慧，就是坚持着真诚的理想，不计后果不顾所有地投入其中，对自己付出的价值深信不疑。

也许，大作家都是天生的。有些人很努力很努力，最后还是没有理想的结果。不是因为他们意志不够坚定，不是因为他们品德不够高尚，不是因为他们行为不够正派，而是因为他们缺乏点点最致命的东西，那就是天赋异禀。

对于这种东西，我们都无法否认它的存在。遗憾的是，我生性倔强，既作出决定，自然不会想着回到从前。我只能硬着头皮，咬紧牙关，按照内心对世界最真实的理解，去走完极少数人走的道路。如果选择某个词，我会选择“作家”。我相信老师鼓励我的这段话，他说：“将来，你肯定是你们那代人的作家！凭你的坚持，我相信你完全没有问题！”

我坚信，通过艰苦卓绝的努力，没有天赋的人，也可以绽放出光彩。正是这种朴实的想法，让后来包括现在的我，经过某个拐点后，淡泊以明志宁静以致远，不知不觉间已有科研般的沉静心，像是由里到外，完完全全换了个人。我不太会，因为工作这种事情就否定以往的努力，就否定努力所象征的意义。于是，我不再思考坚持的意义，不再询问为何执迷执着，只是抱着求知的欲望跟好奇心，重复中有增量地推进自己。像高尔基在《童年》末尾写的那样：“你不是一枚奖章，我脖子上不是挂你的地方，你到人间混饭吃去吧……”我要在保持自我完整的同时，努力融入社会。

融入的同时，继续阅读形形色色的书籍，汲取形形色色的思想，走形形色色的地方，结识形形色色的朋友……然后，塑造形形色色的文学人物，孕育形形色色的思想观念，创作出形形色色的文学作品。

多少年以后，我会以敏锐的洞察力非凡的毅力，像前辈们写出波澜壮阔的长河小说。
