——冷碗碗著《青春》

男人的梦想往往源自女人，也许那个女人随风而往，随着岁月的风沙埋葬在遥远的记忆深处。可当初许下的梦想还在延续，而你渴望优秀的热血仍在沸腾。人生最为窘迫的时候，除了坚守底线，再不能做其他事情。

我没有想到学生时代末尾，我会以失败者的面目回到生养我的山村。那天夜晚，清冷的月光照了进来，照到书架上，照到床上，照到我忧伤的心上。我想起从前种种，想起从前种种时，我的泪水缓缓。老天爷啊老天爷，你为什么要让我忧伤？为什么要让我日日夜夜忍受煎熬？我已经知道错了，你还要让我经历那么多痛苦那么多绝望？！

我走在窗边，脑袋晕乎乎的，有点儿醉酒的感觉。晚风吹拂着我，我痴痴地注视着远方，我看见月夜朦胧，远处的山峦浮现出隐隐的轮廓。这时候，远方有狗儿对着月亮汪汪，声响忽而很近，忽而很远，像是对着月亮喊叫。

伫立许久，睡意涌上心头，我仰躺在床上思索着某些事情。我知道，我出山的时机还没有到。只要时机没有到，许许多多事情都是没有办法的。人世间，即便没有偶然的因素，所有的现状还是特定时空里必然的结果。我所经历的事情，既有环境，又有时代，还有自身的原因。某种程度，毕业后，我没有顺利工作，其实是宝贵的人生事情。

一个人若是觉察到天命，他会变得从容淡定，变得有条不紊，变得坚韧不拔……即使他像陀螺，像战士，像运动员，他也不觉得高强度训练，有多么艰辛，反而觉得内心特充实，有日新月异的理想，跟激动人心的浪漫。其实，我很想很想外出务工挣钱，但那个阶段，我还不能外出。我要在家里耐心等待，等待的同时，努力充实自己。

我理解到，艺术与人类的情绪紧紧相连，艺术家引导着人类的心理力量，艺术的目的让我们和自身的情绪有所联系，或者激发出新的感受。在某个特定年纪里，我总是闹情绪，总是散步情绪，实际就是觉醒了艺术才华。 艺术才华觉醒以前，心灵必然要日日夜夜忍受煎熬。就是在那些煎熬的岁月深处，我明白，我的青春即将消失殆尽。

现在，我的青春已经消失，我不再是懵懂的少年，我保持着清醒的认识。既不懈追求美好的生活，又不奢望太多。我告诉自己，我已经上了年纪，不再年轻。所以，出人头地建功立业是我活着的意义。至于别人怎么看，我不在乎。如果不能避免回答，我就说：我不想辜负某些人的期待，而且。农村孩子对社会最大的贡献其实是出人头地摆脱贫困。

现实就是现实，现实就是权力是道理，现实就是财富是道理，现实就是地位是道理，现实就是影响是道理，但现实就没有道理是道理的。所以，我感谢曾经的自己。毕竟，曾经的自己足够浪漫。现在，我已梦醒，自然要服从现实。我按照现实的法则重新塑造自我，因而我已经变得自私，变得不愿浪费时间培养敌人，变得不愿浪费时间投资别人的女人。我没有那么伟大没有那么无私，我就是这样自私的人，自私到尽可能让自己成为现实世界的强者。

至于青春岁月遇见的那些人，经历的那些事，到这里，我已经写完，所以，我允许自己遗忘。
