# 结束

Date: March 16, 2022
album: 美丽的山村

岁月流转到这天中午，这部作品暂时告一段落，它将与我的书信前往我曾生活的某个地方。我觉得时机已经成熟，也许写得不好，但是，我相信这部作品的价值与意义。我走出房走向附近快递站，填写完信息，付款走人。

我走出快递站，我站在路过等绿灯。路那头，有小群人亦在等路灯。绿灯闪现，我迎着无数张陌生的人往前。在这瞬间，我想起“这个功夫可好在人前卖弄？假如你见别人有，不要求他？别人见你有，必然求你。你若畏祸，却要传他；若不传他，必然加害，你之性命又不可保。人性就是，看得人好，却不希望别人比自己好。”

是啊，在我们这个国家，即便胸怀大志，也要隐藏起来，也要懂得低调隐忍。对于那些不确定的事情，我们还是要藏在心底，不要宣扬不要高调，以免出现不必要的意外。我走在路上，心头涌荡着某种欢乐，身轻如燕般美妙。

坐到书桌跟前，我回想起，有位朋友曾告诉我说，毕业后，学生时代学的那些东西基本都派不上用处。我幸运的地方在于，我没有调整人生路线。我依旧延续着学生时代的志趣，坚韧坚实地往前推进着自己。有意思的是，由于我的人生志趣，我实际处于不败之地。别人工作的时候就是工作，我工作的时候是工作，又是体验生活。

在我们这个世界，每个人都是独立的个体，他们都有自己的思想，他们都有自己的追求，并不是所有人都可以用世俗的眼光，甚至以世俗的标准去评判。我曾听闻，面对巨大的现实诱惑，他只要往前便可以获得许多人羡慕的东西。然而，他要做出自己的选择，这选择注定他的颠沛流离。但是，如果重来，他想他的选择还是不会改变。

他属于某个群体，那个群体不惜牺牲未来，他们亦要选择自己的人生道路，我就属于那个群体。我清醒地意识到，人生是选择，无数选择造就现状，造就今生所能抵达的极限。只是，此生精力有限，我只能选择某个领域，在那里面坚韧坚实地耕耘。这个世界，没有谁可以保证自己的选择正确，没有谁可以保证自己的事业绝对成功。

再说，人世间大多事情并不是因为成功才坚守，而是因为坚守才成功。在家那些岁月，爹妈在家，我就认真做饭炒菜；爹妈没在家，我吃几根白菜几根面条。中午出去完成步数，其他时间，我都在这儿坐冷板凳，忙忙碌碌的神神秘秘。我不否认，我有学术科研探索真理的潜质，但选择了读书创作，我就要沿着博览群书学识渊博才华横溢著作等身的道路走下去，绝不后悔。我凝神注视着窗外，天边那轮红日，像深秋的蜜糖橘若隐若现，我默默祈祷这部作品顺利降生人间。

我知道，人生前期的作品像摸爬滚打的年轻人，多多少少带有青涩的色彩，多多少少带有不成熟的地方。真正有潜力的作家并非出场就可以惊艳读者，而是随着叙事修为的精进，推进着自我的文学建设。他们的作品就像他们的人生高高低低起起落落，有质量高超的有质量低劣的。但总体上来说，他们用大量作品在岁月的长河深处夯实自己的文学地位。

　　这段旅程结束，我往哪儿都都是上坡路，我大概会是兵来将挡水来土掩的读书人。现在，我的心情渐入佳境，找工作不再局限于专业相关。反正就是找饭碗，从事文学的平台，提供经济基础。把时间精力用于自我的建设，把老天爷的笔训练得完美掌握，至于对象那是老天爷的事情，我只能随缘。所以，青春岁月到这儿因为这部作品落霞帷幕。
