# 美丽的女孩子

Date: April 19, 2022
album: 象牙塔

思源与假山中间的水泥路上，我们擦肩而过；梧桐苑门口不远的地方，我记住了她那张清秀的脸；军训结束的总结大会，我在众多迷彩服里发现了她……或许，我的记忆出现了偏差，其实无所谓，因为我愿意将错就错。

某个傍晚，窗外，夕阳西下，在图书馆北三楼东边，我们偶遇。情急当中，抽出小纸条，刷刷地写：“你好！我是中文系的学生，我觉得你特别有气质，我很想认识你。如果愿意，就在书签的空白处写下你的名字；不愿意，就请你写俩字：谢谢。”我把小纸条递到她手里，她在纸条上说：“我是……但是，我有男朋友啦，所以，可能不太好哦。”

此时，夜晚降临，我的心情大好。如果没有勇气，我们不会相识，这是浪漫的邂逅！

小学期，腾飞塔附近，我们再次邂逅。聊天啊聊天，我们有了正式的接触。那几天是西安的雨季，天空阴沉沉的，偶有淅淅沥沥的小雨。心儿总想着对方，在不经意间总会遇见。所有的巧遇，其实都是宿命般的安排。小学期那段时间，我们总会在那条路上偶遇。有次没有偶遇，她还在微信上发消息告诉我说：“咦，今天没有遇见诶。”

她的皮肤很白，她的气质很美，她的修养很高……她温柔，她聪明，有着独特的灵魂。遗憾的是，我们遇见的时候，她已有男友。有次，我鼓足勇气约她逛书城，想要深入了解。在那以后，我感觉他们感情很深。虽然后面，我有借书给她，有约她图书馆自习。我曾在心底幻想过，他们会分手，还觉得自己下流。毕竟，横刀夺爱不地道。

好几次，我遇见他们，心里隐隐约约觉得……某天夜晚，我鼓足勇气问她。她告诉我说：“男朋友。”

“多换换男朋友，挺好。至少可以明白自己哪种类型的男孩子。”给她发这样的消息，我的心有点泛酸，心想我是不是很白痴？她没有搪塞我说同学朋友什么的，而是很坦诚地告诉我说：“我还以为你会说这么快。”

次日清晨，我们到逸夫科学馆里练乒乓球。她告诉我说：“其实，爱情没有那么复杂，就是日久生情。我不能理解罗密欧与朱丽叶，他们明明没有情感基础，却可以为对方死去。”我知道，但我没告诉她为什么。

我以为，我们的缘分到此结束。就是在这学期末尾，我们在校园的路上偶遇，随后在康桥吃中午饭。于是，饭后，我鼓起勇气问她：“我可以约你逛次书城吗？”我注视着她离开的背影，思绪忽然飘向远方，心底隐隐觉得不对劲。

直觉往往是正确的，它包含着某种命运的力量。这天夜晚，我想起她，就跟她聊天。聊着聊着，我没的对话变得时断时续。“你怎么忽然就消失了？”她告诉我：“和人打电话。”我已经预感到：“你们准备复合？”

心碎的是，很多时候，我们这个世界小得可怜。又是在第二天傍晚，我们在梧桐苑门口遇见。她背着自己的书包，胸前挂着男式黑色书包。这个书包，不用问，我都知道是谁的。“你在这儿干什么？”她告诉我说：“等他，他上厕所。”她说话时的温柔多有种安详，这种安详大概只有女孩子心有所属，并且甘愿奉献与付出时才会有。

走在前往图书馆的路上，抬头注视着晚霞，只觉今夕落寞，忧伤的情绪涌上心头。这个女孩子很好，可是，我们来自不同阶层，来自不同阶层的人，在今天很难很难有结果，只因为我们的先天条件差距太明显。

故事快要结束，在这个故事末尾，她告诉我说：“不要把我写进你的文章，我不希望被别人看。”

然而，我还是只不由自主地把这段经历写到这里，因为我不想忘记，又或者说，我想要把它写出来再忘记。人世间有太多忧伤，有些忧伤就像这样，故事的女主已经远去，她永远不知道自己竟成为别人的执念。她已经把我删掉，今生今世，我们不会再有联系。天底下，只有两个人知道这个女孩子是谁。一个人是我，一个人是这个女孩。

我把她写进我的文章，而她把我像路边的野草，遗忘到岁（月中）。
