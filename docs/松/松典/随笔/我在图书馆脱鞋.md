# 我在图书馆脱鞋

Date: April 18, 2022
album: 象牙塔

某个下午，教室外面，空调轰隆隆运转，教室里面，我们正听着老师讲课。

有位同学转过身来满脸嫌弃地瞧了我几眼。我觉得奇怪，没放在心上，继续听课。课间，这位同学呵斥我：“哎呦呦，我都不知道该说你啥好，你怎么到图书馆脱鞋？！”那瞬间，我的脸像蒸熟了的螃蟹甚至尴尬。

我问他：“你都不去图书馆，你怎么知道我在图书馆脱鞋？”这位同学，不是谁，而是我的室友，可以说知根知底，所以没有那么多遮遮掩掩的。“有人把你挂到表白墙上去了。”这种事情让我觉得有点莫名其妙。

刚进大学那会儿，我天天到图书馆北边。那里有沙发，中午可以倚靠在沙发上睡觉。没有睡觉，我就坐在那间大教室读书创作。有图有真相，照片里，我确实有在图书馆脱鞋。我脱掉凉鞋，把它们堆到旁边。

也许是我太投入，不注意场合，没在意有人注视着我的行为。我不知道我是什么时候把凉鞋脱掉，什么时候光脚板踩在地板上。可能是从小在农村养成的习惯，夏天我总是踩着光脚板，这样舒适凉爽。

当时，我很痛恨那个脑袋被门挤的人，他估计是看不惯我这种随意的行为，随手就把我的照片放到表白墙上。当天下午，我周围有许多人都知道了。他们当中还有人把这件事发在朋友圈，谴责我不讲公德，让我极度尴尬。

次日上午，近代史纲要课前，我大大咧咧地坐在老师身边，跟他说话，说什么交大为什么历史系，说什么听说中文系要被取消啦。反正，我就是话很多很多，仿佛说不完。随后，老师走进教室，有位哥们跑到我跟前，大声嚷嚷着说：“他妈的，你到底有没有公德心啦。你在图书馆脱鞋，还无所谓！”我没有回怼，只是觉得大清早的，我很无辜。

走进教室坐到朋友身边，朋友问我：“你们打起来啦？”我恨恨说道：“他还敢动手？我刚刚应该怼回去，只是没有想起来。”朋友说：“他干嘛骂你？”我告诉朋友说：“他看不惯我在图书馆脱鞋，跑来伸张正义。”

这件事发生在课前，还是近代史纲要这种大课堂课前，那间教室坐着书法艺术雕塑体特以及我们四个文试专业班。给我们讲课的老师在课堂上说：“图书馆是公共场所，以后不要到图书馆脱鞋啊！”当时，我很尴尬。

当天下午，又有老师在课堂上说：“成大事，不要不拘小节！不要因为小事情失去大成功。”

后来，每当我想要在图书馆脱鞋，我就不自由自地想起这件事情。这件事情让我深刻意识到，有时，随意的动作可能给自己带来不必要的麻烦。再后来，我偶尔遇见在图书馆脱鞋的。他们跟我那时何其相似，他们把凉鞋放到旁边打着光脚板。然而，他们没有我的经历，没有像我被弄到表白墙，没有像我被人伸张正义，没有像我被老师在课堂说教。
