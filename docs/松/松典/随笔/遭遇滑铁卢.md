# 遭遇滑铁卢

Date: May 13, 2022
album: 象牙塔

为完成学业，亦为将来就业，我斗胆私聊某负责人，意外成功，抓住了机会。于是，结束考试第三天，我到平行线的青龙寺校区报到，开始整整两个月的实习兼职。这两个月，七七六，上午七点干到傍晚七点，周一干到周六，吃住都在学校，中午可以午睡，夜晚到图书馆学习。五千五百四十元，写文四篇，读书三十八本，这是具体的收获。

因为起得早，有位同事把她的钥匙给我，我每天早早去开门，检查教室……随即给人测温、负责签到（后来，有位家长评论我的朋友圈说，平行线竟然让你这种人去测温！）跟班、听课、换水、监考、整理仓库、协助教学……除了站在讲台给学生讲课，其他事情或多或少都有涉及。期间，小师傅曾说：“你变了！你刚来的时候，让你做点事情！你说我不干！有种小男孩长大的感觉。”我自己都能够感受到，跟什么人都能说得来，做事的效率还很高，可以左右逢源。

在这里，我认识了五位学长，认识其他老师同事。我的嘴皮子贼顺溜，搞得最后“你现在的任务就是闭嘴！”只是，每天傍晚抢食堂，吃着吃着……我的心微微酸涩，我联想到，多年以后，我是个普普通通的语文教师，而那些……别人的成就，衬托出自己的落寞，我怎么证明自己？我失去了爱情，还失去了作为男人的尊严。

实习结束，初次面试结束，我悄悄来到西花园的某个角落静静等待第三次考核。最后，我悄悄来到西花园的某个角落静静等待第三次考核。其他同学初试复试结束就结束，我还有额外的考核：负责人单独面谈。这位老师很坦诚地告诉了我她的想法。她的评价是“总体满意，可造之材”。当时，我天真地相信着他们，相信他们愿意收我。

从两个月兼职，初试复试单独面谈，到最后的岗前培训，前前后后跨越五个月，最终换来遗憾的结果，我忧伤而痛苦。诚然，它确确实实是种宝贵的经历，可没能得到offer，怎么都不能否认，它是我的失败。

我曾私信询问负责人：“可否告诉我为什么？”她告诉我说：“主要是团队合作方面，很多在你的观念可能是比较小的问题，在工作中会产生比较大的影响。项目在成长初期，你也在成长的关键阶段。所以，未必能给你提供宽松的，尽力翱翔的环境，对你会是严格的挫磨；对你的培养，也需要投入很大的挑战，而现在项目需要大家同心锐意，飞速进步，不可能给单个人额外投入太多时间与精力。所以，出于对你和项目整体的综合考虑，这是对双方的艰难抉择。”

负责人继续说：“你的优点和缺点一样突出，尽力发挥优点的同时，一定要适当导引缺点转化为自己的能量，别太急于得到肯定或肯定自己，放平心态，你会进步更快。真心祝你破茧成蝶。推荐你好好读书，同时多做社会实践。”

“这几天，我每晚都会联想到你，这是种心灵感应，你应该有为你的决定犹豫不决。否则，我也不会老联想到你。”负责人回应我：“确实有犹豫。主要是担心对你的心理产生负面影响。这是你蜕变的关键时期。这时，你敏感而渴望认同，自卑又自负，没办法胸有成竹地把外界与自我的关系平衡得那么稳健。许多大小事都会对你产生影响，这影响的好坏在生命里会经历许多，你得沉淀许久才会恍然。所以，挫伤担心你的锐气。现在看到你这段话，我真正有点欣慰。”

初入社会，我遭遇了滑铁卢。这让我深刻理解到，人活着，不是所有的付出都有回报。有时，不仅没有回报，反而会付出某种代价。在这件事里，我失去时间，失去情感。我没有想过自己会落选，更没有想过自己落选的理由。收到通知的瞬间，我的心，挺不是滋味，有种希望落空的感觉。但这种忧伤让我意识到，我只能做力所能及的事。

无法挽回的人事，我只能归咎于命运，归咎于缘分。她曾告诉我：“以你的性格以及在大学那些经历，突破某个拐点，你的进步会很快，一日千里锐不可当！那种进步的速度任何人都无法与你匹敌，而你几乎就是心想事成，干什么都可以成功！所以，我觉得你前途无量。但是，我不知道你的拐点什么时候来临，也许是明天，也许是半年以后，也许是更久。”

拐点需要契机，需要你内在的积累。”最后，我坦然接受了忧伤的结果。再后来，我感谢他们的拒绝。
