# 追姑娘太猛

Date: April 20, 2022
album: 象牙塔

即将远离校园那段时间，我经常遇见她，想要跟她说话，却失去年轻时候的勇气。她的性格温顺，对谁都很随和，很喜欢学习，属于很好相处的女孩。以我的眼光看待，她会是我学生时代喜欢的姑娘里最出色的。

刚进大学那会儿，我时常和她说话，偶尔和她走在一起，时常在校园不经意间偶然遇见。那年寒假，我在图书馆门前遇见她，她问我：“你怎么还不回家？”我告诉她说：“我想再看几天书。”她看着我：“你很喜欢学习。”

那年暑假，我主动约她逛书城。她告诉我说：“我昨天才和朋友去，你找别人吧。”

某个夜晚，她问我：“暑期高数重修是在哪件教室？”我告诉她在哪间教室告诉她什么时候上课。次日清晨，她跑来听课了。她走进教室瞬间，我觉得她很美很美，洋溢着某种神圣的光环。我又是意外又是惊喜，还有种说不出的欢愉。

重修期间，她认真听课，我在旁边埋头看书。岁月静好让我觉得安详、恬淡，诗情画意。我们吃饭聊天，约图书馆。那天夜晚，从图书馆出来，我们到田径场散步。璀璨的星空下，我们谈论着文学，诉说着人生。

那时，她告诉我：“像你这样读下去，很有可能成为作家。”也许那个阶段，我没有明确意识到这种选择，而是在心底深处有着隐隐的轮廓。“男生可能不适合当作家。”她反驳我说：“像陈忠实，像贾平凹，他们都是男生啊。”

可能是我没有意识到，她有着自己的主见，忽然被她这么一说，能言善辩的我反倒无地自容。

重修结束当天上午，我们走在梧桐树间。那个阶段，我不像现在慎言慎行，我大大咧咧随意任性，神经似的说错某句话。我说她有点傻。有关这句话，后来，我跟他们班的人打乒乓球。不知怎地就说到了她。什么？你说她傻？课堂里，她提的问题确实傻乎乎的。那天，我们本来要去食堂吃饭的。她生了闷气，饭都没吃成，而我心情郁闷。本来心里就装不住秘密的我，当晚就写情书表白。没有经历过，不知怎样处理与应对，只是胡乱地表达自己滚烫的心。

也许，爱情的直觉是正确的，但是由于年岁的缘故不懂得表达方式，我只能是伤痕累累。我想不开憋不住，写情书表白。她告诉我说：“你整天想入非非的，再这样，我们连话都没有说的了。”没有亲身经历，大概不会明白我的感受。我控制不住情感，鬼使神差般总要发点思念的文字。再后来，我表白这件事很快传开，只有我自己不知道。那时，我曾天真地以为自己再勇敢点，皮脸再厚点，没准就可以感动对方。只是，遇见的时机不对，再怎么努力都是悲剧。

随后某天傍晚，我们在梧桐苑偶遇。时至今日，我依然清晰记得当年，我看着她，很没有底气地说：“我喜欢你。”那瞬间，我觉得自己心噗通噗通，很快就要跳出心房。我说完那句话，转身就跑。我人生首次当面表白，当面跟女孩说我喜欢你，就这样潦草狼狈地结束。或许对方也曾感动，可感动仅仅是感动。从那以后，我们无可挽回地走远。

很长段时间，我们都会在校园遇见。我很想很想鼓起勇气去把有些事情说清楚。我觉得说清楚以后，大家好聚好散。那是某个平静的夜晚，走出图书馆，我忽然瞧见她，我跟上去想跟她说些话，但是只能远远看着，直到失去机会。

主观方面，我没有想要尾随，但造成客观事实上的尾随，我不否认。跟在人家后面，这确实是尾随，虽然我拒绝承认这是某些人说的那种尾随。这些事情，在某个小群体闹得沸沸扬扬，甚至有些人觉得我是人渣彻彻底底远离我的生活。在特定的年纪，由于自身的不成熟，由于各种原因，不仅没有得到爱情的认可，反而落得酸涩的下场，这是痛苦的。

贫穷可以让男孩发愤图强，屈辱可以让男孩发愤图强，爱情同样可以让男孩发愤图强。在那以后，我夜以继日，疯狂般推进着自我的建设。我想要成名成家，以此证明自己，证明自己给有些人看。原先与世无争的少年就这样裂变，就这样变得对功名利禄有着强烈的欲望，很快就脱变为胸怀大志的人。人性就是这样，在某些方面没有得到满足，我们就会想要在其他方面寻求认可。所以，自卑不是坏事情，条件成熟，有可能转变为梦幻般的力量鞭策自己前行。

随后偏激偏执的言论背后，隐藏着爱而不得而不甘心的痛苦，这种痛苦让我的情绪最大化。所以，在那以后，我正式进入艺术家们必然要经历的阶段，那是在春蚕吐丝孜孜不倦后面的阶段，那个阶段是“作茧自缚”。我不说话还好，我只要说话就要刺伤别人。归根结底，那种狷狂的状态是内在的冲突与外在的环境综合作用的结果。

有天深夜，我拖着疲惫不堪的身体，走到图书馆南门口。窗外阵雨不断，许多人堵在门口，我插空走出图书馆。义无反顾地跑出去，我在雨中奔跑在雨中呼喊，我清晰瞧见水花飞溅。有那么瞬间，我撞见了她，她的背影，我永远不会忘记。从她身边经过那瞬间，我在心底告诉她：“我会很努力很努力让自己优秀，优秀到让你看到，我对你的情感有多么强烈。”

感情的事情永远不是说断就断的。爱情其实很卑贱。只要你曾认真爱过某个女孩，只要那个女孩愿意，哪怕就是远在天涯海角，你都会跑到她跟前；哪怕就是她曾伤害你无数次，她淡淡地对你示以微笑，你都会忘记。

虽然我没有亲口告诉你，我在日记里曾为你哭泣；虽然我没有亲口告诉你，我真的很爱很爱你……悲莫悲兮生别离，乐莫乐兮新相知。成熟以后，我站在夕阳边上，回忆着白痴的青春年少，却不想嘲笑幼稚的自己。

今生今世，我死心塌地地喜欢着这个女孩，却不得不把她遗忘，这是在偿还前世我对她的拖欠。
