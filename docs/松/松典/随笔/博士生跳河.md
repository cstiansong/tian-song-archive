# 博士生跳河

Date: April 21, 2022
album: 象牙塔

刚进大学那年，“博士生跳河”这件事闹得沸沸扬扬。我不是事件的亲历者，但是，这件事已经传进我的耳朵，已经闯进我的生活，让我们这些人议论纷纷。也许有些人会反感我旧事重提，可我还是要表达看法。

博士生跳河，这件事情诞生，死者的女友发文引发全网热议。经过多方调查，导师确实有让他到家里打扫卫生，确实有让他陪同超市购物，确实有让他洗车的情况。前面那些，我还可以理解，我无法理解的是，这位老师为什么要去干预到学生的私生活。具体点说是，她有点事儿婆，去参与学生的爱情，去说学生的女朋友怎样怎样。

博士生选择跳河结束自己的生命，他把自己的痛苦转移到活下去的某些人身上。他这么跳河不仅带走他们家的希望，亦给导师带来无穷无尽的麻烦。现在，我要从旁观者的角度出发，把这种事情当做现象深究原因。

我的朋友曾告诉我说，天底下所有事情都不是无缘无故的。是啊，我觉得博士生跳河这件事不是无缘不顾的，属于是主客观综合因素影响的结果。从内因出发，这位博士生要负主要责任。他出生湖北农村，是家里仅有的大学，可以说是家里的希望，肩负着某种责任，这种责任让他觉得自己如果不能出人头地，就会很失败很失败。

从小学习优异，被学校保护太好，没有经历多少挫折与磨难，这样的结果只有可能让他的心理脆弱，甚至在特定年龄遭遇挫折，从而导致否定自己的人生，否定从前付出的所有努力。比如说，这个阶段，他最大的挫折，其实是学业，他无法在规定年限完成博士论文。实际上，他本身又没有学术探索的能力却在外在环境的逼迫以及主观意愿方面强求超出自身水准以外的东西，自然而然会抑郁，自然而然会让自己的心灵日日夜夜忍受着某种煎熬与折磨。

有人说，他读完本科读完硕士，又发表多少论文，还凭借实力考上博士。不说在探索学术的道路上有多大的学术成就，博士毕业总是可以的吧？很遗憾，考试分数越高不意味着学术能力越强。考试考的是模仿、答题，学术需要的却是继承、坚韧以及创新。而且，做学术，不管是理工科，还是人文社科，我觉得家学渊源很重要的。来自寒门出生草根，从事学术的可能性越来越低，因为今天学术的门槛越来越高。这个角度来说，博士生跳河证明了寒门难出贵子。

说实话，就凭他是农村草根，读完交大硕士已经不容易，博士不在他的能力范围以内。人类痛苦的根源往往是理想跟现实的差距。如果理想太高，又不能及时根据实际调整，当事人的人格极有可能被理想与现实之间的鸿沟撕裂。

等到有天理想已经走投无路，并且斩断自己跟家人跟学校跟社会所有联系，结束生命显得理所当然。

我想，如果死者可以早点意识到自己没有学术探索的能力，大概就不会走到今天。他想要出国，他的学术能力达不到，导师怎么可以放他出国？他想要毕业，他的学术能力达不到，学校怎么可以让他毕业？当然，如果他的性格像我这样极具反抗精神，像我导师让我给他写材料都说自己很忙没时间直接拒绝……我想，跳河那种悲剧就不会诞生。

家庭、学校，甚至导师，其实是外因，最重要的是这位博士内在出现问题。这位博士内因的根源在于学业，无法顺利完成学业像枷锁牢牢锁住了他的精神，让他无法轻易解脱出来。所以，取消导师的研究生招生资格，支付安抚资金，学校的处理公正客观。按照家属的建议开除导师，处罚太严重有失偏颇。毕竟，这位导师只是违背师德。

说到底，自杀终究是自杀。这个世界，自己要让自己倒下，又有谁可以拯救你于患难？
