# 在省城生活

Date: March 25, 2022
album: 美丽的山村

清风徐徐，空气中漂浮着香樟特有的清香。这天中午，阳光无遮无拦地流泻下来。踩着细碎的树影，我的耳边是风吹树叶的沙沙声。 也许是春天已经到来的缘故，我觉得这样的阳光和煦温暖，衬托出某种娴静。

每天中午都会走出房间，走到附近散步，散步的同时磨炼意志。磨炼意志的同时用心观察出现在视野范围以内的事物，源源不断地收集创作可能需要用到的素材。凭借这种目的性极强的方式，我渐渐习惯这个城市。

这个城市的树种不像延安是槐树，不像西安是梧桐，而是常绿阔叶的香樟。通过树种，我们可以看出城市的气质，可以明白到底适不适合自己。当然，我将来肯定会有机会，肯定会到全国各地看看，用心去体验不同城市的气质。与此同时，认真读书，努力学习，寻找机会，收集素材，扎实创作，慢慢把作品的水平弄上去。我这辈子大概就是那么回事了。

我们这个世界总有科学无法解释的事情，我们往往把这类事情归结于天意。有位长辈告诉我，谁谁来这儿，他莫名其妙地脚痛，无法生存下去。回到老家担任村干部，担任村干部的同时在县城弄快递点。我说这是命运。

类似的事情其实很简单，它们像摆在你跟前的饭菜，你能不能吃完，心底总是有感觉的。有时候，这种瞬息的直觉反而要比理性分析以后的结论踏实可靠。所以，很短的时间里，我已经知道，我会生活在省城。

生活在省城，告别天堂远离校园，我接触到真实的社会，这样的社会是创作的源泉。随着时间流逝，我渐渐走出某个幻想的世界，恍然觉得社会像网络世界，人群的素质鱼龙混杂、泥沙俱下，不能及时便是善恶、是非，其实很正常。实际上，刚出社会不懂江湖的规矩，不是什么原则性问题，偶尔犯错，可以原谅。毕竟，人是错误中成长起来的。

这个阶段，我的心像明镜，时不时打扫干净即可，不会像曾经，盲目急躁，大跃进。至于为什么要生活在省城？我曾告诉某位老师说：“我想在城市书写农村的故事”。这些故事都是虚构的，但故事深处有许许多多真实的经历。有些情节有些情感有些语句有些原话，它们通通是我的记忆。寻根溯源说到底，这是现实与理想的折中结果。

现在，我就在省城，我就在省城某间屋子，我敲击着键盘，键盘声噼里啪啦像潮水起伏。就是这种时候，我仿佛站在高处看尽看全那片天空与土地，看懂生活在天空下土地上的人们。我的爷爷出卖体力种地，只能解决温饱问题，永远在农村生老病死；我的父亲拥有修房技能，可以提供稳定环境，许多年沿着河流修房造屋……

生活单调、冰冷、寂寞，甚至觉得孤独，但是长年累月这样坚持，我可以清晰感受到自身在成长、在成熟，缓慢却坚韧地孕育出人生另外的可能性。余生漫漫，我会控制自己支配自己，努力把这种可能性变成客观存在。

此刻，夕阳斜斜照在西边，金黄色的霞光像薄纱倾洒在省城上空，只觉心情颇好。
