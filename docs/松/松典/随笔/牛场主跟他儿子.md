# 牛场主跟他儿子

Date: April 10, 2022
album: 美丽的山村

这天清晨，明媚的阳光散落在油绿的白杨树叶上，蔚蓝色的天空停泊着四五朵白云。我沉浸在华丽的辞藻当中。许是由于久坐的缘故，有些儿疲倦。我打开手机，微信消息跳进视野，朋友喊我去见个小学同学。

他是牛场主家的儿子，他们的牛场就在我们村寨附近。走进牛场，我特意数了数，有二十多头牛。按照现行市场，这些牛价值三十多万。这位哥领着我们在牛场转了转。聊天内容有许多，在这儿不用赘述。期间，他告诉我说：“这里面最大的牛就是你爷爷卖给我们的。”这件事，我有所耳闻，只能怪我爷爷喝醉了酒，极其愚蠢。

走出牛场，站在路边，朋友继续跟牛场主家的儿子交谈。我在他们身边眺望远处的风景。

远处，和缓的山坡上，有母牛哞哞的呼唤声，像是母牛在寻找不在跟前的牛娃。许多年前，我曾在山上放牛。那时，我梦想着将来有天，我可以承包大片土地，在土地上放羊大量黄牛。每当想要吃牛肉，我就宰牛解馋。

岁月的年轮转到今天，我才想起少年的自己是多么多么天真。我想起少年的自己是多么多么天真的时候，还想起曾经有位前辈告诉我说：“这个社会对农村娃是不公平的。有许多从农村出来的，靠努力改变了命运，但他们又因从小生活在农村，对金钱抵制不住诱惑，最后又很失败。”我站在小路边上倾听着牛场主家的儿子诉说故事。

脑袋在想，这个村庄坐落在大山深处，这里没有河流，没有交通干线，没有特色产业……领取国家工资的人寥寥无几。大多数是打工人，没有待遇，只有勤劳的双手，与运程。出生在这儿的孩子，他们起点低，没有经济实力，没有人脉背景，没有信息渠道，没有良好的教育环境，他们的脑袋又不是很灵光。哪怕他们诚实、勤劳、善良……但因为没有考进大学，他们几乎没有通往中层社会甚至上层社会的可能。在这里，考大学，不是唯一出路，却是最佳选择。

遗憾的是，考大学，对他们来说，特别特别难。或许没有偶然的因素，事情最终的结局可能改观，可那都是特定环境里必然的结果。究其根本，艰难的始作俑者，不是别人，而是他们自己。小时候，我跟牛场主家的儿子没什么两样。我也打牌、玩跳棋、打泥巴炮、欺负女生、偷鱼偷西瓜、偷偷下河洗澡……小伙伴们干过的事情，我没有例外。可如今，众多小伙伴里，我毫不谦虚地说，如今的我很特别，我跟他们有着明显的区别，那就是我有作为作家的宿命。

牛场主的儿子说：“我这辈子哪儿都不能去，只能在这里养牛啦。”我没有否定，没有肯定，心想，此前，有次我奶奶把外套遗忘在茶土。天色朦胧，我乘着夜色微茫，去茶土找回外套。抄近道，路过养牛场，大狼狗冲出来，把我吓得不注意就滚落到下边土头。从土头爬起来，我才想起，我跟牛场主家的儿子是老朋友。他继续说：“跟那些外出打工的比起来，养牛这个要自由许多。”吃的东西可能也会好许多，说着指着附近那些鸡鸭，示意我们隔段时间就可以宰几只。

像这样的人包括我自己有许多，如果询问我们，现在的人生是我们渴望的吗？我们会告诉你，还有其他办法吗？没有其他办法。人生是选择，又是没有选择的选择。高加林以失败者的面目重回农村，刘巧珍已出嫁，他还有其他办法吗？

时间流走，许多刻在骨头上铭记在心头的事情，像断线风筝远去，不知飘落何方。不说是小学同学，就是大学同学，这些关系要有就有，要没要就没有，除了强有力的关系，其他都是负资产。想到这儿，我示意朋友该走了。

在那以后，牛场主家的儿子结婚，我有去参加。当天傍晚，吃了顿正席，就没有去他们家了。半年后，再次遇见，太阳快掉进山里头，天边有深沉的晚霞，又有几只雄鹰在空中打转。飞来飞去，给落日增添雄壮感。

我在马路边散步，有辆白色女式摩托在身边停下。原来是牛厂主家的儿子，他身后是老婆，他老婆怀里是他们的孩子。几句客气话说完，我招呼他们慢点开路上注意安全。两个多月后，饭桌上，我妈说，牛场主家的儿子被警察抓走。我忍住没有说什么话，随后缓缓说道：“这能怪谁？说到底，还是当爹当妈的小时候没有教育好。”

是啊，在今天，农村依然有许多父母认为，给孩子吃给孩子喝，就是教育他们。等到有天以高昂的代价换来粗浅的认识，又有谁会同情他们？牛场主的儿子因涉嫌参与团伙诈骗被警察带走，这样的消息很快传遍周围。

议论像雪花纷纷扬扬，只是，没有人会当面说。毕竟，事情没搞清楚，事情跟自己没有关系，事情……谁会因为臭嘴巴得罪谁，进而跟自己过不去？再说，他们还要在那片天空下那片土地上生活，后面少不了互相帮助。越是落后的地方，消息越是像纸里的火焰很难包住。这些地方，哪家鸡上午丢了，下午就传遍全村，很快成为谈资。

在那以后，牛场的事情落到牛场主身上，这位上了年纪的牛场主起早贪黑。许是因为白天太热，夜深人静，机器切割草料的声音从那边传来。我放下手里的书注视着窗外漆黑的夜，思索着为什么到底为什么而活？有的人生来带着光环顺风顺水，有的人生来命途多舛流离坎坷，前者的人生充满鲜花与掌声，后者的人生全是泥泞与污浊。

再后来，有天傍晚，我在路上散步偶尔牛场主，简单招呼几句。也许是因为成熟，因为成熟以后，我不愿意评论谁，不愿意诋毁谁。我懂得尊重生命，懂得尊重男人，懂得尊重女性……其实，大家都不容易。

我转身注视着牛场主，他已经走到很多很多的地方，远远看去像极了黑色的小圆点，即将消失在路拐处。也许是因为此情此景深深触动我的神经，不知觉间，我竟生出似曾相识的熟悉感，觉得有点儿落寞孤寂。

是啊，枯藤老树昏鸦，小桥流水人家。古道西风瘦马，夕阳下西，断肠人在天涯。
