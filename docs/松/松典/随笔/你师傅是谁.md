# 你师傅是谁

Date: June 5, 2022
album: 象牙塔

天底下，有些话，如果没有及时诉说，以后或许没有机会；有些心意，如果没有及时表达，以后或许失去意义。有天夜晚，我私信我师傅告诉他：“我给您精心准备了礼物，以表徒儿对您的感激之情。”次日清晨，我把盆栽、书籍送到他办公室，跟随他前往饮水思源那儿合影做纪念。期间，这位老师说了许许多多让我受益匪浅。

那年那天正在秋初，太阳高高地挂在天边，没有任何动静，仿佛被人固定在那里。而且，阳光明媚而热烈，湛蓝的天空仿佛比夏季高很多。我坐在朋友身边，我的朋友认真做笔记，而我埋头读书。在埋头读书的同时，不停抖腿。许是不停抖腿的行为被老师发现，他让我起来回答问题，然后旁敲侧击地告诉我学习的时候要专注。

彼时，我刚进大学，属实是桀骜的少年。我觉得这老师讲课没有水平，听他课是浪费时间。但是，我憧憬着进入中文系，憧憬着进入中文系学习我所热爱的文学。即便他讲课没有水平，我还是厚颜无耻地跑去要微信。

忘记了是谁把我拉进影视鉴赏群。在这个群里，我高调发言观点新颖，虽然没有哗众取宠的意思，但已经造成哗众取宠的事实。那个阶段，在许多课堂上我都是刺头。许是担心被封群，他把我踢了。

因为这门课，有许多热爱电影的学生围绕在他身边，而他想要借此机会成立电影协会。有段时间，我积极参与他筹办的系列活动，还想下学期选这门课。然而，在参与的过程中，我觉得有些事情不是他认为的那样，也许他认为的那样在他的认知里是正确的。但是，那些活动，对我来说没有用。不仅没有用，还要浪费我的时间。所以我选择退出。

再后来某个暑假，他让我去看看他们电影协会办公的地方……我依然觉得没有用。说到底，就是社团没有用。女生多的社团，我鼓励男生多多参加，很有可能找到女朋友。找到女朋友以后，就可以退出社团了。

进入中文系后，有他的两门课。西方文论跟影视鉴赏，前者是枯燥的文论，后者是费时的影视。虽然成熟许多，但有把握不学，还能通过，我依然不会浪费时间。这两门，我直接私信他不要挂。与此同时，我选了他当我的导师，亦即我师傅。我的师傅姚明今每次说话含蓄话里带话，往往高谈阔论，仿佛什么都讲了又什么都没有讲。

有次，我们坐在路边，他告诉了我许多事情。比如说，他年轻的时候，到浙江那边实习。实习期间，跟带队老师吵架，直接导致他寄人篱下大半年。实习结束，他到中学教书，但他不懂世故不懂人性，到处嚷嚷着要去考研。校长知道后找他谈话，告诉他说：“你把这学期教完，就去考研吧！我看你不像是中学老师，像大学老师。”就这样，我师傅好不容易找到工作，又被人家解聘。被迫考研，考进西北大学。西北大学毕业，考进西安交大。哲学博士毕业后，留校任教。

相对于当初那些顺风顺水进入中学教书的人，这个跟领导吵架导致寄人篱下，这个嚷嚷着考研的陕师大毕业生已经是副教授、硕导。因缘际会，遇见了同样桀骜的我。只是，我更加桀骜，更加刺头，更加瞧不起，更加唯我独尊。

有次，他把我喊到办公室，给我讲了许多道理。他看着我说：“你的人生会波澜壮阔，不说是波澜壮阔，也会离奇曲折相当精彩。”后来，我渐渐明白了他为什么这样说，因为性格决定命运，命运早已安排。

临走，他请我吃烧烤。期间，他说到“有人都问到创新港那边去了，他们问，你师傅是谁？”

我师傅是姚明今，指导我完成毕业论文的导师，虽然没有指导多少，但我毕业了就认了。
