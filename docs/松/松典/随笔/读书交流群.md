# 读书交流群

Date: April 25, 2022
album: 象牙塔

读书交流群

小学期结束，我采纳朋友们的建议，组建读书交流群。此前，我在各种课程群高调发言，惹来有些人的欣赏，惹来有些人的抵触。随着时间的流逝，有着相同兴趣的人越来越多，他们多次建议我组建读书交流群。

于是，小学期某个傍晚，我把某些老师同学拉进群，在极短时间内完成建群。从那时到现在，这个读书群依然存在。我相信，今后很长段时间，它都会存在。并且，它会随着我个人越来越成熟而越来越规范。

建群次日傍晚，有位老师遇见我，他伸出大拇指告诉我说：“你真牛，一下子建这么大的群！”那时，我没有意识到许多问题，只是单纯想把这件事弄好。在建群那个暑假，读书交流群涌进大量高年级的学生，这些高年级学生里面有许多怀揣着恶意而来。起先，我们还能正常交流。后来演变成争论、吵架、互怼，甚至有出现谩骂。

那个阶段，我不是懂得忍耐的人，惹急了我，看不惯的踢走，恶意复杂的踢走……为此，我动用群主权力设置进群条件，实行连坐，拉人者负连带责任，一律踢走。我没有思考，为什么这个群，我越是踢人，进群的人就越多。

大概就是那些被我踢走的人心底不服气，不服我为什么可以当群主，不服我为什么要把他们剔除交流群。这些人进群以后各种钓我，引诱我发表偏激言论，进而投诉我投诉读书群。读书群封了，足够有些人激动很长段时间，甚至比他们赚了几十万还要激动！然而，他们不服气，我亦不不服气。在这种冲突当中，闹得全校沸沸扬扬。

有次，我到食堂吃饭，有人高声喧哗，他在重复着我说的话。“书呆子的呆，不在于读书多，而在与读书少。”这位同学边说边嘲笑我是傻子。然而，他不知道，他嘲笑的傻子就在他们身边。“即使肄业，我也会是全交大最强文科生。”这些人，我不认识，但是，他们知道我的存在，知道我的存在，就是因为读书交流群的存在。

那段时间，那些人都觉得他们聪明，觉得他们成功，仿佛我就是全交大最烂最差劲的学生。但是，我没有放弃自己，不仅没有放弃，反而固执己见地走下去。现在想来，我的个性强烈，有点儿固执己见，不是什么坏事。因为我就靠着这种东西走到今天，虽然这个阶段颇为悲观，但是，希望尚在人间，理想的火花还没有熄灭，就可以。

每次封群，只要我知道被封，我都会重建。所以，自始至终都存在读书群的人，到今天已经寥若星辰。对于那些已经远去的朋友，我表示感谢我们的相逢；对于那些还在身边的朋友，我表示珍惜我们的缘分。

时至今日，读书交流群已是第五代。与此同时，读书交流群扣扣版读书小天地诞生很长段时间。

这个世界就是这样奇怪，有些人越骂你，反而会把你骂出名。你的出名，不管是遗臭万年，还是流芳百世，你总会在这样的过程当中收获原本静谧岁月所匮乏的利益。因为在中国你是谁不重要，重要的是，你成为了谁。

如果说搞破坏的黑子有什么用处，那就是他们在黑我的同时顺带宣传我。谣言多了以后，有些人就会好奇，就会好奇我到底是怎么样的人？所以，有着自己思考的人，他们总是想要认识我。在他们想要认识我想要理解真相的同时，我认识到许许多多的朋友。这些朋友，时机成熟，他们总是出于各种理由伸出援助之手，让我坚持走下去。

就像后来有朋友告诉我说：“其实，你拥有的资源远远高于普通学生。只是，你不懂去利用你的微信群。老实说，你的知名度因为你的言论在高于我们两个年级低于我们两个年级之间相当出名，你不懂得利用这种东西。”

如果不是这位朋友提醒我，如果不是这位朋友提醒我说：“人在欲望中容易迷失自己，这时候，你可以反过来利用他们，利用他们达成自己的目标。”是啊，人还能比人聪明？我猜想，人永远不会比人聪明。
