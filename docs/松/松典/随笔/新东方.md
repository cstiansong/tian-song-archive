# 新东方

Date: June 12, 2022
album: 象牙塔

许多年前，我曾是中学生，我曾在操场边读完《在绝望中寻找希望》。那时候，我在想，如果将来某天，我可以见到俞敏洪该有多好。没想到，刚进大学那年，我真的见到了俞敏洪，这让我有种心想事成的感觉。

这位来自江苏淮阴农村的孩子，历经人间沧桑，历经人间磨难与挫折，已经属于成功人士。那时候，他在全国做“自强之星”的活动，来到宪梓堂给我们做演讲。少年时期，我初生牛犊，不管人家把座位划分区域，就凭借着某种莽撞冲到前排中间找到合适的位置。也许是命运的安排，我后面坐的是张迈曾书记，书记旁边就是俞敏洪。

彼时，现场有点儿吵，可我听不见，我听见的是心跳的声音，很紧张。深呼吸，转身对着俞敏洪招招手说：“俞老师，你好！我读过你的《在绝望中寻找希望》，很喜欢你！”他看了看我，微笑着说：“谢谢！”

时至今日，我依然可以回忆起些许演讲内容，比如说：“如果你因为没有背景，没有资源，没有人脉，没有……你失去的就是俞敏洪！”又比如说，“来这里的路上，我看过你们的百部经典，我读完了大部分，我觉得，像你们这种学习，这点读书量有点儿少。”再比如说，“最后，我送大家一句话：‘不屈服不买账不放弃，最后还要坦诚！’”

演讲结束，我从宪梓堂走出来，走在回图书馆的路上。读万卷书不如行万里路，行万里路不如阅人无数，阅人无数不如高人指路。话所如此，我却认为关键在读万卷书。我要像俞敏洪在大学在图书馆读很多很多书！

演讲结束，我从宪梓堂走出来，走在回图书馆的路上。读万卷书不如行万里路，行万里路不如阅人无数，阅人无数不如高人指路。话所如此，我却认为关键在读万卷书。我要像俞敏洪在大学在图书馆读很多很多书！

在现实主义横行泛滥的时代，浪漫主义消亡，理想主义暗淡。可我不愿意按部就班随波逐流，不愿循规蹈矩人云亦云，我要追寻理想的道路往前冲。我知道自己输不起，但我有着我的信念，我会用行动证明它。

我把想法告诉朋友，朋友鼓励我：“你想成为怎样的人，你就怎样努力。梦想不是沉重的负担，它是自由飞翔的翅膀。只要你喜欢，不让你觉得痛苦，你就应该勇敢地去尝试。”我勿忘记初心般坚持到告别天堂。

光阴荏苒，岁月如流水。学生时代末尾，我面临着就业的选择。我想从教育行业出发，所以，我的关注点放在教育培训。没能进入前面那家补习机构，我有尝试新东方。新东方确实有新东方的文化，比如说，它可以允许你多次尝试，允许你有自己的风格。与此同时，又不完全像俞敏洪说的那样。毕竟，它是营利机构，而且各地各有特色。

多次尝试教育行业，最后宣布退出中学的舞台。那年初秋，新政策下来，新东方关掉某些部门，辞退大多数新教师。在那些新老师里面，有位是我的同学，我们一起去面试的。有些时候，我们只能用命运解释这种事情。

再后来，我到省城遇见位来自新东方的老师。我告诉她，你应该感谢新东方，因为新东方让你学到许多，很训练你的教学能力。事实上，新东方的老师水准远远高于绝大多数中小学老师。只因为绝大多数中小学老师，仿佛端着铁饭碗，可以高枕无忧般，早早就失去对教育事业的热情。我觉得，新东方是家出色让人尊敬的企业。

其实，新东方自诞生起，让创始人注入了某种宝贵的东西，那就是企业家精神。企业家精神是企业的内在，它很宝贵，可以让企业的基业长青。所以，新东方孕育出许多企业家，还得到许多人的认可，包括我这个人。
