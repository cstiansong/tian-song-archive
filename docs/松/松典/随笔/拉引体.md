# 拉引体

Date: June 11, 2022
album: 象牙塔

随着时间的流逝，我的运动方式，从跑操演变到骑自行车，从骑自行车演变到拉引体，从拉引体演变到散步。在这样的演变当中，有个明显特点，那就是越来越高效省时间。在这些运动方式里，拉引体很有意思。

每天从图书馆出来散步到东南田径场的角落上。在那儿有单杠有双杠，我就拉最高的单杆。有朋友曾劝我说：“拉旁边矮的，最高的比较危险。”然而，我固执己见地以最高的标准训练自己。所以，有次，我像往常到那儿拉杆。有位老人家跑过来跟我说：“小伙子，我在那儿看你，你一蹦就蹦上去啦！”老人家是文革结束后第二批考进西安交大的，在国企干到退休后，每天来这儿锻炼身体。“我还以为你跳不上去，结果你跳上去，还能拉这么多，我很佩服你！”

因为拉引体而认识的人有很多，印象最深的还是两位教授。这两位教授关系很好，都是电信学部能源与动力学院的。由于我这个人比较自律比较准时，他们都为我的这种特质感到惊奇。除此以外，我是个非常会说话的。我说气话时滔滔不绝源源不断喋喋不休，仿佛在荒岛流浪的鲁滨逊积累太多太多话，突然在那瞬间倾泻出来。

倾诉欲望强烈还好，关键是要说话的内容可以让许多人信服，让许多人觉得你肚里有货。这两位老教授觉得我说话格调高原又有格局，所以很多时候，他们愿意跟我说话。有些理工科的，想进来却没有办法，因为他们知识贫瘠。他们可以把专业学得很好，但他们知识的广度远远不如我，所以，他们不能像我可以随时随地随意地跟人交流。

后来某天傍晚，夕阳漫天，我像往常到东南田径场散步，偶遇其中即将退休的老教授，这位老教授把我叫到跟前，现实批判了教学的急功近利，又跟我分享说：“谁谁谁带着他的团队获得什么国际奖项，那个奖项不说是世界第一，但在他那个领域，他就是世界第一！他有个地方让我特别钦佩，这个人在他那个领域做基础研究，孜孜不倦地做了二十多年！他为了写篇文章要研读一千多篇文献！那个年代，不像现在的教室实验室都有空调，夏天做实验热得浑身汗水，他才跑到空地用凉水冲自己……孩子，你前面说的是对的，要写东西，首先得积累东西，厚积薄发晚成才能大器！”

是啊，我就每天进步点点，岁月流逝以后，即便沙子都可以聚集成宝塔，即便腋毛都可以聚集成裘衣，即便铁杵都可以磨练成针……说这些话的时候，我真的不知道自己究竟实在说谎，还是在说真话。说实话，我真的不知道是自己虚构了记忆，还是记忆虚构了我的人生。但我不管这些事情的真实或者虚假，我都愿意相信它们是个美丽动人的故事。

在那些岁月，遇见了朋友遇见了老师，我觉得这些都是我拉引体顺带收获的东西。我每天就在那个时候走到那个地方，认认真真拉完三十个引体，随即回到图书馆。拉引体不流汗，既能锻炼手臂，又训练腹肌，所以，它高效率省时间。有些人对锻炼的理解有偏差，他们总是觉得每次要怎样怎样。其实，每天花费点点时间锻炼，长期坚持足以身强力壮。

有时候，翻书时间太长，我会觉得书页上的黑字像大队大队在爬动，有些儿让我眼花。这时候，我会果断停止阅读或创作。巧合的是，这时往往是傍晚，拉完引体后，我就能够恢复状态。所以，有些人觉得我精力旺盛。

我并不是精力旺盛，而是按照科学的方式长年累月地训练自己。这个世界，绝大多数人，压根轮不到拼搏天赋。有些人说什么没有天赋，只是在为自己的懒惰寻找借口。此刻，有关拉引体的记忆像河水那样潺潺流淌，像落进河流的雪花，很快就要消失不见。所以，我要迅速结束这篇文章，把那些记忆关进文字的牢笼，永远永远不能释放。
