# 珍珠与沙粒（编者名）

Date: March 21, 2022
album: 美丽的山村

这天，我考完某场考试，沿着南明河往回走，河风阵阵，香樟树叶在清风中摇曳欢呼。我的心境像静谧的湖泊，对这些蕴含着希望的行为不抱有期望。没有期望，就不会有希望落空的痛苦。与其说是豁达，不如说是冷漠。

是的，相较从前沉迷诗歌与小说的自己，现在多有种冷漠。曾经那颗热情的心已经被人性的枷锁牢牢锁住。走在街上，有些人给我发传单，我会说谢谢。现在，我看都不想看，压根不接受。世界就是这样，性格再简单的人终究会成熟。现在，我认为，如果有用就是对的，如果没用就是错的。有些虽然正确，因为没有用，就是错的；有些虽然错误，因为有用，就是正确的。说到底，世间种种都是要讲价值的。我没有拖欠他们，没有义务帮助他们……拒绝合情合理。

此刻，我目光空洞步伐缓慢，神思遨游痴痴地想：那些早早懂得人性与利益的人，他们站在山上看风景。只有我这种无知浅薄的人，因为挫磨还在山下寻找原因。这个阶段，我既然找到了原因，就要做出相应地调整。

虽然心底不服气，虽然修为涵养让我不再喜形于色。但，现实就是现实。我已经用事实证明了从前的认知有硬伤。既然已经失败，失败的东西就没有继续存在的价值。岁月的年轮滚到这里，就只能按照现实对自我进行重塑。

春天，蓝天白云，跟前的树叶却像秋天从枝头脱落，我看着这片树叶飘啊飘，伸手把它接到手里。

就在昨天，不经意间，我遇见某个场景。这个场景让我想起，曾经的人生观世界观价值观支离破碎，自我没有形成，遇见不顺心的事情，总要强调客观原因，总要把失败归咎于其他。实际上，我压根不成熟，

人们常说因果因果，世间种种的确有因有果。毕业后，我所遭遇的事情都是学生时代积攒的业。所以，需要许多挫磨去消业，去优化认知。那些岁月里，我不够世俗，我不够捞到，我不善于配合服从，我不懂利用资源实现目标。

我的人生走到今天这个地步，我感觉自己把什么事情都看清楚，只是变得沉默寡言，不愿意说破。

从低谷爬起来的人，从低层爬上来的人，他们的眼里只有自己的目标，只有自己在乎的人事，只有在乎自己的人事。我变得自私冷静，世间的荣辱与我无关，不是因为我淡泊名利，而是我知道它们不属于我；我变得功利实用，摆在跟前的事情，我会用头脑在心底用称去称量，就像商人精准地计算着利弊得失；我变得精致利己，无论是为人处世，还是说话行动，我都会保护自己经营自己，我首先想到的是自己的利益……从某个瞬间起，我就变成生意人。

生意人？是啊，我们把所有事情当成做生意，我们思考所有问题都从人性与利弊出发，就可以看清这个世界。无论交朋友，还是找对象，又或者选人站队，都是在做生意。作为生意人，我不仅会考虑投入产出，而且懂得相互利用实现彼此利用的最大化。那些费力不讨好的事情，可以避免就尽量避免。不要浪费时间不要停留，节省时间专注于自我建设。

也许沙粒跟珍珠同样珍贵，但是，没有人愿意因为沙粒放弃美丽的珍贵；也许云朵跟天空同样美丽，但是，没有人愿意因为云朵放弃浩瀚的天空。什么是成熟？成熟就是选择自己的道路便只顾日夜兼程风雨无阻；成熟就是懂得站在人性与利益的基础上仰望星空；成熟就是知道自我的归属意识到活着的使命不愿意在无关紧要的事情上停留……

也许，沙粒跟珍珠同样漂亮，但是，没有人愿意因为沙粒放弃美丽的珍珠；也许，云朵跟天空同样洁净，但是，没有人愿意因为云朵放弃浩瀚的天空。文学创作者，大浪淘沙沉者为金，优胜劣汰胜者为王，赢家通吃胜者全拿，所以要力争上游，追求卓越，成为行业领域的佼佼者。
