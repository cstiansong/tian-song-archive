# 哲学老哥

Date: April 27, 2022
album: 象牙塔

文试结束那个暑假，因为想清楚自己要学汉语言文学，想清楚自己到底想要什么。我没有回家，而是选择留在图书馆读书。由于日日夜夜长长久久出没于读书馆某些地方，我认识了这位哲学老哥。

哲学老哥来自江西，是管理学院毕业的本科生。本科毕业没有深造进修，而是直接参加工作，在深造某家公司做职员。职员的生活可以让他衣食无忧，但是无法让他满足，无法让他感到快乐，无法让他找到生命的意义。于是，他征得父母同意，辞掉工作，返回校园全力以赴备考研究生。我刚刚认识他的时候，他刚刚还是备考研究生。

那是盛夏，那是盛夏的中午，他坐在我斜对面，我瞧见他跟前摆放着大堆书籍，因为这大堆书籍，我跟他搭讪，我们很快就认识了。我们的生活作息相似，经常去吃饭，经常去散步。这位馆友就像哥哥对待弟弟，给我讲述了许许多多的道理。正是他讲述的些许内容，让我思考自己的大学，甚至思考自己的人生，究竟该何去何从。

这位老哥本科学的是工商管理，考研考的是哲学，因为年纪的缘故，我被他称呼为“哲学老哥”。哲学老哥认真复习认真做题，看着看着就瘦了。这样的过程让我深刻理解了诗句：“为伊消得人憔悴，衣带渐宽终不悔。”

他曾告诉我说：“学科之间本质没有严格的界限，也不应该有严格的界限，只因学术的价值在于探索真理。”那个时候我觉得他是那样博学那样很有深度。在遇见他以前，我觉得只有傻子才会想着学哲学。

后来接触多了以后，我觉得他们不是傻子，傻子是那些以为追求智慧的人是傻子的人。夏天远去，知了在呐喊中死去；秋天远去，桐叶挂在枝头依依不舍。等到冬天雪花纷纷扬扬，笔试很快结束。笔试结束，哲学老哥还在准备复试。如果我告诉大家，哲学老哥考研考的是五道口那所大学，大概绝大多数人的人都会肃然起敬。

人与人之间的区别，其实不在于智力，而在于勇气。也许，有了勇气未必成功。但是，有些人压根就想不到那些地方，压根就没有勇气敢想那些事情，所以，他们人生所能抵达的境界，往往没有前者高远。

哲学老哥的笔试成绩高达四百二十七，但是，复试成绩不理想，最后告以失败。那天夜晚，我陪着哲学老哥回到他的住处，他的住处拥挤摆满书籍。我不知道是灯光，还是他泛起泪水，他的眼睛竟闪烁着光芒。

“我不会考研了，我感谢父母给了我机会，给了我理解与信任。作为回报，我不会考研了。”我看着他，他继续说：“人生有命，坦然接受命运吧。”从他的住处出来，走在回寝的路上。梧桐树间，路灯清冷，我忽然觉得，真正优秀的人，不是曾经拥有多少，而是积极进取努力尝试。也许，他们来自蛋壳的孵化，但他们会像乌龟那样缓慢前行，用时间换取未来用汗水换取希望，用恒心换来卓越的成就。回到寝室，我径直走向座位，很快在信笺纸上奋笔疾书。

次日傍晚，我约了哲学老哥，声称自己要跟他吃饭给他践行。这天傍晚，我们从图书馆出来，偶遇朋友。我把这位朋友叫上，我们仨就在梧桐苑吃饭。饭菜还是食堂那些饭菜，但这顿饭吃得很有意义。随后，我跟我的朋友送哲学老哥走出校园，走出西南门。走出西南门，我把提前准备好的书籍以及夹在书籍里的书信交付给哲学老哥。

哲学老哥握着我的手告诉我说：“你既然有自己的目标，那就勇敢去追求。只是，要适当照顾学校那边，毕业证还是很有意义的。”我点点头，挥挥手，注视着远去的哲学老哥，在心底默默说再见了我的朋友。

前往图书馆的路上，朋友告诉我说：“有其他朋友很正常，但你能认识到这样特别的朋友，还是很有本事的。”我没有否认，没有肯定，而这位朋友亦不在乎我的态度，而是自顾自地说：“我越来越觉得你很有意思。”

这天夜晚，哲学老哥在西安火车站跟前发朋友圈说：“再见了西安，再见了智慧的灵魂，再见了我的朋友们！”
