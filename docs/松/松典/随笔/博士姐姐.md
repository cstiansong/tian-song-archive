# 博士姐姐

Date: June 11, 2022
album: 象牙塔

博士姐姐是我的朋友，她是我在图书馆认识的，至今仍有联系。最近这段时间，我们都还有详细交流。末尾，她祝愿我：“你是很有才华的人，希望早点有伯乐发现你挖掘你。”友情不会因为时空改变而改变的。

那年那秋那个下午，我坐在图书馆某间教室的某个角落，博士姐姐迎面坐在我对面。我这个人翻书久了，就会往前注视很长时间。对面的人，无论男女，大概都会以为我是在看他们。实际上，我是在调节视力，调节视力的同时思索着某些事情。这天，博士姐姐忽然笑了。个人猜测，她大概是以为我在看她，还这样明目张胆。

在这以后，我们互加微信，偶尔到食堂去吃饭。随着时间流逝，博士姐姐越来越欣赏我，越来越佩服我。她曾坦诚地告诉我说：“你教教我，你怎么做到这么自律这么有执行力的？”我告诉她说：“努力读书的同时锻炼身体，自然而然就会变得自律执行力强。”后来，她无意中告诉我：“我听了你的，没事就去拉单杠，肩膀很舒服。”

由于我们经常在某间教室经常面对这面学习，虽然她年长我两岁多虽然我们来自不同专业，但其实，我们属于是同窗。那种寒窗苦读的友情，没有常去图书馆的人永远不会明白。有次，我从图书馆出来，她突然打电话让我到彩虹楼，让我它们搬东西……博士姐姐经常给我吃的，我们经常交流着某些东西。“其实，你真的很优秀！”

博士姐姐曾告诉我：“你有想法有内容，博览群书学识渊博，你为什么不做学术？我觉得你很适合做学术，而且，学术潜力非常大！”我回应她：“我不适合做学术，学术需要思辨，需要逻辑，需要系统性思考，需要探索与创新，而我并不适合在条条款款下做那些论文。”我看着她懵懂的神情继续说：“读书创作是捷径，很快就可以脱颖而出。”

含苞待放的花蕾在草丛深处积蓄着力量，很快就要在清风中舒展开来。认识博士姐姐那会儿，她还是马克思学院的研究生，转眼间，我们都要毕业都要远离校园。博士姐姐准备考西安交通大学的博士，遗憾的是，她没有考上。我安慰她：“你没有考进西安交通大学，可能会考进中山大学。”那时候，她两边都在准备，我觉得可能性很大。

后来有天，她高兴着跑来告诉我：“我考进中山大学啦！”我为她高兴。包括我们的相遇，所有的所有都是命运使然。许多事情，还没有遇见时，其实就已经注定了。我不是宿命论者，但我相信宿命是真实存在的，无法否认。

我给博士姐姐送了毕业礼物，毕业礼物是本学术著作，学术著作前面有我颇为真诚的寄语。那天傍晚，我们从康桥苑出来，沿着梧桐东道往北门走。走到东花园的核桃树下，我注视着越来越倾斜的太阳，西边忽然泛起耀眼的光辉，那是最后的晚霞，似乎在诉说着某个凄婉的故事。我忽然想到某个女孩，想到某个我不得不遗忘的女孩……

我想起我曾告诉博士姐姐：“我的梦想是女人赋予的。如果没有经历爱而不得而不甘心的忧伤，我不会那么奋进。”也许博士姐姐被我对女孩的痴情感动，她眼睛深处有种动容。但是，我知道，我们之间只有友情。

她曾很生气地告诉我：“我不知道为啥，你怎么有点桀骜不驯唯我独尊？！”我没有解释，只是看着她说话：“你的光芒不要太耀眼太刺眼，那样会把许多人的眼睛都刺瞎！”经历了挫伤，我已成熟而内敛。

今天，博士姐姐走在攻读博士的路上，我依然驰骋在文学的草原。希望再相逢，我们都更好。
