# 郑老师

Date: May 5, 2022
album: 象牙塔

时间会在不断的重复当中凝固，然后，像玻璃那样碎了满地。在我们的人生路上，我们总会遇见许许多多的人。在这许许多多人里，总有让我们想起就觉得钦佩，就觉得怀念，就觉得遇见他们是我们的幸运。

郑老师就是这样的人。他曾给我们讲授自然科学发展概论的化学部分，还是理学院的党委书记。这样的领导，对我们这些孩子很宽容，没有某些领导那种官僚作风。他爱讲道理，讲的道理浅显易懂，容易让人接受。

那年，他曾在课堂上告诉我们：“人要有梦想，人不仅要有梦想，还要有实现梦想的信心。我大学的时候，楼下住着的同学后来成了国家主席！当然，那个时候，我们不认识。这告诉我们，年轻人具有无限可能！”我很安静地听着郑老师将故事，郑老师继续说：“我家楼下住着院士，这个院士没有我聪明，老婆也没有我好，但他就是院士！”

教室里，我们这些刚进大学的孩子眼睛放射出疑惑的目光，仿佛在问为什么？“他有梦想的时候，我没有！人要有自己的想法，没有想法的人生只能按部就班！”那时，我固执己见想法诡异行为古怪，独来独往不合群，不跟没有水准的人交流，整天忙忙碌碌神神秘秘地奔走在某些地方。我疲惫，生活单调，洋溢着激情，享受到勤奋自律的幸福。

我可以理解郑老师讲述的故事，因为我几乎就是按照他讲的故事走完大学那段路。那个学期末尾，我在课程群转发某篇文章，有人跳出来骂我。然后，我退群了。没过几天，就在梧桐树下，我偶遇郑老师。

郑老师问我：“你怎么退群了？”我回答老师说：“有些人不喜欢就退了。”

“他们不喜欢，你就退了？人生路上，不要在乎别人怎么说你，你想做什么就做什么！要做自己喜欢做的事情。”要做我喜欢做的事情？回到从前，没有他们这样的人，我大概会畏畏缩缩，早放弃了自己的理想。

那个学期结束，我忙着自己的事情，每天都有清晰的目标规划，展现出某种异常强大的生活作息。我的成长，郑老师看在眼里，因为我老爱发朋友圈，老爱在读书交流群说话。纵观整个大学，郑老师都在群里，他默默关注着我的言行，偶尔给我适当的建议。后来，在某个阳光明媚的清晨，我在校园偶遇郑老师，激动得摇晃着手跟他招呼。

“你能毕业，不是老师们对你的纵容，而是交大对那些爱学习、肯努力的，还是比较宽容的！”是啊，我很感谢宽容的老师，感谢宽容的同学，感谢宽容的学校。郑老师告诉我说：“形式的东西是有用的。”

我点点头，因为我曾猛烈批判形式，猛烈批判形式主义泛滥。他说完这句话继续说：“所以，你找工作才会不顺利！但是，从长远来看，顺风顺水并不是什么好事情。因为顺风顺水不懂得反思，没有反思就没有进步。”说完这些，我斗胆跟郑老师合影。就在崇实书院附近，我们合影了，我觉得这是我的幸运。我很感谢郑老师。

告别郑老师，我走在前往图书馆的路上。梧桐树下，我想起，前面偶遇的学弟，这位学弟告诉我说：“今天，郑老师在课堂上提到你了！”我表示疑惑。学弟告诉我说：“许多年前，我遇见个学生，他叫田松。你们应该像他学习，他每天都有自己想做的事情。”那时，我觉得很是欣慰，因为有许多明眼人还是看出了我属实胸怀大志。

很快走到图书馆门口，我抬头望去。天气晴朗，蔚蓝色的天空停泊着几朵白云，偶有大群大群鸽子像树叶飞过。我想起赫尔曼·黑塞曾在《克林格梭的最后夏天》里这样写道：“我看得很清楚，只有从事精神工作，我才有存在的可能：在一切工作中我首先考虑的是我的文学工作，我只依靠文学生活，既不怕家庭解体，也不考虑手头拮据，甚至把一切置之度外。这项工作倘若不成功，我也就完了。”哪怕最后，我庸庸碌碌，我都会收获问心无愧这样的礼物。

对于他们这样的人，我无法回报，但，我可以在春节私信他们阖家安康身心愉悦。除此以外，那就是，像郑老师曾在课堂教导我们追求点点滴滴的进步，追求点点滴滴进步的同时追求成功追求卓越！

我希望自己越来越优秀越来越成功，进而不至于人生最后辜负了某些人的期望。
