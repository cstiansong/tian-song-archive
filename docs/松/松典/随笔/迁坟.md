# 迁坟

Date: April 6, 2022
album: 美丽的山村

大太阳挂到天空，走过岔路口，往坪场坳方向走。很快进入小路，我的眼前是群山绵延，相对和缓的山坡。在这些山坡上生长着各类野草，开遍各样草花，它们随着山风唱出天籁。周围随处可见大大小小的蝴蝶翩翩起舞。

在这片天空下，在这片土地上，我已生活多年，我的童年少年全在这儿。这里的风，这里的树，这里的草儿，这里的溪流……我闭上眼睛可以在脑海清晰勾勒出它们。顺着小路，我继续往前走，路边有株野山桃，让我想起从前。

那时，这株野山桃只有小孩胳膊般大。春天，它会舒展几朵桃花；夏天，它会有几颗山桃。暑假那会，经常到坡上砍柴，一天来回两三趟。每次经过，我们总会在这儿歇脚。每次歇脚都会掏出鸡儿，对着野山桃尿尿。也许，就是因为童年无忧无虑，想尿野山桃就尿野山桃，想对着群山嚎叫就对群山嚎叫，所以，我的性格深处有种难以驯服的野性。

转眼间，野山桃有大碗那么粗，它的树皮皲裂，各种老皮多得掉渣，树干有许多刀痕，显示被人剔了许多枝桠。长大后，我注视着野山桃，野山桃树除了生长的位置没有改变，外观布满岁月的褶皱，有明显变化。四下无人，我像少年时掏出那个，对着这株历经风雨的野山桃尿。人成熟以后，就会有架子，就爱搞特殊。现在，我尿得高。

野山桃消失在我身后，我沿着山间小路往前。我知道，距离野山桃不远的地方，有座土坟。从来没有人跟我说起，这座土坟里埋葬着谁。后来，无意间听到长辈们摆龙门阵。期间，我根据各种信息推断出它属于谁。

现在，这座土坟被人掏平，土堆里的东西被转移到其他地方。在农村，我们把这种行为命名为“迁坟”。

这座土坟的后代认为这个位置不能让活着的人走运，神不知鬼不觉地把它迁走。我曾以为风水这类东西属于封建迷信，现在以为是传统文化。人世间许许多多的人归根结底是传统文化塑造的，传统文化塑造了这些人。这些人塑造了风俗习惯。千百年来，这些风俗习惯代代传承累积。越是偏远落后的地方，这些东西越是根深蒂固。

我曾听闻父亲评价他说：“不踏踏实实搞，净搞那些没有用的。懒样没有搞高，搞到最后啥都没有搞成。”我没有说话，我知道他还有话说：“现在这个社会，赚钱才是道理！你就是说得再天花乱坠，没有搞到钱，有个啥子用嘛！”

这对夫妻属于长辈，属于看着我长大，而我看着他们老去，看着他们走到与村里人不怎么往来的地步。冷眼旁观，我把他们看得透看清楚。所谓，性格决定命运，他们夫妻的性格都有严重问题。男人好高骛远，有点儿清高；女人小气吝啬，极其不厚道。这样的夫妻，多有两个孩子，结局大概不是这样。遗憾的是，生下女儿，再不能生育。
