# 大雁塔

Date: May 2, 2022
album: 象牙塔

佛塔是从古印度那边传过来的建筑，但是，在传播当中，它汲取了华夏文化，实属于古代劳动人民的智慧结晶。在众多佛塔当中，大雁塔执牛耳独树一帜。有关大雁塔的来历，我曾专门去翻看资料查证。

相传，玄奘法师从天竺带回佛像、舍利以及梵文经典。为装放这些宝物，为翻译经书，玄奘法师在统治者的支持下，在长安城慈恩寺内修建了大雁塔。在随后漫长的历史长河里，这座佛塔不断修缮，逐渐成为今天我们看到模样。刚进大学那会儿，我们进去转悠，心想来都到了，就应该进塔去看看。只是不曾想到爬楼还要交钱。

再后来，我偶有路过，每次路过行色匆匆，总有要忙碌的事情。从心底来说，我觉得掏钱登塔不如到书店转悠买书。想想都是，有些地方的旅游景点全国闻名，主要还是宣传到位。外地人不熟悉，图新鲜想看看。

学生时代末尾，有位朋友来找我，我领着他到大雁塔走走。他想进大雁塔看看，我心想有啥好看的，但是嘴上说：“可以啊，我们上去看看吧。几年见一次，机会难得啊！”进入慈恩寺，工作人员告诉我们正在翻修。

我缓缓舒了口气，大好，天助我。没办法，朋友怪来得不是时候。随后，我带着朋友到附近走走。广场上，我停下来告诉朋友说：“这是玄奘法师。据说，这大雁塔是专门给他修的，让他在里面搞翻译。”说完这句话，我补充说：“可以说，他是古代比较有排场的翻译家。把梵文翻译成文言文，还要专门给他修栋佛塔。”

说了许多，这位朋友下午有其他事情，到附近找家餐厅，我们吃饭。“玄奘法师就是运气好，生在那个时代，生在那个时代，做了件今天看来极其简单的事情，所以，他可以功成名就，可以流芳百世。”朋友被我逗笑，他告诉我说：“你说话有意思！”后来，有人以玄奘法师西行取经为故事起源，创作出伟大的浪漫主义章回体长篇神魔小说。

我曾以为，最没有本事的是唐玄奘，他可以上位，主要还是靠关系，靠背景，靠团队。然而，这种认知比较片面。朋友告诉我说：“《西游记》里面，唐僧团队最有本事的不是孙悟空，不是猪八戒，而是唐玄奘。”我看着朋友，让他把话继续说下去。“为什么是唐玄奘，因为唐玄奘有理想信念。没有唐玄奘，孙悟空猪八戒他们永远是草寇。”

“草寇只能打游击，小打小闹，难以形成大气候。”这位朋友说得很对。他继续说：“唐玄奘每次被抓，孙悟空想办法营救，猪八戒动不动就想走人，沙和尚任劳任怨没本事。唯有唐玄奘不改初心，只管往前走下去。”

是的，领袖人物决定团队的方向。没有方向，再强的团队都没有凝聚力，都没有建设的可能性。想到这儿，我觉得唐玄奘确实很了不起。在古代，他这个人抛弃富贵抛弃功名，不为美色酒色所动，执意要到鸟不生蛋的地方取经。凭这点，人世间许多人已望尘莫及。在这瞬间，我隐隐意识到，古今中外，非偏执狂狂不可以成功，非死脑筋不可以流芳百世。

可能是我的思想比较下流，哪怕是唐玄奘这种人物，我都觉得取经只是他实现人生价值的手段。

回看身后的大雁塔，我想起曾在某本书里看到它的照片。四十多年前，它周围是油绿色的麦地，天空蔚蓝。从远处看，周围是平原，唯有这座土黄色的佛塔直插云霄，只觉得那时候的大雁塔有灵魂，而现在是僵尸。
