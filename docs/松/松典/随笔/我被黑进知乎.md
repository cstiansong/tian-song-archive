# 我被黑进知乎

Date: May 1, 2022
album: 象牙塔

有天下午，老师在讲台上讲课，而我坐在教室后排翻看着卡尔维诺，沉浸在清新俊丽的文学世界。课间，我打开手机。有学妹转发这篇知乎文章给我。她告诉我说：“学校内部的事情，被他们搞到这些地方，很过分。”

说完这句话，她继续说：“我觉得很有必要让当事人知道！”这篇文章是从某个群里面传出来的，而那个群今天依然存在，那些群友多是被我踢走的人。他们当中有些人仿佛对我恨之入骨，总要总想搞事情黑我。这篇知乎文章就是那种意念下的产物。我猜测某些人弄，只是没有确凿证据，亦不想跟他们纠缠。但是，我没有意识到问题的严重性。

没几天，这篇知乎文章疯狂传播，传播到许许多多人手里，甚至惊动到学校领导那儿去。有天夜晚，从图书馆回到寝室，我舍友告诉我说：“我读到了关于你的文章。”我有点痛苦地叹气，他继续说：“你变成了学校内部的网红。”

当天夜晚，我认真读完这篇文章，随后又读完后面那些黑章。我在寻找，真有那些问题，我会改正；没有那些问题，我就笑笑。我仰躺在床，注视着白净的天花板思索着相关问题。问心无愧，我觉得无所谓。

什么透支下辈子的运气考进西交，什么短短四个学期挂科十七门，什么辱骂金禾副教授，什么物化女性……差点把我笑得痛哭流涕，这些人闲得蛋疼，吃饱了撑着没事干。后来，我跟老师走在一起。期间，他说到这篇文章，他觉得有些是事实，有些是无中生有。我问他哪些是事实？他又久久回答不出来。我很无奈这样，可这是现实。

现实是众口铄金。我们可以简单理解为，大家都说白鸽是黑色的，我说白鸽是白色的，但是，大家偏说白鸽是黑色的，我这个说真话的人反倒是说谎。现在想来也是，古今中外，讲道理是最没有用的。大家都是按照自己的理解，看自己想看的事情，听自己想听的事情，读自己想读的文章。我只能说，他们觉得是那样，那就是那样吧。

再后来，有天清晨，我忽然瞧见，有中学同学转发那篇知乎文章，她把那篇文章转发到班群，然后评论说：“那个时候，我们就看到会有今天。”我什么都没有说，潜水很长段时间，我就从退出那个群了。再后来，有天我从图书馆出来到梧桐苑吃饭，有位同学过来告诉我：“你是……你可以联系知乎把那些文字删掉！”

我被黑进知乎，这完全是出乎意料的事情，我压根没有想到，亦想不到。各种课程群说话，有人掂对我，我想要有个说话的地方，于是逐渐读书交流群。进去那些人太多，他们就嘲笑我，就讥讽我，就……久而久之，那些被我踢走的人专门建了某个群在那里面各种诋毁抹黑。说到底，还是他们自己的问题，正经人不会把自己的情绪宣泄给别人的。

后来，两篇浏览量高达十多万的黑文已经找不到。据可靠消息，学校为消除负面影响，把那两篇文章删了。路遥知马力日久见人心，这样的事实已经证明了我的正义。只是，那段时间，从有人加我微信，然后骂我，骂完我，又教导我，教导我，又骂我。就这样反反复复，我的心理素质异常强大，强大到可以孤军奋战。

现在，追根溯源，问题的根源在于，我建了读书交流群，我积累了太多东西，整天在群里挥斥方遒，不是抨击这儿，就是批判那里。可以说是大放厥词狂放不羁，睥睨天下唯我独尊，谁都瞧不起。于是，有些人看不惯，又闲得蛋疼，按照自己的需要写文章骂我。实际上，我跟他们压根不认识压根没有交集，但，他们觉得是在伸张正义。

我在别人不断抹黑不断钓鱼的岁月中逐步成长，逐步成长到今天在某学校臭名昭著。
