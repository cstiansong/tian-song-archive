# 熊猫姑娘

Date: April 21, 2022
album: 象牙塔

送朋友到火车站后，回来的路上，我在路边理发店把头发染黄。有些人无法理解我的行为，就像无法理解我诡异的想法。只是，我这个人从小在别人的白眼里长大。我的内心强大，强大到可以孤军奋战，不在乎别人。

那些岁月，我顶着金黄的头发走在校园内外。从外观来看，像极了某些电视剧里面的小混混，偶尔口吐芬芳爆粗口，有点儿流氓，还经常在课堂怼老师。这样的人，你简直不敢相信，他后面是温文尔雅知书达理的读书人。

有个性的人往往不会被大多数人接受，但是，有个性的人会有喜欢。因为我们这个社会就是提倡个性，但有个性的人注定要承当特立独行的压力。有意思的，强烈张扬的个性极有可能让当事人获得某些原本不属于他们的机会。这些原本不属于他们的机会却有可能改变他们的命运。毋庸说，我就属于这样的人，就因为个性得以迅猛成长。

在某个大课堂，我坐在最中间，坐在最前排，我的课堂上摆放着课外书，认真听着课堂名。我说话直接，没有弯弯绕绕，还是杠精。我经常去纠错，经常去插话，经常打断老师表达自己的想法。如果我发现老师哪儿讲错，我会毫不犹豫地指出来，还以为自己很能。说实话，我这种人有时候得罪了别人，自己还不知道，还把他们当做朋友。

但是，就是因为这样的性格，熊猫姑娘才得以闯进我的人生。她的眼圈常年是黑色的，仿佛老是熬夜老是睡眠不足，我用文学化的语言把她描述为“熊猫姑娘”。现在，我想把自己跟熊猫姑娘之间的经历写出来。也许，故事的主人公已经消失在岁月深处，但是，只要我把它们写出来，我觉得曾经那些经历，就很有意义。

有天课堂结束，我留在教室翻书。熊猫姑娘把我叫到走廊上，她想要跟我打听某些事情。

人生若只如初见，熊猫姑娘告诉我说：“我看到你在课堂上的表现，就想问交大什么时候变得这么开放？竟然有你这种把头发染黄，竟然有你这种踩着拖鞋上课，竟然有你这种课堂上说话这么直接，还不给老师留情面的人。”大大咧咧的人往往没有城府，我就是这样的人。说了好些话，我就加了她微信。当天夜晚，我就把这件事情写出来，还猜测她是来学校收集素材的作者。我把这段文字以文件的形式发给她。她回我说：“我不是什么收集素材的创作者，我只是这段时间比较闲，所以来你们学校听听课。”有许多话，我已经忘记，却还可以记得她这样描述我：“你内心的丰富程度远远超出我的想象。”

是啊，我的内心是敏感又丰富多彩。那时，我比较孤独，我比较爱凑热闹，哪儿热闹，我就去哪里去；哪里有热闹，哪里就有我。这个女孩是本地人，是外校的，她的性格跟我差不多，都属于没有城府与心机的。跟我们这样的人相处，心里想什么，你都可以看出来。某种程度，就是因为这种性格会给许多人安全感，所以才会有许多知心朋友。

曲江书城，就是熊猫姑娘带我去的。期间，她曾告诉我说：“不知道咋回事，你说话就像刺猬，动不动就要扎到我，动不动就让我忍不住想要怼你。”如果那时，我深刻意识到这样的状态意味着什么。我想，我后来就不会那么曲折。它实际就是作茧自缚的前奏，这是自我重构所必然要经历的阶段，本质是在寻找自我的表达方式

在我的人生路上，偶尔非主流，偶尔走偏，还是会有相对圆满的结果，根源在于熊猫姑娘说的：“你是好人，心地不坏，只是个性有点高调张扬。”我确确实实没有多少坏心眼，即便有点小心思，大多数人可以看明白。在曲江书城，她曾这样告诉我：“有次，我爸爸下班后，跑到书店买了笔记本，买了文学作品，然后把优美的句子抄写在本子上。”

我知道她的话没有说完，她继续说：“我希望你永远不要放弃读书，你要坚持读下去。”这句话深深影响了我的读书观，甚至让我以为，读书压根不需要方法，只需要坚持，坚持读下去，总有一天，我们会明白。

秋天以后，我有了新的朋友，有了新的目标，有了新的定位……我跟熊猫姑娘渐行渐远。后来，在南门附近，我们边走边诉说着某些话题。她告诉我说：“这个世界上，没有无缘无故的事情，许多事情都是有理由的。为什么我们会认识吗？”我看着她，等待着她告诉我答案。“你老看着我，我觉得你喜欢我，所以，我们之间才会有故事。”

我痴痴地想，年纪相仿的男孩女孩，不可能对异性没有想法，只是多数时候藏在心底不愿诉说。

说完这句话，她问我：“你是不是很喜欢看人，而且，你看人的时候会盯着看很长时间。”我没有否认，因为我知道她描述的确是事实。关于这件事情，我不能否认，亦不能肯定，因为我真的不记得我有没有盯着她看。

我喜欢看人这倒是事实，我总想着通过看人去揣测他们的内心，然后，汲取我所需要的创作素材。

岁月的年轮倾轧着时间，我有计划性安排着跟某些朋友的道别。由此而来，特意约出熊猫姑娘，送了她书。期间，她看着我说：“你变了变得成熟，变得稳重，而且，你已经很优秀了。”我挤出笑容，这个笑容像她描述那样成熟、稳重。

可能是缘分即将走到尽头，我惊奇地发现，曾经无话不说的朋友已经在找话题在找共同语言。

我站在天桥上目送着熊猫姑娘的远去，回想起从前她领着我从彩虹桥那边前往汇丰商城。期间，她告诉我说：“希望将来你可以很出名，这样，我就可以告诉别人说，谁谁是我的朋友。”想到这儿，露出神经质的笑容。

从回忆深处走出来，我整理思绪，大步向前，我希望那天早点来临，让他们看到我绽放光芒。
