# 我曾抓到野鸡

Date: March 31, 2022
album: 美丽的山村

有次，我们村寨有位伙伴去坡上砍柴。听人说，他爬上树剔树枝，弯刀不小心落到地上，惊起野鸡。野鸡受到惊吓，企图在惊吓中寻找安全，这只野鸡舒展着翅膀噗嗤噗嗤飞向对面山头。小伙伴，下树，寻找弯刀。

他震惊了。在草丛深处，隐隐有大窝野鸡蛋。这些野鸡蛋有的蛋壳褐色，有的蛋壳浅褐色，有的蛋壳洁白色，有的蛋壳淡蓝色……他把野鸡蛋捡到衣兜里运回家，又回到老地方扛柴回家。从那时候起，我就在幻想着哪天走运，我可以在山上捡到大窝野鸡蛋。然而，在农村生活多年，直到我长大，知道我离开农村，我都不曾捡到野鸡蛋。

所幸的是，后来，大概小学那会儿暑假。那天，天清云淡，老天爷很久没有落雨，导致许多水田都快要干裂。

我妈让我跑到半山腰望谁，望望田里的水有没有干涸。沿着田埂走，我忽然瞧见有只野鸡，有只野鸡卧在水田里边的土老坎上，仿佛是在洗澡。它的呼吸急促，不断喘气，热得不行的样子。青天白日的，大概是中暑了。

这只野鸡脸颊绯红，颈部紫绿，羽毛是黑褐色的。我没有多想，像小偷悄无声息地靠近。

我估计它被热昏了头脑，等它瞧见我。瞧见我以后反应过来瞬间准备飞走。只可惜，它瞧见我，我跳起来猛扑过去，我的动作麻利、粗暴，把野鸡摁进稀泥巴。我右手掐住翅膀，左手握住鸡脚，激动着小跑回家。

夕阳跑过西边山头背后，东边的天空漂浮着淡淡的桃红色云彩，有粉红色的晚霞。远远看去，这样的画面是那么美丽，是那么不可形容的美丽，是不可准确形容的美丽！这样的傍晚，我们家多有道干菌子炖野鸡肉。后来很长段时间里，这件事都是我吹牛的资本，我看见人就跟人说，没看见人就跟自己说，自我感觉太好太牛。

时间流逝太多，我已经忘记野鸡肉味。我已经剩下，记得自己曾抓到野鸡，曾宰杀野鸡，曾把宰杀的野鸡烹饪。

现在，我把这种发生在很久很久以前的事情，凭借着少年时期的记忆慢慢写出文字文字，有点儿像是牛在反刍曾经的回忆。亲爱的朋友，在这瞬间，我觉得，有些事情，随着岁月远去，它们没有模糊，而是越来越清晰。

写到这里，我明白，创作者只有懂得什么是生活，懂得什么是人生，才会用笔书写出深刻的文章。

后来，我有进行阅读活动。在阅读活动当中，我偶然翻阅到“野鸡”的来历。封建历史时期，因为有位皇后叫吕雉，皇帝为避讳，他下命令将“野雉”改名为“野鸡”。那以后，人们渐渐习惯叫野鸡，或者说毛鸡。

再后来，我阅读现代汉语词典，曾看到有这样的解释：野鸡，暗妓私娼，尤指沿街拉客的。我觉得，有这种解释很正常，毕竟野鸡缺乏道德感，野鸡崽都是不知道爹的。再后来的后来，偶尔会发现有些野鸡胆肥，跑进村找吃的。

你知道吗？清晨，我可以清晰听到野鸡咯咯的声音；中午，我可以清晰听到野鸡咯咯的声音；傍晚，我可以清晰听到野鸡咯咯的声音……仿佛漫山遍野都是野鸡在呼喊。只是，没有人在乎它们，因为村里没有多少人。
