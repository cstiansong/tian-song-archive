# 好兄弟

Date: June 13, 2022
album: 象牙塔

真正的朋友，值得托付与信任，他是我们经历风雨，依然可以并肩作战的人。

——浪淘沙著《象牙塔·好兄弟》

读书越多，走路越多，认识的人越多，经历的事情越多……我对所谓的人际关系越发冷淡，单纯不想把时间浪费。我以为，朋友并不是越多越好。相反，能够长期与某些人保持联系，其实是很有意义的事情。

有些人的远去固然惋惜，但岁月流逝后，我越发珍惜留在身边的人。那年，在南洋书院背后的足球场上，我们初次相识。性格方面，我们截然相反。如果说我是那种说起话来滔滔不绝源源不断喋喋不休的人，那么，这位朋友就是沉默寡言半天说不出话，甚至觉得说话浪费时间的那种。但是，性格的截然不同其实不影响我们成为好朋友好兄弟。

文试时期结束，这位朋友转到金融，从东区搬到西区，我执意要帮他搬东西，人家拗不过我的固执，索性接受了我的热心。后来，这位朋友约我到西安世博园游玩。游玩期间，突然告诉我，到他家去吃饺子。于是，我就到他们家去吃了饺子。回忆起来，这是很有意义的事情。从那以后，我们成了很好很好的朋友，对待彼此毫无保留。

在随后漫长的岁月，我们有到小雁塔转悠，有到兵马俑转悠，有到拔仙台转悠，有到陕北延安转悠……我们走过许许多多的地方，我们在许许多多的地方留下我们的足迹，留下还有现在觉得温暖的回忆。

坦白说，由于长长久久地保持着联系，当我提起笔书写这篇文章时，我觉得没有多少可以书写的素材。当我觉得没有多少可以书写的素材时，我还是要鼓起勇气努力写下来，因为这位很好很好的朋友，于我是很重要的人。

有朋友曾告诉我：“爱情发生在刹那，友情却是日积月累。”后来，我在翻阅莎士比亚的作品时，遇见相似的表达。我想，哪怕没有这些经历，我还是会明白这种深意，并且，深刻地理解到所有情感都可以进化到亲情。现在，这位朋友依然走在经济金融的道路上，而我还像当年在图书馆某间教室奋笔疾书。搁笔至此，我走到窗边。

窗边那轮明月像张笑脸，仿佛在跟我招呼。海内存知己，天涯若比邻，真正的友情是不会因为距离而消亡的，反到会因为距离让人追忆，让人在追忆中把对方想象得更好。青山不改绿水长流，友谊的小船永远不翻。

“希望将来有天可以在中国文坛看到你的身影！”我也希望让这位朋友看见。
