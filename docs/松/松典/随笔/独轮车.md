# 独轮车

Date: June 12, 2022
album: 象牙塔

经常在图书馆，经常看到这位朋友到看书，很想认识他。有次，他坐我身边，跟我说了几句话，我主动加他微信。从那以后，我们的交往正式开始。有次，在梧桐苑吃饭，他告诉我说：“其实，我认识你。”

“有次，我去你们班旁听，有两个人挤兑你，你还坚持自己，是个有意思的人。”

有天傍晚，我们从图书馆出来。他跟我说：“我告诉我女朋友说，我在图书馆认识了你，然后，她告诉我，你是个‘名人’”我觉得很惊奇，为什么他女朋友都知道我的存在。亦是从那时候起，我明白自己臭名昭著。

这位朋友跟我属实是性情相投，同样有主见同样有个性同样，还同样有点傲气。区别在于，他的条件远远优越于我。因为那时候，他曾告诉我说：“我爸爸就是个大专生，凭借自学进了华为。那个时候，华为还是个五六百人的大厂。”他说完这段话，我随口问了问：“估计现在是中高层，年薪几百万？”朋友说：“华为给钱比较慷慨。”

他告诉我许许多多事情，让我印象最深的是，他说：“取经团队，最有本事的，不是孙悟空，不是猪八戒，不是沙和尚，而是唐僧。虽然唐僧没有本事，但他有理想有信念有追求！没有唐僧，其他人是干不成事情的。”

那时，我已经成熟，不再以批判、否定的眼光看待问题，但我心里依然在想：“唐僧虽然没有本事，但唐僧有背景，他是佛祖的弟子，又有观音菩萨辅助，暗中还有五方揭谛六丁六甲等等。”转念又想，我应该顺着对方的思路走下去。毕竟，没有唐僧这个核心人物，取经团队确实不能修成正果。足见，明心见性有理想有信念有追求很重要。

这位朋友很有个性，他有个性的地方不仅在于他的想法，还在于他的运动方式。“你没有在校园看见过骑独轮车的？”我摇摇头，问他：“独轮车是什么东西？”这样的问题无疑暴露出我没有说谎。这位朋友来自深圳，那是开放与包容的城市，他们家又比较民主。这种环境下，中学毕业，他就写信给父母说自己已经长大，让他们别干预人生。

那学期末尾，寒假以前，我们约在东南田径场，我试了试他的独轮车，确实很难很难。

对我很难很难的独轮车，他却骑着出门远行，骑到华山脚下。亲爱的朋友，试想：没有龙头，没有座椅，没有方向盘，全靠自己平衡的独轮车，被他骑到华山脚下，我忍不住想要说：“牛逼到天上去了。”

随后，我有两次约他出去练车。有次，偶遇他同班同学，我们仨一起去的；有次，我约了位朋友，我们仨一起去的。说起来，我已经很久没有摸车了。但我明白，那两次摸车，对我来说，确实是很宝贵的经历。

后来有位学长问我：“以你的个性肯定认识了许多有想法的朋友吧？”我想起的就是他们这样的人。像他们这样的人，有鲜明的个性，有独立的思考，有明显的自信，有坚守的底线……还意气风发锐气进取。

毕业后，他去了荷兰留学。我羡慕他们这些可以到国外体验到异域文化的朋友。
