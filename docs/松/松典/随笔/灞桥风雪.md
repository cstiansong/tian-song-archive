# 灞桥风雪

Date: May 1, 2022
album: 象牙塔

灞桥，在西安城东边，有桥梁横跨灞河，这个地方是灞桥。汉代人送朋友往往会送到此处，为表达对友人的珍视，人们常折柳以送友人。陆游有诗云：“结茅杜曲桑麻地，觅句灞桥风雪天。”灞桥的风雪不是风雪，而是柳絮飘扬。

魏晋时期，曾有个经典场景。这天，谢太傅在寒冷的冬天举行家庭聚会，跟孩子们解读诗文。不一会儿，屋外雪花朵朵。瞧见这等场景，谢太傅询问他们：“纷纷扬扬的白雪像什么？”胡儿说：“就跟把盐撒在空中差不多。”有个姑娘说：“就像柳絮被风吹拂得漫天飞舞！”现在，我们把这个反过来问：“纷纷扬扬的柳絮像什么？”

纷纷扬扬的柳絮像什么？当然像雪花。这年暮春，某个周末，清晨阳光明媚，天空高远湛蓝。我们走南门，瞧了瞧南门附近的早市，又在附近吃早餐。每个地方的饮食总有些许差异，比如说河南的胡辣汤。

从早餐店出来，扫码摩拜单车，在高德地图的引导下，我们直奔灞桥而去。沿途，经过某个地方，我瞧见小书店，跟身边的女孩说：“我们把车停在附近，然后，进去看看吧。”当我埋头于在各种旧书当中，仿佛寻找着什么宝藏时。女孩子跑来告诉我说：“你还在找书，我们的车都给人骑走了？”这是非常无语的事情，却只能存在特定时期。

到附近找了许久，重新找到两辆自行车。骑着自行车，我们穿行在公路上，路边是深蓝色的鸢尾花，仿佛在为我们唱歌。是啊，美丽的柳絮宛如雪花，我有听到它们在呼唤我，对我诉说着它们的美好，诉说着它们的纯洁。

此情此景像首温柔的诗，像场美丽的梦。有些许瑕疵，现在回忆起来，还是很美好。

从生长的环境来看，我们在许多方面可以达成共识，她属于理想的伴侣。后来，有老师告诉我说：“你应该找个从农村出来，又像你这样接受高等教育的女孩。我跟我对象许多时候就有点那个。”那时，我告诉老师说：“《平凡的世界》里有个女孩从农村出来，她是郝红梅。郝红梅知道农村的艰辛，所以，她想通过婚姻改变自己。”

说这话的时候，我想到这个女孩。这个时候，我们已经失去联系，除了相忘江湖，别无选择。老师面对我的回答，没有肯定，亦没有否定，而是沉默。他对这种事情肯定深有同感。农村出来的女孩，如果有选择，不会再选择农村出来的孩子。不是她们势利，而是人性，人性趋利避害。与其埋怨现实，不如和光同尘，努力改变自己。

那天，我们沿着灞河骑行很长段路。期间，我领着她沿着小路直直骑行到灞河边上，有中年大叔在河边钓鱼。女孩站在河边，附近杨柳依依，夕阳照在灞河上。这瞬间，我觉得，她像柳絮散发着芳香，隐含某种情调。

我见她有些疲倦询问她，我们坐地铁回去？她说可以。我背着她的书包骑在前面，偶尔回头看她，很快察觉到她的自行车不对劲。在路边停下，“你骑我的自行车吧？”我的猜测没错，她的自行车有边刹车坏了。

人间自是有情痴，此恨无关风和月。后来，我们走散了。有天，我到图书馆北边去学习。我起身到饮水机前接水，随后来到中间。我站在窗边，晚霞染红了梧桐，染红了西边的天空，我瞧见他们，我瞧见他们在腾飞塔前。什么是成熟？成熟就是懂得沙粒与珍珠的区别，成熟就是不会因为云彩而放弃浩瀚的星空。我是沙粒，我是云彩？

是啊，我是沙粒，我是云彩，我还是路人。我在她的人生路上，就像灞桥风雪，就像那些零落的柳絮，很快就会被雨水侵蚀，最后消失得无影无踪。很久以前，我们曾走在梧桐东道上，争论着谁读书多？

我告诉她：“那是因为你认为他做出巨大成就，所以，你觉得他读书比我！”女孩不服气，“那你就去证明自己啊？”是啊，男孩子努力尝试，还是无法挽回什么，他还可以做的，仅仅是证明自己，证明自己给别人看。

许多年后，到底是什么人让自己发愤，到底是什么经历让自己对强烈渴望功名利禄已不重要。
