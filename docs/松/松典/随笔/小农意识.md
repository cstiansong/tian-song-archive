# 小农意识

Date: March 29, 2022
album: 美丽的山村

这天，天空那么蓝，那么深沉，那么安静，灰白的云彩从城市边缘飘过来，赋予这座城市光与影。虽说天气已到春末，穿着短袖走到街上，还是会觉得单薄、寒冷。没准很快会流鼻涕、咳嗽，甚至感冒去医院吃药打针。

这些岁月，我克服从前怎么都改不掉的毛病，对有些事情的认识亦没有从前那么偏激偏执。

于是，我知道，我工作的时机已经成熟。时机成熟，最近会有机会给我提供经济基础。凭借直觉与理性分析，我会选择适合自己的工作。去工作，在工作中提升实践能力，在工作中积累创作素材，在工作中不断充实自己，在工作中随着能力的提升迎接新的挑战。至于，文学的理想，我只能说，我会迈着步伐，默默的，无声无息的，以这种方式慢慢靠近。

我已经把定位想清楚，我是农民的后代，从小生活在农村，农民的思维深深影响着我的人生，无形当中给予我强烈的小农意识。由于掌握的信息不全面，我不愿意接受有些新鲜事物。我没有强烈的物质欲望，我的生活有基本的保障，我就会觉得满足。为满足温饱，为舒适生活，为人生志趣……我可以在某块土地用心耕耘。这是我的小农意识。

亲爱的朋友，你知道吗？有时候，我会到街上走来走去，看看街边那些人如何生活，如何快乐，又是如何忧愁。我仿佛可以得到点点生活的意义。这种观察像翻书。也许，翻来翻去，天长地久，还是可以读懂。只是，它们属于别人。

虽然我不想失败，但是，我知道，我无法避免失败。我可以做的仅是失败以后汲取经验教训，努力提高自己，别放弃，继续往前。我容易跌倒，但跌倒后，有足够坚强的意志爬起来。现在跟从前有什么区别，便是我承认自己的普通。

我普通，我心底有理想，但我还是很普通。我普通到，我应该养家糊口，应该结婚生子，应该……

我幻想自己在雨后天晴的下午，坐在香樟树下捧着文学经典细读。我的周围是蝉鸣，是麻雀的啁啾，是阳光在叶面流动，是时间像清水唱着欢快的歌儿。这样的生活大概是许许多多青年梦寐以求的，让人心生羡慕。然而，它不是文学人生，文学要贴近人生，要跟命运紧紧联系，要从人生从命运深处汲取生长所需。

它像树苗，需要土壤，需要阳光，需要雨露，需要环境去生长。我只有把自己融入到社会深处，才有可能挖掘出深刻的宝贵的东西，进而为文学作品注入源源不断的活力，让那些作品将来有天脱离我还可以存在。

此前，我阅读到查尔斯·达尔文说：“要体验大自然，你不必到遥远的热带雨林去。大自然就在你身边，就在我们的公园、花圃和当地的自然保护区。”同样道理，要体验生活，并不非得需要到某些生活里去，我们只需要用心感受、用眼睛观察、用身体去触摸。哪怕自己的生活单调、苍白，还是可以书写出丰富多彩的故事，引起读者的共鸣。

到这个阶段，无论理想，还是现实，我都需要出去工作，只有工作才可以赋予人生以新的可能性。
