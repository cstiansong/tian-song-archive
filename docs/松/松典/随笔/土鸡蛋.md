# 土鸡蛋

Date: April 4, 2022
album: 美丽的山村

前面宰鸡，我动了恻隐之心，所以，我们家有只母鸡幸运下来。只几个月光景，这只母鸡孵出小鸡，小鸡崽迅速长大，跟着老母鸡下蛋。就是说，有段时间，我们家每天可以捡到两个土鸡蛋。这事搁在很久以前会让我很满足。

童年时期，我时常盼望着捡鸡蛋，捡到鸡蛋堪称我在马路边捡到一块钱！那个人生阶段，我们这些孩子过生日，当妈的想得起来，就煮两个土鸡蛋；想不起来，就凑合着过去。反正，没有损失。转眼间，我成长成人。

母鸡下蛋，跟我没有太大关系，只是没菜时，我跑去捡几个来，弄个紫菜蛋花汤。有次，我妈念叨：“现在，这些人是越大越懒，鸡蛋都不爱去捡！”我没有接话，心想，有了某种追求，许多事情，我确实不放心上了。

每天两个鸡蛋。天长地久，我们家专门用来存放物品的房间。在那间房里的木沙发上，有四五十枚土鸡蛋。这些土鸡蛋，漂亮，蛋壳坚硬。后来，有天饭桌上，父亲说，那些土鸡蛋就不吃了，等他们回来，送给他们。瞬间，我觉得这位农民，值得钦佩，他很厚道。相较于城市，农村社会依然保持着某种人情美。当然，随着市场经济的侵入，这种朴素的人情美正被唯利是图的庸俗人生观取代。即便这样，他们身上依然残存着某些美丽。在这般寻常的日子，我忽然懂得，文学不仅是对人性阴暗面的揭露，不仅是对社会丑陋现象的鞭挞，还是对人们面对生活与苦难时所展现出来的真善美的讴歌。

腊月，大雪纷纷扬扬，反反复复。地面上，积淀起厚厚的雪花，有些地方的冰柱如狼牙交错，仿佛等待着落网的人。在这天下午，两只母鸡脖子见血，成为饭桌上的佳肴。那以后，土鸡蛋数量不再增加。有次，我去储物间企图找点水果吃，土鸡蛋已经不在那儿。没有多问，像这种事情，我不太愿意去浪费时间。懂得东西，没必要多说。

许多年前，在某个暮春时机，含苞待放的花树，仿佛接到通知，在春风中次第开放。

我在附近村庄读小学。那时，我们中午走路回家吃饭。有次，走在路上，我瞧见老母鸡咯咯叫，很快察觉到这是老母鸡刚下蛋以后的表现。好奇心作祟，立马进行搜索。我很快在杉树下，有个隐秘的地方，找到大窝土鸡蛋。

闷声发大财，我把这件事情藏在心底。回到学校，跟小伙伴说起，他让我把土鸡蛋捡来，带到他们家煮。次日中午，我把偶然发现的这窝鸡蛋捡走，装进书包，运到伙伴他家。然而，可能是心理作祟，吃起鸡蛋来，我觉得怪怪的，仿佛拖欠人家什么东西，极其不安心。随后岁月，渐渐跟有些人保持着某种距离，不做这类偷鸡摸狗的事。

许多年后，大学寒假，我跟朋友路过这位伙伴家。朋友跟他说了几句话，我站在身边，没有跟他招呼。他说他刚刚扛着木头，肩膀压得酸痛……这位昔日的伙伴坐在他家垓阳坎上，我注视着他，猛然想起土鸡蛋。

在家那些日子，我有好几次遇见大娘大婶背着背篼，到这些地方收土鸡蛋。这些人，不用问，我都知道，他们家肯定有人要坐月子，需要土鸡蛋补身子。后来，有天夜晚，小学同桌问我：“你们那边有土鸡蛋吗？”

我回她说：“没有。养鸡麻烦，现在的人不怎样养鸡。”再后来，这位朋友有了第二个孩子。这位朋友是我的小学同桌。我看着他们恋爱，看着他们结婚，看着他们生孩子，看着他们养孩子……如果没有读书，我的人生跟他们相似。然而，那是正常人必然要走的路。没有读书，我的人生依然会跟他们相似，只因因为读书延缓了进程而已。

窗外，天空出现月亮，云朵像绵羊跑来跑去，晚风阵阵，没有白天燥热。这是很美丽的画面，这样的画面非常愉悦人的眼睛，因为它非常沉静优美。窗内，我在想，多年以后，我有老婆，我有老婆怀上了我的种。

我大概会让爹妈在村里养许多土鸡，收集许许多多土鸡蛋，这算是风俗的延续。
