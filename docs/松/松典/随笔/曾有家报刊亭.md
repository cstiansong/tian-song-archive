# 曾有家报刊亭

Date: April 20, 2022
album: 象牙塔

很长段时间，甚至就在前几天还有朋友建议我，从小文章开始，从报刊杂志开始，逐渐积累起名气。他们在不同时间不同地点异口同声地告诉我说：“哪有像你这样上来就写长篇小说的！”他们的建议，我心领，却拒绝。

我想得很清楚，我没有理由按照他们的建议走他们以为的道路。有关这件事情的思考，我可以追溯到很久以前。很久以前，我刚进大学。那时候，东南门附近曾有家报刊亭。这个家报刊亭摆满各种各样的杂志，其中不乏大量文学杂志。那年，我想着自己能不能先在某些杂志上发表文章。所以，我定期阅读杂志，隔段时间来这购买杂志。

翻看杂志，依葫芦画瓢，我写完二十多篇文章，找来投稿邮箱发送出去。大多数邮箱都没有回复我，只有极少数编辑偶尔回我些许简洁的文字，这些简洁的文字往往是建议，往往是不符合本杂志的文风可以另投。

不是我不懂得坚持，而是有天，我忽然发现，我有捷径可以走。捷径就是写书出书。那时，我心想着走杂志期刊这样的道路，不知要等到猴年马月。我有这样的想法源于有天，朋友告诉我说：“包括杂志在内的出版社已经是夕阳产业，它们的利润来源稀薄。关键是，杂志养着大堆作者，除非你的作品特别出色，否则很难的。”

联系到亲身经历，我很快认可这位朋友说的。写许多小文章不如认真创作长篇。也许，前期创作出的长篇质量欠佳，但随着叙事修为的精进，总会有出彩的部作品可以大获成功。从这以后，我彻底放弃传统路线。

那些岁月，我很少走出校门，多数时间都泡在图书馆。我随意翻看着书籍，仿佛是为摆脱孤独。我每天都在汲取文学的力量，每天都是文学的生活，自然而然生出文学的想法，自然而然想要成为文学创作者，即作家。

我不再去东南门那家报刊亭买杂志。后来，偶遇从那儿路过，我忽然发现，曾经摆满杂志期刊的地方已有零食热狗矿泉水等消费品。我意识到，照这样下去，夕阳产业很快要退出历史的舞台。

再后来，有人把东南门附近那家报刊亭拆走。那以后，大多数学生不知道那儿曾伫立着某个时代的缩影。我想，到校门口买杂志，这种事情，极有可能成为某些人的回忆，就像某些长辈回忆自己曾苦苦等待来自远方的信。

有天，我在图书馆偶然读到一篇文章。根据叙述内容，我判断出，这篇文章的作者是编辑。他在这篇文章里说，杂志再也不可能回到曾经，回到曾经那个辉煌的年代。在那个辉煌的年代，农民在杂志上发表文章，直接可以实现阶层跨越，可以调到县委宣传部。在今天，这种事情已经不可能发生。杂志属于是泥菩萨过河。泥菩萨过河，为延续寿命，它们不得不依附于某些平台，不得不依赖于国家扶持。至于发表杂志那些文章，大多只有作者编辑等相关人员阅读。

我有认真思考这个问题。故事不可能灭亡，人永远需要精神填补内心，所以，故事不可能灭亡，所以文学将于人类共存。但是，承载它们的形式不可能不更迭。就是说纸质书很有可能被电子书取代，实体书店很有可能被网上书店取代。事实上，这个趋势越来越明显。与此同时，人们的选择多样化，不再局限于故事，不再痴迷文学。

不能与时俱进的事物终将在时间的长河中被淹没，这种更迭虽然残酷，但这就是时代。
