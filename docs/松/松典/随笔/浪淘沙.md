# 浪淘沙

Date: May 21, 2022
album: 象牙塔

白天夜晚，我把自己关在房间，不是哗啦啦地翻书看，便是噼里啪啦地修炼文字。我相信着凭借扎实的基本功和艰苦卓越的训练，我可以不断创作出超越原先水准的作品，进而在书籍与文字间重获新生。因为日新月异的激情，我有种精灵自由翱翔的幻觉，而书籍跟文字为我提供源源不断的激情，让我觉得这样的生活，于我便已是最理想的。

我的心头像百灵鸟在梧桐树间欢呼，唱着优雅动人的歌，诉说着源源不断的激情。我热爱着文学，阴差阳错进入书籍的世界。随着性灵自然的选择，滋生出文学的梦想，还愿意为此付出生命，甚是全部。那种献身的决心鞭策着我寻找笔名。于是，我找到了“浪淘沙”。当时，我心想着在中国古典文化，比如说唐诗宋词里肯定可以找到。

“浪淘沙”忽然飘进我的人生，让我有种强烈的感觉。我随即动手查找来源，它出自唐代诗人刘禹锡的诗歌。在这首诗歌里，我惊奇地发现有“莫言”。思来想去，我决定用这个词牌名用这首诗词作为自己的笔名。

浪淘沙，这是我的笔名。它出自“莫道谗言如浪深，莫言迁客似沙沉。千淘万漉虽辛苦，吹尽狂沙始到金。”我自认为有三个寓意，大浪淘沙，沉者为金；借鉴莫言，沉默是金；江河百折，吾志不改。余生，从事文学的都是浪淘沙，而不是田松。父母给予我的名字，简单、普通、大众，没有文学气味。那个名字，没有文学的联系，但我知道那是父母的礼物。我不愿意抛弃，我还是他们的孩子，农家永远朴素的孩子。只是，我的写作都是浪淘沙，浪淘沙就是我身体里讲故事的人。

有了笔名，我像吐丝的春蚕，就用书籍堆砌成围墙，把自己驻扎其中，不希望有人打扰，不曾想要麻烦别人。我就畅游在书籍的海洋，源源不断地汲取文学的力量。读进藏书吐出芳华，渐渐在周围形成坚硬的壁垒，任何人都难以敲破，这是传说中的“作茧自缚”。我知道，将来某天，我会破茧而出化蛹成蝶。翩翩起舞的蝴蝶不是我，而是浪淘沙。老天爷选择了我，让浪淘沙进入我的身体，我有责任完成这种使命。只是，我的性格幼稚还晚熟，需要时间与耐心成长。

需要时间与耐心成长，这样的信念背后蕴含着，我相信努力，我相信足够努力可以战胜天赋异禀的人，甚至绽放出与天才同等的光芒。那些相信背景相信天赋相信科学相信技能相信资源的人，大概会嘲笑浪淘沙相信努力吧。

只是，浪淘沙不会屈服于现实屈服于命运的。因为我偶尔幻想着将来离开这个世界，我创作出来的文章可以流传，可以化作许多人继续生活的希望。失败时，不要气馁；不顺时，不要抱怨；痛苦时，不要难过；挫折时，不要放弃。我觉得，完成自己的使命，在文学这个领域有所成就，我便已知足。所以，余生，我都是浪淘沙。

我是浪淘沙，我会读个性的书籍，走个性的道路，写个性的人事，前半生读破万卷书，后半生行万里路，恰当精妙地记录遭遇的人事，把握时代的脉搏，不断描写细节进而深入灵魂，缔造出文学王国。
