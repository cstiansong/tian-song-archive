# 华岳仙掌

Date: April 25, 2022
album: 象牙塔

有次，有位朋友在朋友圈说自己想去爬华山。于是，我私信他自己想去。小学期结束，我们乘坐地铁到西安北站，又从西安北站乘坐高铁到渭水河畔华阴市。从出高铁，热风翻滚着彼此起伏，让我觉得自己是热锅上的蚂蚁。

山脚下，有商贩向我们推荐草帽推荐手套，拗不过商贩的热情，索性购买防备意外。后来，我们以事实证明了这是明智的行为。购买门票进入山门，我们往前走，只见山峦起伏，像极了金庸小说里勾勒出来的华山。

山门消失在我们身后，忽然走来的是外国人，他们这是下山。朋友告诉我说：“你注意到刚刚那几个外国人没？”我点头，他说：“他们的表情可以说是生无可恋！”我心想，不就是下山吗？下山至于搞得那么夸张吗。

然而，随着时间流逝，太阳啊真叫个歹毒，山路啊真叫个曲折，坡度啊真叫个夸张！刚进去那会儿兴致勃勃，堪称昂首阔步的我们后面像乌龟般行驶的公交车，走走停停，停停走走。后来丢到面子，在路边捡起木棍拄着走。沿途有贩卖黄瓜，我们就买黄瓜；沿途有贩卖西红柿，我们就买西红柿……矿泉水喝了许多，水瓶随手扔得满山都是。

爬山越深入，坡度越大。有些地方陡峭得垂直，爬山爬山，拉着两边铁链爬上去。即将抵达山顶时，有处地方小路完全就是悬崖峭壁。但负责旅游开发的人，他们从上边挂出两条铁链，登山者需要臂力需要勇气。那时，我胆儿比较肥硕，闷头闷往前，全然不顾自己死活。等到我爬上去后，我那朋友说自己不行，然后，退回去走远路。

就这样走走停停，我们成功登顶。华山海拔不是很高，但，周围是平原，自身属于是断块山脉，就像天神用斧头劈开的，让人忍不住感慨大自然的神奇，大自然赋予华山独特的险与峻。随后在前往东峰的路上，在那条羊肠小道般的山脊之上，我坐在最高处眺望远方，只觉天地空旷。在我感慨天地空旷时，朋友却说自己不敢看，说自己腿在发抖。

东峰饭店附近，我们租帐篷。帐篷搭建在山顶相对和缓的地方，左侧是悬崖峭壁，右侧是茂密山林。在我以为，我们可以在山顶仰望星空时，天边突然惊雷滚滚，据说会有雷阵雨。很快有人来通知我们，帐篷不能住，原价退回。

随着人流来到饭店附近，我在猜想，这样的房屋光是修建恐怕就要花费许多。在这里，许多东西都散发着不可思议，甚至诡异的气息。那时，我羡慕着可以在这家饭店入住的人，他们不用像普通人挤在拥挤、清冷的走廊上。朋友冷得咬紧牙关去租用大衣，而我穿着单薄的衣裳依靠着墙壁上昏昏欲睡。这个时候，我明白了某个道理。

在这个世界上，有些地方确实很包容，它们欢迎全世界的人到那儿去旅游。但前提是，你得有钱。你有钱，就可以享受到那些地方绝大多数资源。如果享受不到，只能说你的金钱不够。现在想来确实这样啊。

天色朦胧，我从睡梦中醒来。朋友告诉我说，我们现在要爬到山顶去，爬到山顶去看日出。

匆匆忙忙很快找到合适的位置，像是行注目礼，注视着远方，注视着远方浅黄的云海。随着云海越来越红，金黄色的小沙粒逐渐长大，逐渐长大成为金黄色的蛋黄，逐渐长大成为金黄色的火球……很快就升到云端之上，升到云端之上绽放出耀眼的光芒，可谓是让人要在清晨流出冷汗。顺着人流，我们来到西峰乘坐缆车。

缆车缓缓，我眺望着远方，眺望着远方的山峦。我的脑海却在回想，沿途那些梯步，那些凉亭，那些护栏……甚至那位给沿路商店挑送商品的老人家，甚至生活在这些地方的人们，我觉得他们都是华山孕育出来的故事。

想到故事，我就在想，改变命运的不是大学，而是读书，读书带来知识，知识改变命运。读万卷书行万里路的人，未必是富豪未必是大学者未必是大作家，但他们会随着时间的流逝越来越强，直到有天可以强到让人觉得不可超越。想到这儿，我觉得天地空灵，大学只需要埋头苦读，而不是蹉跎岁月。无论是谁都会用自己的独特书写出故事。

哪怕现在我书写的是随笔，这里面有大量虚构，这些虚构掺杂着想象，但全是我内心真实的想法。
