# 法学生们

Date: June 12, 2022
album: 象牙塔

当你接触的人越多，你就发现：比你层次高的人鼓励你；同层次的人欣赏你；层次比你低的人，才会诋毁你。

——稻盛和夫

按理说，人文学院与法学院是平行的，我不应该认识那么多法学生。但事实上，我的的确确认识了许多法学生。在许多法学生里至今仍有联系。我知道为什么，因为读书交流群。有位法学生是老群友，他拉了他们进去。

前期属于是网友，后期因为偶尔在钱学森图书馆相逢，天长地久，竟然也认出彼此。

有次，有位法学生忽然联系我：“毕业以后，很难见面，我们准备请你吃个饭。你看看你什么时候方便？”于是，我跟他们有了具体接触。在某间羊蝎子店，我们说了许多话。许是心意相通，有跟某位朋互送书籍。

那天夜晚下雨，我们在羊蝎子店等雨停歇。期间，我注视着这些朋友，心想：这些朋友，有事情，你找他们，他们可以帮，尽量都会帮。没有事情，共同语言其实不是很多。当然，并不能因此否认我们之间的友情。

饭局上，有位朋友说：“当时，我还在想，如果平行线没有收你，怎么办？结果后面……我特意去看了看泰发祥，觉得很合适你！”说这话时，我已经认准了泰发祥，并且，误以为自己会顺利，他们不会搞出事情。遗憾的是，后面还是出了事。我知道是什么原因。就是这位朋友曾告诉我的：“你要把自己的个性收起来，对别人不要太苛刻！”

是啊，少年时期，我简直就是个不懂事毛头小子，牛皮轰轰的，说话很冲。等到我在泥巴地里打了几次滚，被社会锤炼，就没有在空气漂浮，亦失去那种锋芒。遭受忧伤，经历挫折，我依然深爱着这个世界。而且，我不会后悔自己的选择，反倒会感谢曾经的自己足够勇敢。今天，我不认为那些经历有什么错，因为现在更理想。

那天，我们走路回到学校。期间，又说到许多。有位朋友建议我去写贵州的地方特色。他的建议是正确的。跟他讲了许多以后，他得出结论：“其实，你已经有想法了。”是啊，那时候，我就有想法了。如果说读书创作是农村孩子出人头地的捷径，那么，书写地域文化里面富有特色的东西，就是农村孩子出人头地的捷径中的捷径。

南门附近，校园里，柳树边，朋友提议说：“我们在这里合影？没准以后真成名了，我还可以拿出去吹嘘！”说起来，我跟这些人的认识蛮具有偶然性。但静下心来思考，又存在着某种必然的因素。这就是缘分吧。

最后那段时间，我想回请吃饭，但他们说心意领了，请吃饭就不用，他们有事情忙。从那以后，直到今天，我们都没有见过，将来很长段时间都不会遇见。昨夜，有位师姐曾说：“毕业后，很难见到大学那些朋友了。”

我看着师姐说：“深有同感。”此刻，我坐在图书馆某个角落，窗外是深沉寂静的夜晚，静悄悄的，没有任何声息。窗内，我坐在电脑跟前噼里啪啦飞速地敲击着键盘，诉说着我对法学生们的追忆。

走出社会，渐渐明白，人世间，聪明人才会欣赏聪明人，我们能够相遇就是种幸运。
