# 走出大山

Date: April 16, 2022
album: 象牙塔

我出生在云贵高原东北部边陲小城，边陲小城的大堡村。大堡村距离著名佛教胜地梵净山并不遥远。在地图导航里，可以清楚瞧见大堡村位于它的山麓。大堡村安静恬淡，群山环绕，树丛繁茂，人在其中走，压根看不见。

大堡村人常年为生活奔波，相对于城里人，他们具有朴实朴素的人情美。老天爷就把我降生在这里。作为村民，我有很长段时间都生活在土地上。五年级，老师让我们写作文，写关于县城的。那时，我压根就没有到过县城，不知道所谓城市到底是什么模样。我按照自己的理解写完，老师却告诉我说，你应该写你没有去过，这些都是你的想象。

很久以后，由于学习太投入，不幸近视。我不得不佩戴眼镜，不得不进县城，于是某个周天，我跟着朋友去县城。那时候，我觉得县城好大好大，大到我偶遇书店，进去转悠后，掏钱买到《钢铁是怎样炼成的》回家。

大堡村属于是山村，没有交通干线，没有旅游资源，只有贫瘠的土地，只有山前蜿蜒着的清溪，以及零零散散的稻田。在这里，许多人忍受着贫穷的折磨，却没有办法改变现状。当然，相对于从前，今天已经很好很好啦。

大堡村属于是山村，没有交通干线，没有旅游资源，只有贫瘠的土地，只有山前蜿蜒着的清溪，以及零零散散的稻田。在这里，许多人忍受着贫穷的折磨，却没有办法改变现状。当然，相对于从前，今天已经很好很好啦。

在我们这个大家族，迄今，没有谁属于公职人员。人生路上，这些家庭成员很难提供帮助，甚至会因为他们而束缚我。只是，我从小在这里长大，对这些人很了解，他们虽然是穷人，但穷人有穷人的自尊，不需要别人恶意的怜悯。所以，不存在说像某些文章里面说什么找对象收取高价彩礼扶持男娃，什么拖累走出大山的孩子。他们不会的。

小学是在邻村完成的，后来这所乡村小学倒闭，至今只剩空荡荡的教学楼，只有我们当年栽种的香樟在教学楼跟前孤独地歌唱。父母除了给我提供吃住，没有能力在学业上支持走下去。所幸，我在特定时机遇见有些人。

有些人让我不断突破自我的极限，仿佛又得到命运的加持，我总能在关键时刻作出正确的选择。乡镇初中毕业后，我考进重点中学。在云贵高原东北部那所重点中学，我接触到许许多多新事物。虽然成长的历程有些儿艰辛，但是，最后幸运般考出让班主任都觉得神奇的分数。说起来，我还是要感谢那些老师朋友，他们给我的前行注入力量。

我曾是问题学生，却在高考中脱颖而出，其实有命运的因素。千百年来劳动人民曾总结出，富不过三代穷不过五服。我的高祖是农民，我的曾祖是农民，我的祖父是农民，我的父亲是农民，我是农民的后代。作为农民的后代，我的父亲凭借着勤劳与技术成为村里的个体户。我自信，我们这个大家族到我这代将奔赴城市，彻底摆脱土地。

后来，回学校领取录取通知书，甭提自己有多开心。乘车回到大湾河对岸，我几乎跑着回家，翻山越岭已经不是难事，只因为我太激动，我激动得恨不得挥舞着通知书，边跑边喊我考上大学啦！那段时间，全家人都好开心！

没有办学酒，但有亲戚随礼，以表示祝贺。在那里，考进大学意味着走出大山。
