# 别了青春

Created: March 21, 2024 10:34 PM
Text: 　　雨天，撑伞走在梧桐路上
　　瞧见桐叶蝴蝶般脱落
　　忽然想起我告别了我的青春
　　我要像小鸟寻觅诗和远方
　　像悟空那样西天取经
　　为曾经的荒唐流下泪水
　　我为曾经的幼稚写下诗歌
　　所以，我想要请求你：
　　原谅我无法在这个年纪证明
　　原谅我在这个年纪痛恨平庸
　　原谅我决定永远遗忘了你
　　这首诗歌激荡起生命的浪花
　　油菜花开遍天涯的时候
　　我哼着小曲想念远方的你
　　起风了天空飘起淅淅沥沥的雨
　　桐叶还像蝴蝶那样脱落
　　我觉得像冬夜漫天飞舞的大雪
　　在祭奠逝去或远去的青春
　　别了我的青春，青春啊青春
　　2022.6.27象牙塔·钱馆
