# 情人的眼泪

Created: March 21, 2024 10:34 PM
Text: 　　柳树和芦苇丛里，鸟儿们在编织鸟巢
　　温顺的柳条，酣然沉睡，轻轻摇曳
　　可是，这时，河水泛起了血红的泡沫
　　我心已死，适可而止，便留有全尸
　　孤零仰躺在荒原，我彻夜仰望星空
　　人生是苦，回首苍凉，何必孰是孰非
　　鸟饱尝芳香的美果，我却心如死水
　　就像山顶的积雪，被太阳晒得个融化
　　那晚，她心潮起伏，火焰在胸膛燃烧
　　微笑留在她嘴角，她好像有话要说
　　情人的眼泪，全是虚假的，别相信了
　　世界与我无关，它已走远，毫不存在
　　白杨树生长的地方，我的情思环绕你
　　陪伴你，思念你，以温柔的目光注视你
