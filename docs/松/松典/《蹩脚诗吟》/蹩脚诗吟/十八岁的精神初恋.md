# 十八岁的精神初恋

Created: March 21, 2024 10:34 PM
Text: 　　我就知道，用力用心表达爱意。
　　却不明白，呵护照顾彼此……
　　多年以后，我翻看泛黄的日记：
　　我十八岁喜爱的那个姑娘，
　　曾是我的精神初恋，她很美。
　　那时候，我们都还是孩子。
　　我就是想和你看场电影啊 ，
　　我就是想和你约图书馆啊，
　　我就是想请你吃冰淇淋……
　　可你，为什么，为什么要这样？
　　这些年，你的容颜我模糊了。
　　可我偶有想起你的时候……
　　彻底失去你以后，我才发现：
　　我曾是多么多么，真心爱你。
　　我们没有牵手，也没有拥抱。
　　我却认为，你是我的精神初恋。
