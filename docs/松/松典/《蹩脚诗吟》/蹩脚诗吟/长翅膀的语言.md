# 长翅膀的语言

Created: March 21, 2024 10:34 PM
Text: 　　我轻轻把主楼摁进大地深处
　　让自己变成梧桐的血液
　　自由地驰骋在辽阔的草原
　　我把天边的云彩踩到脚下
　　让无垠的宇宙成为内心
　　我却是天地最神秘的人物
　　开启天才模式神仙附体
　　现在我觉得没有什么不可以
　　跟成为作家联系在一起
　　笨拙的孩儿通过勤学苦练
　　走到人迹罕至的地方
　　只要奋进的热血永远在沸腾
　　只要渴望的种子在心底
　　终有蜕变那天可以化身为神灵
　　未来的路从内心出发到远方
　　不要输给风雨慢慢儿奔跑
　　用诗人的热情孩童的天真
　　让笔下的语言长出天使的翅膀
　　像美丽的歌缓缓飞出唇沿
