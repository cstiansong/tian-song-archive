# 原来，这是读诗、写诗啊

Created: March 21, 2024 10:34 PM
Text: 　　雨天，窗外的云层压得老低
　　我在钱学森图书馆，的北三楼
　　北三楼西边读诗。原想睡午觉
　　可我没头没脑地读诗，读诗
　　一个字，两个词，三个句子……
　　一页两页，五页九页，十三页……
　　原想在红皮沙发睡午觉的，我
　　忽然笑喷，原来这是读诗啊
　　雨天，窗外的云层压得老低
　　我在钱学森图书馆，的北三楼
　　北三楼东边写诗。原想睡午觉
　　可我没头没脑地写诗，写诗
　　一个字，两个词，三个标点……
　　一条鱼，两匹野马，三只绵羊……
　　备忘录，积累了几句，蹩脚诗吟
　　忽然笑喷，原来这就是写诗啊
　　我感叹：原来，这是读诗、写诗！
　　如果这样，诗人读诗写诗……
　　实际就是吃饭睡觉，玩手机~~
