# 为你唱歌

Created: March 21, 2024 10:34 PM
Text: 　　一、嵇康
　　我想为你唱歌唱出生命的绝响
　　像弹出广陵散的嵇康
　　乘以清风追随天边那轮明月
　　像海湾拥抱激起泡沫的浪花
　　像沙滩奉献飘摇的椰子
　　二、歌谣
　　愿意亲吻你身体每个角落
　　以真诚的钟情把你扑倒在怀
　　牵着你的手走完红毯
　　海龟般在深色的海洋奔赴沙滩
　　像海螺回荡痴情美丽的歌谣
　　三、离骚
　　我欣赏你像蔚蓝色天空
　　没有你生命必将暗淡
　　姑娘啊姑娘～爱情啊爱情
　　你像风像初升的太阳
　　你投我以轻轻浅浅的温柔
　　我却再不能把你忘记
　　你深深地让我陶醉在梦里
　　梦里你前世今生的回眸
　　让我想为你唱歌，唱这首歌
　　像嵇康的广陵散成为绝响
　　像屈原的离骚回荡在汨罗江
　　四、痴情
　　姑娘啊姑娘爱情啊爱情
　　想和你深入交流
　　冷漠的表情却让我受伤
　　野兽受伤偷偷跑进山洞
　　在洞里舔流血的伤口
　　心里还惦记着给姑娘唱歌
　　没有你，我再有成就
　　窗外的繁华与荣耀
　　也只惹我心烦甚至意乱
　　五、摩托
　　如果有辆摩托我带你去飞
　　有句话我想悄悄和你说
　　亲爱的姑娘我永远爱着你
　　纵然春风欺骗你我也不会
　　让深爱着的姑娘受伤
　　海浪即便划破堤坝的脸颊
　　堤坝也不愿另寻新欢
　　六、守护
　　心上人跟前我永远是小孩
　　说话吞吞吐吐手脚忙乱
　　回忆起来有些许遗憾
　　眼含泪水微笑着也说幸福
　　海鸥跟海洋说我能瞧见你
　　海洋回海鸥：我永远守护你
　　七、懊悔
　　海风呵海风我愿用生命去
　　赌博：爱你这辈子没有怨言
　　我和你说我爱你会让我懊悔
　　几个星期几个月甚是几个学期
　　没和你说我爱你却会让我
　　后悔这辈子几个春秋几个冬夏
　　八、绝响
　　太阳啊太阳在海天交际
　　的那地方照亮我求爱的人生
　　姑娘啊姑娘你在殿堂尽头
　　的那个地方等待婚戒
　　没有强迫我想为你首唱歌
　　像嵇康弹出广陵散
　　生活的旋律似这首诗这首歌
　　还在为你唱歌唱成绝响
　　九、飞翔
　　姑娘来吧来到我身边
　　让我为你唱首绝响的歌
　　姑娘来吧来到我身边
　　让我为你写首甜蜜的诗
　　姑娘来吧来到我身边
　　让我为你点燃乡愁
　　我像大海那样为你欢呼
　　像海鸥那样为你飞翔
