# 且在人间的精卫鸟

Created: March 21, 2024 10:34 PM
Text: 　　我是且在人间的精卫鸟
　　我想要填补那浩瀚的东海
　　我是推石头的西西弗
　　重复的过程寻找生命的意义
　　青春，就像自由的小鸟
　　忽地死在灰色的天空
　　熟悉的背影，与牵挂的人
　　再也寻觅不到踪迹
　　有的人，再也不会遇见
　　而那些流逝的青春
　　注定要活在记忆深处
　　有那么个画面，那么瞬间
　　让我想起记忆中
　　难忘而忧伤的某个人
　　那个画面，那个心魂竟然
　　让我猛地痛苦，与难受
　　我明知道，她已不在
　　可还是忍不住，想她念她
　　牵挂她，且在人间的精卫鸟
　　又或，神话里的西西弗
