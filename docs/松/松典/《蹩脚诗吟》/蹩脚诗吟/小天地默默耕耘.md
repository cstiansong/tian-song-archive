# 小天地默默耕耘

Created: March 21, 2024 10:34 PM
Text: 　　多少清晨，多少夜晚，多少痴心……
　　宁静与祥和，为我阻挡世俗的聒噪。
　　我不想谈论那些虚伪的话题，
　　某些东西若是说出口，味就变了。
　　亲爱的朋友，你可知道可明白？
　　时间没有顺序，年轻没有对错。
　　那些别人以为的弯路，乃至错误，
　　若没有痕迹，潜能则不被激发。
　　我发誓，我不想把手里的牌打烂，
　　只是，我不能辜负青春，我要勇敢。
　　我要勇敢地赌博：决心改变命运！
　　在小天地默默耕耘，志存高远……
　　读些书，走些路，识些人，有些经历
　　以后，我相信未来注定焕发光彩。
　　哪怕我多么多么怀念，那些儿岁月。
　　冷漠的时间淡化矫情，残留朴实，
　　洗尽铅华般，留有珍贵的回忆~~~
　　我还是我，我还在小天地默默耕耘。
