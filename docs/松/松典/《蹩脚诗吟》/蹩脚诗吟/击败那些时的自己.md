# 击败那些时的自己

Created: March 21, 2024 10:34 PM
Text: 　　有些人真的不再遇见，或许
　　那种意义彻底消失。突然间
　　闯进生活带来了，某种意义
　　然后以各种理由离开或不见
　　凡事成就了回忆，就连虚假
　　也那样真实。活生生地真实
　　即使拥抱，终究也守护不了
　　还不如，全心全意变好再见
　　也许你已离开，可我还记得
　　我还记得，那个静谧的夜晚
　　采撷夜空里的红晕，与星星
　　送给你作离别的礼物，不见
　　不见，再也不见，我记着你
　　永远记着你，即使忘记所有
　　我的朋友也就是我写的日记
　　也会帮助我记住你的，就像
　　我曾想要死死抓住你，就像
　　我曾想要用文字记录那记忆
　　从那以后，我疯狂读书码字
　　我要击败那些时候里的自己
