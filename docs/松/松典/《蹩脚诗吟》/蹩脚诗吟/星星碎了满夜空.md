# 星星碎了满夜空

Created: March 21, 2024 10:34 PM
Text: 　　天上的星星，我许久未曾撞见。
　　地上的人儿，也许，再也不见。
　　温柔的姑娘，我想把你迎娶~~
　　用星星的碎片，为你搭建世界。
　　静谧的夜，温柔的水，还有流星……
　　那晚，我在图书馆漫无目的地找寻，
　　心想，即便人离去，记忆也还在。
　　我只记得有你，却忘记了我的存在。
　　铁道星河之夜，阅读星星点点，
　　用星星的碎片，点燃前世的乡愁！
　　遥远的村庄，月亮悄然爬出地平线。
　　千追万寻的东西已失去了意义。
　　用星星的碎片，为你搭建世界。
　　我的思念化作星星，碎了满夜空。
　　然而，星星不明白太阳的热烈~~
　　就像你感受不到，我爱你有多深。
