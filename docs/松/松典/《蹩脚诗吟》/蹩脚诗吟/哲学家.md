# 哲学家

Created: March 21, 2024 10:34 PM
Text: 　　有人说，贵人相助是运气
　　我想那样说没有问题
　　毕竟可遇而不可求的事
　　人世间有多少人非常幸运
　　我常常以为自己就像哲学家
　　思考宇宙般深邃的主题
　　往往就在我细胞燃烧时
　　夕阳无限我运着篮球
　　从家砰砰砰小跑到附近坳上
　　来来回回绕着圈子
　　那时候以为自己像是哲学家
　　思考着宇宙般深邃主题
　　我看着路边青涩的高粱红了
　　路边油绿的红苕叶也黄了
　　夏天结束了，而我赋闲在家
　　忧伤从我心底深处爬出来
　　我竟似辜负了有些人的期待
　　不知觉，走到落寞的田地
　　我跟哲学家似的滚落出泪水
　　而这时天空下起了雨
　　让人分不清自己有没有哭
　　2022.7.26故乡·大堡村
