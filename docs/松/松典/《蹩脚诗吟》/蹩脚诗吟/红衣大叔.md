# 红衣大叔

Created: March 21, 2024 10:34 PM
Text: 　　我拦住辆巧克力色的小车
　　喊老人家跟红衣大叔坐后面
　　打完疫苗后，我们又遇见
　　他主动迎上来跟我拉起家常
　　兄弟，我请你喝酒！我不喝。
　　抽支烟？我笑笑，也不抽！
　　我边听他说故事，边拦车
　　直到有位大婶喊来自家班车
　　他跟我说，他怎样怎么样
　　前面外出打工，包被偷
　　衣服被偷，可他年轻时候却
　　偷人家摩托而被蹲局子。。。
　　那时候摩托一万多，工钱
　　却只有五六块，被判了三年
　　我脑子努力还原时代背景
　　心想，这不是一报还一报吗
　　临走，他喊住我，我停下
　　红衣大叔指自个儿脑袋瓜子
　　“兄弟，脑袋瓜子要灵活！”
　　狠狠点头，我挥挥手说再见
　　可能是坦诚起了影响吧
