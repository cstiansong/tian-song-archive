# 留在人间

Created: March 21, 2024 10:34 PM
Text: 　　左手边上是热情的恋歌
　　右手边上是绝望的哀嚎
　　经过某个拐点我忽然就
　　成熟许许多多，我以为
　　这是大学也是青春结束
　　食堂人来人往莫不相识
　　都淡漠地吃着美味佳肴
　　只有我在旁边伤春悲秋
　　这种东西仿佛属于少年
　　而我只能将其藏在心底
　　也许不会再看见，也许
　　从前，对人间事太热情
　　少年的我难以把握距离
　　然而，这就是我们真实
　　的人生，生离甚至死别
　　亲爱的朋友啊朋友有时
　　我想用美酒把自己灌醉
　　永远逃离人世间的轮回
　　可我不再年少代以青年
　　有志的青年有诗跟远方
　　我像小鸟那样飞向蓝天
　　把自己的影子留在人间
　　2022.7.23故乡·大堡村
