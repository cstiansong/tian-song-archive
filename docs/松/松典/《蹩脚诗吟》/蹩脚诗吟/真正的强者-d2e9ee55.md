# 真正的强者

Created: March 21, 2024 10:34 PM
Text: 　　记忆犹如锋利的刀子
　　很快就把我结痂的伤口割伤
　　汩汩流淌出来的新鲜血液
　　闪烁着光，闪烁着希望
　　虽然有些人并不是这样想
　　但我就这样认为这样做
　　无论对从前的痛苦与忧伤
　　还是现在对这个丛林
　　有些时光被忘却没法重现
　　所以才值得我们珍惜
　　学学他行万里路
　　我的偶像余秋雨在书里说
　　阳光普照山河
　　并不需要活得山河理解
　　春风吹拂整个大地
　　也不在乎大地的表情
　　读到这儿，我想起了曾经
　　把生活费给某个女孩儿
　　请她到城里帮我购买余秋雨
　　后来，我走火入魔魔怔了
　　这个女孩儿劝诫我说：
　　你以前不是喜欢余秋雨吗
　　你应该学学他行万里路
