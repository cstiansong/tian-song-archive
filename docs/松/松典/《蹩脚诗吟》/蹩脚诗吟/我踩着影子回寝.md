# 我踩着影子回寝

Created: March 21, 2024 10:34 PM
Text: 　　我踩着影子回寝，我与影子相随。
　　这是我单调枯燥的大学生活……
　　斑驳的梧桐树，像残灯，像疏影。
　　未经世事的稚气，随年岁而往。
　　清风吹柏杨树梢，树叶哗啦啦。
　　烟雾弥漫的从前，有心事跌宕：
　　曾写的日记枯黄了，记忆也模糊了，
　　就连毛毛虫都爬满了整个花盆。
　　时间就像山间的清水哗啦啦地流。
　　你不要哭泣，花儿来年继续盛开。
　　每天活在小天地，吃饭睡觉踩影子。
　　穷愁潦倒的大学生，就像午夜的兰花。
　　静静开放，静静凋零……终究成为：
　　青春回忆的几页插图。还有什么颜色？
　　取决现在：现在改变过去，创造未来。
　　当然，还是要踩着自己的影子回寝。
