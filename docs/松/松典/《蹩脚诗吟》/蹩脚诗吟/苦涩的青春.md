# 苦涩的青春

Created: March 21, 2024 10:34 PM
Text: 　　我抬起布满老茧的双手写下
　　“你可以看不起任何人，
　　但你别在有人的地方流露。”
　　你不是世界中心也不是全部
　　你要改变自己让自己就
　　像沙漠里的树把根扎得更深
　　悄悄在人看不见的角落
　　紧紧纠缠以汲取大地的力量
　　你有你的热情也有你的命运
　　你不能眼看着悲剧诞生
　　却没有作出任何进化甚至改变
　　站起来！以高傲的灵魂
　　去祭奠跟前这苦涩的青春吧
　　所有繁花像绫罗是烟火
　　唯有你是夜空中最闪亮的星
　　你经历浩劫与漫长黑夜
　　才得以寻觅到未来以及光明
　　活像云那样柔软风这样轻
