# 把命运托付给时间……

Created: March 21, 2024 10:34 PM
Text: 　　不管时间，无惧风雨，埋头苦读。
　　即使遭受最可怕的折磨，又能怎样？
　　我是我，我还是，不管时间无惧风雨。
　　当然！我只能：把命运托付给时间……
　　已死的时间会掩埋，昔日的光辉，
　　所谓文学流派，其实，应试教育。
　　灵感？禀赋？都是冠冕堂皇的说辞。
　　我相信自己，相信不屈不挠的努力。
　　所以，我只能：把命运托付给时间……
　　我不奢望有什么奇迹，我只希望，
　　每天进步点点，有付出，也有收获。
　　这就是偏远地区却在城市的农村娃。
　　命运把我从偏远地区的农村带来，
　　或许，有天意，有因果，还有福祸。
　　可我只能，做好此时此刻，没有其他。
　　因为，我只能：把命运托付给时间……
　　所有的所有，我相信时间有交代！
