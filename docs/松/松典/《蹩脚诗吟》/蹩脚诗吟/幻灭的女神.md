# 幻灭的女神

Created: March 21, 2024 10:34 PM
Text: 　　我像小鸟那样飞进你眼帘
　　轻轻啄你的唇儿
　　阳光照射在青草地上百鸟
　　学伊豆的舞女跳起舞蹈
　　它和你诉说我爱你迷恋你
　　连白云蓝天绿水都爱你
　　你没有动没有说话的模样
　　早早勾走了少年那颗心
　　在失魂的瞬间我终于明白
　　夏天没有秋天黄叶飘飘
　　没有你的日子还是太孤单
　　男人没有哭诉的地方
　　而你却冷漠般碾碎枯桐叶
　　男孩在想象中把你完美
　　你是别人眼里普通的姑娘
　　却是他心底的女神
　　幻灭的女神把想象力
　　赋予他让失恋的蛇缠绕他
