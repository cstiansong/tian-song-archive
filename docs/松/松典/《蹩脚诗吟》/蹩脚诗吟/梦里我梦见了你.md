# 梦里我梦见了你

Created: March 21, 2024 10:34 PM
Text: 　　你可知道？某夜，我梦见了你；
　　我心爱的姑娘，梦里我梦见了你。
　　雪花落进河水，就像泪醉落湖面
　　那些儿梦境里，我切身感受到：
　　花间潜行的风，轻抚你的脚踝。
　　你的微笑没了，我想哭，还想你……
　　我喜欢文学，历史，还有春风春水。
　　可它们抵不住，温柔美丽的你。
　　多少次，我看见年老的窗台
　　落满了灰尘，只因我失去了你。
　　深秋，梧桐落光叶，枝头残留着的
　　不是喜鹊，亦非麻雀，乃情意绵绵。
　　花间潜行的风，好像飞舞的绿叶；
　　山间的清风，与淙淙流水，美极了。
　　可想到，梦里我梦见了你，感伤……
