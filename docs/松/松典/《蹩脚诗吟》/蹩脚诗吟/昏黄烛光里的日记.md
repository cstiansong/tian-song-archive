# 昏黄烛光里的日记

Created: March 21, 2024 10:34 PM
Text: 　　还在睡梦里，这片天空断电了
　　充电宝让我听完前两节网课
　　还可发朋友圈，也足够记录跑步
　　晚饭后，我们到村里的篮球场
　　太阳能路灯忽地亮了起来
　　个多小时，流汗、还需要擦汗
　　回家的路上，音乐响彻云霄
　　漆黑的夜，蛐蛐、蟋蟀，还有蛙鸣
　　不要命的飞蛾，在猛烈撞击路灯
　　小跑到别人家，借来照明的烛
　　昏黄的灯光装满了整间房
　　烛光前书桌上，我刷刷写日记
　　六年前，我已断断续续地写
　　今天，我还在坚持着它
　　没什么可以阻挡我，即使凋亡
　　蜡炬即将成灰，不曾流泪，也不愿
　　放弃多年来的陪伴，有时候
　　有些事，真不需要理由去坚持
　　昏黄的烛光给红笔写就的日记
　　晕染出诗情与画意，还让我
　　想起，蓦然回首，那人还在烛光前
