# 我的尿滋出春天来

Created: March 21, 2024 10:34 PM
Text: 　　合上电脑，我写完日记，我打算
　　关掉电炉、熄灯。可尿意席卷全身。
　　开后门，我听见，清水哗啦啦～～
　　走到核桃树跟前撸开裤子，滋滋
　　被尿滋的核桃树，还是我种的！
　　转眼间，它竟碗口还粗！我想起：
　　北伐的桓温，树犹如此，人何以堪！
　　滋滋、滋舒服喽！春天就这样来啦。
　　春天来啦？不着衣裳、□□的紫荆旁
　　有那颗，同样是我栽种的万年青。
　　她长出嫩绿鹅黄的新叶，还有枝桠！
　　春天来啦，春天来啦，撸起裤子，我
　　发现春天来啦！春天似被尿滋来啦
　　有人读到这，无聊的话，那感觉肯定
　　就像头顶有乌鸦飞过，然后：。。。。
　　我的尿滋出春天来，有点鲤鱼冒泡泡
　　然而，人类的历史，何不是如此？
　　真相的背后充满谎言，还有虚无呀。
