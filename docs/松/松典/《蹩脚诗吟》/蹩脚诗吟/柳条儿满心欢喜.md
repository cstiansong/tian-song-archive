# 柳条儿满心欢喜

Created: March 21, 2024 10:34 PM
Text: 　　春日，灞桥的柳絮纷纷扬扬
　　漫天飞舞的雪花，落满你的长发
　　白杨树生长的地方，鸟儿织巢
　　清风阵阵，阳光洒满了姑娘的脸颊
　　夏季，我在香樟下纳凉，纳凉！
　　青青的鸟蛋，静静躺在窝里
　　冬季，我在炉火边取暖，取暖！
　　白白的雪花，静静落在旷野
　　清晨的蜘蛛，爬过夜晚吐露的网
　　我的心何处安放，何处扎营？
　　乡野间的石榴花，落了满地
　　你噗嗤笑了，然后走了，姑娘远去
　　思念漫延到，你的左右手……终于
　　牵着你的手，阳光洒满梧桐东路
　　腾飞塔旁，柳条儿满心欢喜
　　盛夏的夜晚，灞桥的河水随风涌动
