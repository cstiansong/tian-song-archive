# 高傲自负的读书人

Created: March 21, 2024 10:34 PM
Text: 　　热爱读书的小伙，来自农村
　　有着自卑、敏感、与多情的心
　　这样的心，命运却让他读书
　　让他写字，让他读书写字！！！
　　六年读书，六年养气，他知晓
　　他是个高傲自负的读书人
　　不曾询问别人，也不曾谦卑
　　喜欢关起门来苦干，闷头闷脑
　　读书方面，他高傲，还自负
　　也因如此，走了许多弯路
　　他读书第六年，猛地明白
　　高傲自负的读书人，这跟他
　　出身没有关系，命运让他读书
　　这次疫情让他反省，重新定位
　　决心，再也不像以前那样
　　安慰自己，鼓励自己接受，并
　　寻找当担，还有应对的方案
　　坚持文言、英文，辞典的阅读
　　也就是，与时代保持距离
　　纵然高傲，自负，走老路
　　也知道，也有谦卑的心
　　二十岁那年，他不高傲自负
　　他还真没有像样的成绩呢
　　如果，三十岁那年，还那样
　　照样没有像样的成绩！！
　　高傲自负的读书人，的心路
　　有内容，乐趣的破茧成蝶
