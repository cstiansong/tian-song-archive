# 命运的名字叫悲伤

Created: March 21, 2024 10:34 PM
Text: 　　夜凉如水，透骨夜风，吹乱我的头发
　　谁会在乎，谁会可怜，谁又能明白？
　　抓不住的温柔，像追不到的姑娘。
　　花朵凋零，叶儿落，我失去了欢乐
　　绿叶与根，它们是希望，有希望。
　　就不想放弃，可是命运的名字叫悲伤。
　　悲伤记不清，多少流逝的欢乐场景；
　　想不起，多少消失的思念与深情？
　　多少年以后，迎候的已是沉默。
　　落花有意，流水无情的单恋，
　　最难受，最无情，也最最无奈。
　　摇摇欲坠的太阳带走好儿些东西，
　　那些够刻骨、够铭心的记忆慢慢褪去
　　褪色的青春终究成为淡淡的痕迹。
　　命运的名字叫悲伤，希望也是悲伤。
　　希望落空以后，那种悲，悲到骨髓。
