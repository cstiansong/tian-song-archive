# 左右手提着东西

Created: March 21, 2024 10:34 PM
Text: 　　左手提着意义，右手提着不朽，
　　时间的长河，做收藏沙子的旅人。
　　我们仰望星空，我们收藏沙子……
　　左手提着意义，春风吹拂柔柳；
　　右手提着不朽，秋水淋漓梧桐；
　　左手提着意义，右手提着，不朽。
　　不朽是西，意义是东，是东西。
　　左右手提着的，就是东西~~~
