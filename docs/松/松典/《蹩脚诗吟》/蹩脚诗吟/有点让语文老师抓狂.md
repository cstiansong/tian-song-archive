# 有点让语文老师抓狂

Created: March 21, 2024 10:34 PM
Text: 　　山间的清风，与淙淙流水，就像风，
　　而落叶像蝴蝶，它们飞舞起来。
　　花间潜行的风，让清晨的露消失。
　　恒古不变的，唯有，人生无常。
　　爱情的翅膀，承载信念自由飞翔……
　　可是，翅膀太软，就像年轻的肩膀
　　负重前行，怎能飞高，怎能飞远？
　　一滴泪跌落，微弱的烛火熄灭。
　　我再也不谈及某些虚伪的话题……
　　每天就在图书馆读书写东西，
　　有点点期待，却也明白没有奇迹。
　　一页两页……一本两本地读着书。
　　读到末了，窗外的天空，夜色像泼墨。
　　我知道，好写文字都是凭空捏造……
　　它们就像我积累的情绪，倾倒的苦水。
　　于我似乎也没有什么意义，毕竟：
　　这些文字有点让语文老师抓狂。
