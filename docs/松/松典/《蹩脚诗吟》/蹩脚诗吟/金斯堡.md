# 金斯堡

Created: March 21, 2024 10:34 PM
Text: 　　前往广东广州的高铁上
　　我坐在窗户边翻看着金斯堡
　　心却在云贵高原上游荡
　　因为那片高原有音乐在翱翔
　　从广东深圳回来的飞机上
　　我的心却在珠江三角洲
　　因为洲边附近有我的朋友
　　即将坐在窗边翻看金斯堡
　　金斯堡却没法吸引我
　　我的心思真不在他身上
　　保持谦卑的心
　　现在的我理性、精于计算
　　但依然像曾经那样浪漫
　　我愿意顺从内心书写文字
　　保持谦卑的心，勿忘江湖
　　努力把埋在心的深处
　　的那个理想兑现成现实
　　虽然道路艰辛而漫长
　　但我真心渴望那天到来
　　那种实现梦想的感觉
　　让我觉得自己正活在人间
　　有真真切切的欢愉忧伤
