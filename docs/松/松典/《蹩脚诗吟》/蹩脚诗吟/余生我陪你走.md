# 余生我陪你走

Created: March 21, 2024 10:34 PM
Text: 　　夜里神来疯，我给初恋打电话。
　　她曾说我，你的三观不入流。
　　为什么，为什么？如此残忍~~
　　你知道吗？这样对我不公平。
　　我知道，我有点神来疯……
　　可是，我真不像你想的那样。
　　回忆裹挟着青春的记忆，落泪。
　　我可以应对蹉跎，余生我陪你走！
　　即使分离，也可以死灰复燃。
　　我真心爱过你，爱了就是爱了。
　　活在回忆里，我从未生活在别处。
　　我真的不像你想的那样……
　　如果你愿意，如果你允许，我
　　愿意追随你直到地老天荒。
　　灰烬的余烟，或许，消散~~
　　我可以应对蹉跎，余生我陪你走！
