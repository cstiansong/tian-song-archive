# 踩着影子回寝

Created: March 21, 2024 10:34 PM
Text: 　　踩着自己的影子回寝，形影相随。
　　这就是我每天的图书馆读书生活。
　　残灯将要灭亡，已有了坟的寒气，
　　那些未经世事的幼稚，悄然而往。
　　清风吹柏杨的树梢，树叶花啦啦……
　　漆黑的午夜里，沉重的花苞待放。
　　烟雾弥漫的从前，心事支离破碎。
　　日记早已泛黄，记忆模糊了起来。
　　你不要哭泣，花儿来年继续盛开，
　　忙碌里，读完，穷愁潦倒的大学。
　　终究变成，青春回忆里几页插图。
　　毛毛虫爬满整个花盆，也不落后。
　　有时候，有意义与无意义混合了。
　　也就不知道，这究竟有没有意义。
