# 他是个晚熟的人

Created: March 21, 2024 10:34 PM
Text: 　　晚风像鲤鱼游弋在水稻田间
　　山间的溪水哗啦啦地流
　　唱出欢快的歌，灵动的鸟
　　他不再像男孩伤春悲秋
　　生命按照固有的程序走下去
　　时间如流，遗憾孤身奋战
　　没有刻意遗忘谁跟谁
　　只是，岁月自动抹掉了光环
　　人世间最伤感的事情，是深爱
　　你的人跟你说，我不爱你了
　　青春有些事，虽然痛苦
　　却是春蚕吐丝，成长的需要
　　那进化，让他变了个人
　　没有不懂世故的娟狂
　　爱恨纠缠的暴戾消失不见
　　性格简单，他是个晚熟的人
　　晚熟的人继续等待，等待
　　等待的同时把生命注入文字
　　让文字长出翅膀，飞翔
