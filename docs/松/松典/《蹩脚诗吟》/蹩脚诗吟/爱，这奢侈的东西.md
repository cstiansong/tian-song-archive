# 爱，这奢侈的东西

Created: March 21, 2024 10:34 PM
Text: 　　你爱不爱我，答案从未改变
　　就算，再回首，我已泪流满面
　　可我爱的那个人，仍旧是你
　　时光流过，青春独有白痴？
　　或许，花落了，没有爱憎了
　　有的，只是可以表达的爱
　　成年人的爱憎，无须表达
　　哪怕就是爱，也要懂得克制
　　何必要，爱，这奢侈的东西
　　左右摇晃，跌跌撞撞追寻爱情
　　最终，还是那样，何必要？
　　爱，这奢侈的东西，为什么？
　　只因，爱或者喜欢，不重要
　　若心意都知晓，还需要点破？
　　爱，这奢侈的东西，何必要
　　该有的，总有，强求，也没有
　　你敢赌青春，那你的心，也要服输
　　既然，这奢侈的东西，得不到
　　何必再要，爱，这奢侈的东西？！
