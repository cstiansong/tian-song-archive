# 我喜欢你

Created: March 21, 2024 10:34 PM
Text: 　　我——喜——欢——你
　　与其说，这是爱你的宣言
　　不如理解为夜行少年的嚎叫
　　心里没底为自己壮胆
　　冬妮娅跟保尔他们纵然结合
　　也注定不会幸福
　　与其让心爱的人流落红尘
　　跟自己过贫贱的日子
　　不如让她活在记忆深处
　　就是有那么些人
　　要认定她美，纵然不美
　　也是花儿永不凋零
　　她让你遍体鳞伤让你发誓
　　我再也不想看见你
　　然而，再遇见她还那么美
　　那么温柔那么让你想入非非
　　她的眼神她的微笑依然
　　牵动你的情绪，撩拔你的爱情
　　让你想起还是少年的自己
　　曾鼓起勇气说：
　　我——喜——欢——你
　　2020.5.11象牙塔·钱馆
