# 柯希莫的命题

Created: March 21, 2024 10:34 PM
Text: 　　像翁布诺萨的男爵生活在树上
　　他发现，他只能来图书馆
　　余生都摆脱不了书籍的牵绊
　　秦川环绕沃土，似乎惋惜
　　他曾想抓住光、影，与念想
　　农村娃踩着云彩，眺望远方
　　想挣脱脚底这泥泞的土地
　　他的根，却在地里延展、生长
　　有些人，或许永远也追不上
　　仍愿意追随前辈们的步伐
　　突破自我的极限，成就自己
　　文字与书籍共存，不朽的事业
　　此生有限，那也值得追求
　　柯希莫要问前程，可他选择
　　生活在树上，远离人类的硝烟
　　辜负、告别、遗忘某些人事
　　他乃至抛弃青春的气味
　　愿自己成为柯希莫的命题
