# 诗人将死

Created: March 21, 2024 10:34 PM
Text: 　　写诗的热情即将消失殆尽
　　希望坚持到后面的完成
　　直到字斟句酌后它们出世
　　不是因为不再热爱诗歌
　　而是我经历了忧伤与挫折
　　逐渐深刻理解到生命
　　生命的心路历程曲折又复杂
　　不是简简单单几个字
　　又或者几个词就能诠释的
　　我想，随着时间的流逝
　　身体驻扎的诗人将死
　　理性是诗歌的天敌
　　柔软的情思让我很尴尬
　　我们这个社会的氛围
　　不允许上了年纪的人矫情
　　且诗歌的时代已经结束
　　像涸辙的鱼没有未来与希望
　　如果我不能及时踩住刹车
　　那么，苦苦追寻大概率
　　将不如许多人坐享其成的
　　这是没有办法的事情
　　因为理性是诗歌的天敌
