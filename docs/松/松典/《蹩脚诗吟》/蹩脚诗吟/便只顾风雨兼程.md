# 便只顾风雨兼程

Created: March 21, 2024 10:34 PM
Text: 　　猛烈的风雨，我都经历过了，
　　此刻的煎熬，又算得了什么？
　　我只想穷经皓首，翻阅书籍……
　　把眼睛看瞎，把生命挥霍到零！
　　可惜，我每晚都梦见裸体姑娘……
　　我以为，我想姑娘快想疯了。
　　我感觉我有点像聊斋里的书生，
　　为姑娘，我夜以继日地奋斗。
　　夜幕降临，我想姑娘想得难受……
　　我痛苦的眼泪，像断线珍珠，
　　落了满地都是，我还没有知觉……
　　我仿佛喜欢自嘲、激情，还有悲凉
　　风华正茂的我，人生路还很长。
　　强烈的欲望，却使我身心剧裂……
　　多少个夜晚，我辗转、难眠。
　　可我选择这路，便只顾风雨兼程。
