# 最爱你的那个人

Created: March 21, 2024 10:34 PM
Text: 　　一句话，一件事，一个眼神
　　我感动了。我以为，你喜欢我
　　野草在歌唱，有风吹过草坪
　　我想着你，念着你，深爱着你
　　夜凉如水，我鼓足勇气
　　我爱你，并且希望你嫁我为妻
　　可你却说我，像只癞蛤蟆
　　自尊被捅了刀，青涩的心流血
　　痴心不改，我等你心回意转
　　轻蔑的笑意挂在你的嘴角
　　我决心离开，离开你的世界吧
　　我没有勇气，对你不离不弃
　　一句话，一件事，不会一辈子
　　多少年以后，你终于明白
　　最爱你的那个人，其实是我
　　可，你明白了，又有什么意义
　　你把青春献给他，我不要残存的你
