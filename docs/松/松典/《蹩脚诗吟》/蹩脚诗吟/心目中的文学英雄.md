# 心目中的文学英雄

Created: March 21, 2024 10:34 PM
Text: 　　他翻阅了人世间出色的书籍
　　他走完生命个体的道路
　　他看到别人看不到的风景
　　他有真切、非同寻常的感受
　　他用文字恰当地记录风风雨雨
　　他的经历成为他的文学故乡
　　他，就是我心目中的文学英雄
　　他有超绝群伦的学力，跟笔力
　　他有笑傲江湖的见识，与雄心
　　他还有顽固、不可撼动的自信
　　他是开宗立派式样的文学英雄：
　　他以艺术的最高标准琢磨文字
　　以读书的最高境界对待读书
　　以继承的最高程度突破自我的极限
　　他希望他能够跑赢岁月的步伐
　　他要让文字长出翅膀，飞到天空
　　最后，落到大地的每个角落
　　燃烧生命的火焰，永恒、不灭
