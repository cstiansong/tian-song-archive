# 卡尔维诺的男爵

Created: March 21, 2024 10:34 PM
Text: 　　他看月悄悄跑到夜空，乌云掩盖了月
　　他聆听蟋蟀声、哇鸣，和鸟类啼叫
　　他跟随大自然，四季轮回地耕作
　　他养牛，他种地，他还要大片天空
　　他的脸，历经风雨，像风干的橘子皮
　　他把八九十年的岁月，献给土地
　　他的手，饱经沧桑，像干瘪的老树根
　　他把自己的灵与肉沉到泥地深处
　　他说话声老大老大，极似渴望聆听
　　他吵得脸红脖子粗，却似跟自己
　　他老重复自己，重复自己，自己的话
　　他重复自己的话，直到，我都长大
　　他的儿女有七八，有死、有残，有活
　　他的孙辈、外孙辈都有十八九
　　他的曾孙辈，都有三四，也有成年
　　他的内心丰富，他太孤独，他太寂寞
　　他看着我长大，而我看着他，变老
　　他的时间把他削瘦，摧倒、风干
　　他只剩骨架，丑陋的皮，还有精气
　　他要告别天堂，告别土地，告别我们
　　他是谁？他不是谁，他是我身边的老人
　　他就像个精灵，像卡尔维诺的男爵
　　他有时候，不知好歹，蛮不讲理，可
　　他就像个孩子，尽管，他八九十岁
