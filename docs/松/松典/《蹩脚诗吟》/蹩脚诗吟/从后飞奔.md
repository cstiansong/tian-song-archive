# 从后飞奔

Created: March 21, 2024 10:34 PM
Text: 　　窗外有白色棉花般云朵
　　像极了柔软的绵羊
　　窗内是诗人穆旦的语言
　　像爱情像女人的诱惑
　　遥远的从前到近在我眼前
　　近在眼前的所有是未来
　　我们从前慢慢来从后飞奔
　　谁也阻挡不了神的觉醒
　　没土壤没绿草哪儿有红花
　　哪儿有高耸云端的大树
　　诗便是红花便是树梢结果
　　还是娇艳的欲望在流淌
　　也是瞬息的灵感积累太多
　　如山涧清泉飞溅在天涯
　　这辈子写哪个样子的文学
　　早被注定已是宿命
　　从前是慢慢来从后飞奔
　　严酷的现实迟早让少年
　　脱衣服般脱掉青涩
　　青春啊青春消逝在天际
　　那朵云彩那朵莲
　　我想还是会从后飞奔
　　2019.8.5象牙塔·钱馆
