# 我都冲在最前方

Created: March 21, 2024 10:34 PM
Text: 　　一个劲，两个劲……
　　我无休止地往前冲！
　　每次，我都冲在最前方，
　　即使粉身碎骨……
　　偶有两夜，我失眠了，
　　失眠的我心脏难受。
　　那消沉的热诚，与热血，
　　在夜空中，猛烈燃烧。
　　火车醉了，无所谓。
　　因为它有轨道，不会出轨。
　　人要是迷茫，可不行，
　　因为时间宝贵。我感叹：
　　时间匆匆，太仓促！
　　还来不及挥手，说再见。
　　时间匆匆，太仓促！
　　还来不及告白，转瞬消失。
　　我真不想虚伪得过活，
　　可现实残酷，人生悲凉。
　　我什么都改变不了，
　　我确实什么都改变不了
　　可我依旧故我地坚信：
　　千万别掩饰你的野心，
　　那样只会让你虚伪。
　　即使虚伪也许会受益，
　　一个劲，两个劲，
　　我无休止地往前冲！
　　每次，我都冲在最前方，
　　即使粉身碎骨，也无畏
