# 清水，与黄沙

Created: March 21, 2024 10:34 PM
Text: 　　书页纸扉间，我窥见很温柔的姑娘，
　　却又不可言语，以免说出，你好色!
　　茫然忧伤。愁云惨淡里，早已忘却，
　　日记泛黄的事实，记忆不觉间模糊。
　　漫天的花瓣向谁飞扬，又飘落远方
　　忙碌的生活，难以抚弄野草的孤寂。
　　烦闷的钱馆，忘记姑娘究竟在何方。
　　总有千百个理由，我不相信，现实。
　　无奈接受，只因为在一起，就是对。
　　思念就像枯黄的梧桐叶，落在窗前。
　　绿草凋残，花儿谢落。勉强的辞藻。
　　迟早发生的那些事，为何人生无常。
　　那里一湾清水，清水有清水的流向，
　　这儿一剖黄沙，黄沙有黄沙的归属。
　　清水与黄沙，偶然间，遇见后再见，
　　问道人生，不管怎样，是否，是诗。
