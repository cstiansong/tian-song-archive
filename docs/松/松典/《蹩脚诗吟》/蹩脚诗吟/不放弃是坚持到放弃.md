# 不放弃是坚持到放弃

Created: March 21, 2024 10:34 PM
Text: 　　坚持到放弃后，潇洒点离开。
　　早知这样，我还是读书写情书！
　　而不是，浪费时间，与惆怅……
　　心灰意冷，我也就不得不离开。
　　到啦尽头，我还能怎样？！
　　说服自己，没什么，忘记吧！
　　人生苦短，别再管，珍惜时光！
　　多读诗，偶尔写诗，追求作家梦。
　　文字世界，我还可以呼风唤雨……
　　对方有意思，也不会有某些事。
　　我不能对绝情的人，有太多执念。
　　毕竟，喜欢谁或说爱谁，已不重要。
　　我心里踏实，有喜欢的，就好。
　　海退潮，螃蟹横行，礁石突突冒出来。
　　礁石对螃蟹说：有喜欢的，就好。
　　人这辈子，心里踏实，认真爱过，
　　至于是否白头，是否偕老……不重要。
　　不放弃是坚持到放弃，然后潇洒离开。
　　当然，总得走到最后，否则，没有结果。
