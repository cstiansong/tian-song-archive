# 像黄狗那样守护你

Created: March 21, 2024 10:34 PM
Text: 　　我特别想把那些画面记录成文字，
　　因为我曾想，要像黄狗忠诚守护你。
　　肩披黄色的灯光，想起纯朴的你。
　　窗外，雪夹杂雨的心意，滴答滴答。
　　青草住在姑娘的腰间，曼妙生姿。
　　水缸里，住着几只小鲤鱼，悠闲……
　　鱼亲吻干涩的嘴唇，为花带来鲜活~~
　　这些就是转瞬即逝却生动鲜活的画面。
　　我时常做梦，我时常梦见这些画面。
　　有时，无梦的睡眠，就是闭眼到睁眼。
　　遭受可怕的折磨，也是闭眼到睁眼。
　　你那眼睛里，没有了昔日的荣光，
　　可曾想过那个为你而生的世界？
　　或许，来自西伯利亚的寒流寒彻身心，
　　我在北方却过着那南方人的冬天。
　　我曾想，要像黄狗那样忠诚守护你。
　　可终于明白，赤诚抵不住现实的冰冷。
