# 以温柔的目光注视你

Created: March 21, 2024 10:34 PM
Text: 　　柳树和芦苇丛，有鸟儿编织鸟巢。
　　柔顺的柳条，随清风轻轻摇曳……
　　这时，河水冒泡，冒出血红的泡泡。
　　心已死，适可而止，有情人的眼泪。
　　孤零零仰躺在荒原，我彻夜仰望星空。
　　人生是苦，回首苍凉，孰是孰非？
　　鸟饱尝芳香的美果，我却心如死灰，
　　像山顶的积雪，被太阳晒得融化成水。
　　她心潮起伏，火焰在胸膛燃烧，
　　微笑留在她嘴角，她好像有话要说：
　　情人的眼泪，全是虚假的，别相信
　　世界与你无关，它走远，或许毫不存在
　　白杨树生长的地方，我的情思环绕你、
　　陪伴你、思念你，以温柔的目光注视你。
