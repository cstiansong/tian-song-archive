# 我是大堡村的树

Created: March 21, 2024 10:34 PM
Text: 　　我是棵树，我天生就在大堡村，
　　我的根在大堡村生长、延展。
　　随着阅历、见识、视界的增长，
　　我将根深蒂固，我还要枝繁叶茂。
　　我相信，这棵树迟早结出果实，
　　或许，还会有种子飘向远方。
　　但我明白，但我知道，但我铭记：
　　我是棵树，我天生就在大堡村，
　　我的根仍在大堡村生长、延展
　　他将根深蒂固，他还要枝繁叶茂。
　　白云停泊，绿水环绕青山，这儿
　　有棵树，大堡村读书最多的树！
　　他懂：成熟不是抱怨，也不是吐槽
　　而是在这孤单而漫长的岁月里
　　使劲笑出声，还要有从容与自在。
