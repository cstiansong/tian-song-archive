# 我隔着距离望着你

Created: March 21, 2024 10:34 PM
Text: 　　你是太阳，可我是孤独的月亮……
　　你可以爱我，但我不会嫁给你的。
　　我可以爱你，可我要心甘情愿。
　　愿就愿，不愿的话，就请你离开。
　　原谅我，当初不辞而别，我无奈。
　　对不起，我没有勇气爱，我残伤。
　　原谅我，当初犹豫不决，我也无辜。
　　对不起，我要转身离开。我很软弱。
　　思念到底怎么了，它可以远到哪里？
　　远到地久天长，为此，执念天涯？
　　夏日阳光，惨白、耀眼却没有诗意。
　　生命迟早会让我们遗忘很多东西，
　　尽管，那些东西曾以为要记住很久。
　　你是太阳，你是世界的中心……
　　可我是月亮，我隔着距离望着你。
