# 那边是山，不是海

Created: March 21, 2024 10:34 PM
Text: 　　夏日间，徐徐暖风吹过来，
　　星光暗淡，群星渐渐零落。
　　青草间，花香仍旧飘逸着：
　　狂风暴雨，柏杨没有羁绊。
　　学校里，埋首的我在读书，
　　凌云志落空成了水与落花。
　　仿佛散落枯树枝桠的月光，
　　从此以后，我再没有想起。
　　树林里，漫步与思索人生。
　　叶面的露珠，与虫儿吃草。
　　潋滟波光还在图书馆跟前，
　　轻轻颤动。山那边不是海。
　　山的，那边是山，不是海，
　　未经世事，幼稚生了闷气。
　　多年以后，回想起那时候：
　　农村是青春的坟墓，这话。
　　断定，那边是山，不是海。
