# 她在诗的国度流亡

Created: March 21, 2024 10:34 PM
Text: 　　海岸边海浪翻滚，回旋，拍打着礁石。
　　鸥在遨游，沙滩头，有白裙姑娘哭泣……
　　白裙姑娘为什么哭泣？还这么可怜。
　　孤独的心，伤痕累累，她哭诉：
　　他没有承诺，就没有落泪的辜负。
　　他说做不了情侣，也别做朋友。
　　她不想看见他，以免，身心痛苦。
　　青春像个姑娘，站在河的对岸观望。
　　她想，世界原来这样，何必纠缠？！
　　她内心，死水波澜，也毫无风浪。
　　她想：未曾拥有，已分离，算什么？
　　她化悲愤为力量，在诗的国度流亡。
　　她把所有的忧伤都写成了诗……
　　她的行为有点像那个失恋以后，写出
　　少年维特之烦恼，的那个文学家。
　　从此，她志愿在诗的国度流亡。
　　她天涯海角持初心，矢志不渝……
