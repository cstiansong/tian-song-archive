# 多情的乡巴佬

Created: March 21, 2024 10:34 PM
Text: 　　不经意间，乡巴佬走过特定年龄
　　从此，不再有，伤春悲秋……
　　生命按照既定程序，继续向前
　　漫长的寒假结束，他终于明白：
　　我追逐繁星，我渴望城市的繁华
　　我通过大学正在远离我们村庄。
　　其实，我就是个，多情的乡巴佬
　　踏过干裂土地，吹过凉爽的风，
　　拂晓蔷薇色的天空，还残留痕迹
　　可我没有留恋，唯有追逐繁星！
　　因为，我就是个，多情的乡巴佬
　　人与人的邂逅，认识，欢喜……
　　这都是，老天爷恶作剧的安排。
　　今后，只希望，只愿意，只想要：
　　梦想中的高楼，千万别有偏差。
　　因为，我就是个，多情的乡巴佬
　　他希望，他渴望，他还有野心！
　　他渴望实现，目标，还有野心。
