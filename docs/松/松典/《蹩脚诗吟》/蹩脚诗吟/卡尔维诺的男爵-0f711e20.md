# 卡尔维诺的男爵

Created: March 21, 2024 10:34 PM
Text: 　　他看月悄悄跑到夜空乌云掩盖了月
　　他聆听蟋蟀声、哇鸣和鸟类啼叫
　　他跟随大自然四季轮回地耕作
　　他养牛，他种地，他还要大片天空
　　他的脸历经风雨像风干的橘子皮
　　他把八九十年的岁月献给土地
　　他的手饱经沧桑像干瘪的老树根
　　他把灵与肉沉到大地深处
　　他说话声老大老大极似渴望聆听
　　他吵得脸红脖子粗却是跟自己
　　他老重复自己，重复自己，自己的话
　　他重复自己的话直到我都长大
　　他的儿女有七八有死有残有活
　　他的孙辈外孙辈都有十八九
　　他的曾孙辈都有四五也有成年
　　他的内心丰富他太孤独他太寂寞
　　他看着我长大而我看着他变老
　　他的时间把他削瘦，摧倒、风干
　　他只剩骨架丑陋的皮及精气
　　他要告别天堂告别土地告别我们
　　他是谁？他不是谁，他是我身边的老人
　　他就像个精灵像卡尔维诺的男爵
　　他有时候，不知好歹，蛮不讲理，可
　　他就像个孩子，尽管他八九十岁
