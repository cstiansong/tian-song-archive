# 红豆

Created: March 21, 2024 10:34 PM
Text: 　　在地平线边缘，我画个圆圈
　　在圆圈里用手指轻轻
　　写下，“你是我痴情的姑娘，
　　而我是那粒相思的红豆”
　　是的，我是想念你的红豆
　　没看见你时我想念你
　　我看见你时，也要想念你
　　只有你永远消失在我跟前
　　或像涸辙鲤鱼那样
　　可以相互吐泡泡拯救彼此
　　换句话来说便是下面这个样
　　下面这个样，你自己想象
　　每天睁眼闭眼都可以瞧见你
　　每天上班下班都可以看看你
　　你内心的冰块被我融化
　　你脸上的冷漠还被我赶走
　　你想要的那些我会努力给你
　　即使现在没有的也会争取
　　我是典型的相思红豆
　　我想成为你的粥进你的肚
　　化作你的血，还进你的婚房
　　陪你看惯秋月和春风
　　2022.6.11象牙塔·钱馆
