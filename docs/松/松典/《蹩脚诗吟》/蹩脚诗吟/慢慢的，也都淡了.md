# 慢慢的，也都淡了

Created: March 21, 2024 10:34 PM
Text: 　　我栽种的小花，今年绽放过了，
　　明年还有可能招蜂引蝶……
　　我相信，所有的事情，
　　时间都会给命运有个交代。
　　漆黑的夜，站在腾飞塔前思索。
　　你偶有几点黑，可我面对的，
　　却是整个漆黑的夜晚，整片整个~~
　　九十九本书籍，像九十九朵玫瑰；
　　九十九封情书，则像九十九杯美酒。
　　走在回馆的路上，我想此刻的你：
　　在回寝途中，你可否驻足观望？
　　全交大，我博览群书，我读书最广~~
　　可我想不出办法，窃取心上人的心。
　　临近毕业，慢慢的，也都淡了。
　　此生何寄，此心何安，不重要了。
　　联系出现，命定的她自然出现。
