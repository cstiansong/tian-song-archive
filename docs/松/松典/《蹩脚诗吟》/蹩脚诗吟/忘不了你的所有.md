# 忘不了你的所有

Created: March 21, 2024 10:34 PM
Text: 　　该忘的东西，我还是，忘了吧
　　留恋式地带着，也是种负担。
　　知道吗？也许，你知道的：
　　忘乎所以地思念你，我曾以为
　　你同样思念我……没有怀疑
　　想你的时候，就想跑到你跟前
　　跟你说，我爱你，真的爱你
　　文字的话，我可以毫无顾忌！
　　我爱你，很爱你的那种爱你
　　可面对面，却没有那个勇气
　　我不知道为什么，我说不出口
　　或许，这是天意，也是缘分
　　你是流星，深邃夜空里的流星
　　你是我的流星，从我生命里划过
　　你不记得我，也不曾留恋我
　　可是，我忘不了你的所有
　　我知道，我从来没有爱你的资格
　　我的文字，就跟我，同样蹩脚
　　可我仍旧爱你，直到天荒地老
