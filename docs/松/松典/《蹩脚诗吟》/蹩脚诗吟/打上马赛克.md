# 打上马赛克

Created: March 21, 2024 10:34 PM
Text: 　　有时冲动得想把追求打上马赛克
　　转念想想，反正也无所谓了
　　这或许就是我任性随意的个性
　　它到底是怎样来的，也无谓
　　关键是今后的人生要充分利用
　　否则的话，没有好果子可吃
　　山间的夏秋已经远去，我每天
　　都会看见静静燃烧的鸡冠花
　　在它们心底，是否荡漾起激情
　　我觉得我们像老朋友那样
　　深入内心与它们的灵魂对话
　　亲近大自然，亲近生活的角落
　　美学呵无处不在空气当中
　　就让它们把追求打上马赛克
　　进而让我沉下心把事情想清楚
　　想清楚为生活忙忙碌碌奔波
　　不是我想要的岁月想要的人生
