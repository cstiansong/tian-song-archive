# 可曾懂我的悲伤？

Created: March 21, 2024 10:34 PM
Text: 　　随意变幻的风，吹拂大地和杨柳
　　天边的云朵，可曾懂我的悲伤？
　　多少次，我看见年老的窗台
　　落满了灰尘，只因为，我失去了你
　　憧憬的未来，雨的夜晚，忽然倒塌
　　倒塌了的未来，扬起漫天的灰尘
　　因了我的悲伤，灰尘，也沉重了
　　随意变化的风，只吹拂杨柳
　　天边的云朵，你可曾懂我的悲伤？
　　天空，历史，还有春风春水
　　我想你，也想哭，谁叫我牵挂你
　　流血开花，日渐成熟的我，想问你
　　人生何所似，为什么这么悲凉！
　　亲爱的，你可曾懂我的悲伤？
　　我满怀希望，满心欢喜，可还是心碎
