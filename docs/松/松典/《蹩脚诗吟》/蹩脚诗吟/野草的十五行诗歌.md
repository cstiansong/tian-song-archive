# 野草的十五行诗歌

Created: March 21, 2024 10:34 PM
Text: 　　初秋的风早已经离开，冬天的心腹了。
　　还在这儿，这就好像滔滔不绝的人们，
　　说了许多话后，又还有什么话可以说？
　　什么也不想提，可人们依旧还在那儿。
　　在生命的彼岸，那儿有好些停泊的船，
　　那些船儿里有迷人的野草，深情歌唱。
　　闲花、棋子与山峰，忘我般迷恋彩云。
　　然而，云彩听不见，野草深情的歌声。
　　野草曾想，耀眼的流星都会转瞬成空，
　　这就好像是，他也明白阴霾注定消散。
　　物依旧，人依旧，心境却不再有……
　　秋草开了黄花，野草虽然叫不出名字。
　　可是，那已经不重要，这就像梦醒了。
　　虽然吓了我几跳，可是还在半夜的我，
　　除了继续睡觉，没什么可为，不是么？
