# 献给海子的诗

Created: March 21, 2024 10:34 PM
Text: 　　花了好几天，今天读完啦
　　海子的诗全集，主编是西川
　　亲爱的海子，山海关，那是
　　你生命终结的地方，我知道
　　亲爱的海子，我不知何时起
　　迷茫的青春，老在心头纠结
　　诗意，宣泄了鱿鱼的悲伤
　　诗歌，燃烧起青年的文学梦
　　你为情而粟，也为情而亡
　　情为何物？或许，生死相随
　　卧轨，那是你的权利，可是
　　你轻生，那就是，你的不对
　　死亡并不辉煌，你我都知道
　　身体如烂泥，是生命哭诉
　　也是生命的绝望，风吹血冷
　　寂静的月光散落带血的铁轨
　　现实主义泛滥，横行，弥漫
　　文艺这词儿，似失去了光彩
　　浪漫主义暗淡，消亡，式微
　　诗歌这词儿，似失去了生命
　　你的死，就是现实主义泛滥
　　你的亡，就是浪漫主义消亡
　　这，我写给你的诗，愿安好
　　写得不好，也莫怪，也无妨
