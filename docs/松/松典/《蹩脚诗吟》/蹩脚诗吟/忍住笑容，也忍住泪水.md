# 忍住笑容，也忍住泪水

Created: March 21, 2024 10:34 PM
Text: 　　我忍住酸涩的笑容，也忍住了泪水
　　那天傍晚，夜色渐浓，我把控自己
　　忍住酸涩的笑容，也忍住泪水滚落
　　我恐惧未来，可我老想着未来……
　　想着想着，我的泪水就从眼眶流淌
　　我不是爱哭的男人，可是，我哭了
　　泪人儿的悲伤，眼睁睁，看着想着
　　可是，本就陌生的我们，又能怎样
　　认识，了解，相知，爱了，然后走
　　这样的结局，我很在意，也很难受
　　我忍住酸涩的笑容，也忍住了泪水
　　就算泪水的滚落，那也是心的坚强
　　夜莺在寂寞的夜里鸣叫，婉转悠扬
　　可因为夜晚的黑，就好像杜鹃啼血
　　原野尽头，我对这世界，彻底死心
　　今后不再谈论虚伪诸如爱情的话题
　　漆黑的角落，有朵野花
　　漆黑的角落，有朵野花寂寞地开，
　　它说不出爱，可它沉重地爱着远方；
　　沉浸泥沼的它，在漆黑的角落里……
　　独自为你绽放，只因深深地爱着远方。
　　远方有谁？或许有诗意，或许没有。
　　可有清风吹拂，那朵野花在哭泣……
　　怎么啦？原来，清风把花瓣裹挟走了。
　　野花想到的不是自己，而是远方。
　　远方有诗，远方有意，远方有诗意！
　　可是，远方没有那朵为你哭泣的野花
　　亲爱的朋友，你可曾明白，你的身边
　　落花不是无情物，化作春泥，还爱你。
　　落魄的单恋，转瞬消失，野花没啦！
　　那朵寂寞的野花，到哪里去了？
　　它被清风带走，或许，它已到远方。
　　落花不是无情物，我愿野花追随……
