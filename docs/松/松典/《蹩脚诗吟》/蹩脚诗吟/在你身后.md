# 在你身后

Created: March 21, 2024 10:34 PM
Text: 　　此时此刻在你身后
　　有人注视着你有你的幸福
　　你只要伸出手或点头同意
　　他的生命都将属于你
　　他有好些好些话想和你说
　　有好些好些事想和你做
　　可他还没有说没有做的时候
　　却犯下傻得不能再傻的错
　　你知道吗？你在他心底深处
　　像火把就是他的光明
　　在阴暗的角落照亮他的生命
　　他对你没有半点儿□□是你
　　像玫瑰吸引还使他迷醉
　　他满脑子都是你完美的笑容
　　他奋不顾身地想要爱你
　　那瞬间他走在男人的路上
　　他不管你是否遭遇尘世污染
　　不在乎你心里有没有位置
　　你在他心底儿永远完美无瑕
　　你天真纯洁温柔还可爱
　　也许是拖欠，也许是人性
　　不管怎样他都认你好
