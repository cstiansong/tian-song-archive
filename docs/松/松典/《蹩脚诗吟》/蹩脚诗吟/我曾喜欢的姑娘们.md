# 我曾喜欢的姑娘们

Created: March 21, 2024 10:34 PM
Text: 　　有种情愫叫痴心，有种感情叫爱情，
　　还有种念想，我们习惯叫遗憾……
　　我曾在心底说过千万次，我喜欢你
　　可她们终究没有喜欢我，我遗憾……
　　小学遗憾，中学遗憾，大学也遗憾。
　　我慨叹：俄狄浦斯式的悲剧命运。
　　然而，时间洗尽铅华，味道却变了。
　　反倒庆幸自己，打小就喜欢姑娘。
　　虽然从未说出口，我真心爱你~~
　　虽也从未喜欢我，记忆到底，还行；
　　或许，那些姑娘曾为真心而感动……
　　可后来，我们终究败给残酷的现实。
　　我内心最柔软的地方，还记着她们。
　　发生，就让发生，没有偶然的发生。
　　我接受所有的发生，我埋怨自己：
　　没有能力把喜欢的姑娘，留在身边。
　　如果我曾真心喜欢她们，那我此刻
　　应该：勤学，苦练，博览群书地出彩
