# 朴素的姑娘

Created: March 21, 2024 10:34 PM
Text: 　　清冷的河岸边，绿头鸭悠闲地玩耍。
　　我喜欢，朴素的姑娘。素衣似雪。
　　雪白的皮肤，□□的胸，修长的腿。
　　如此，也惆怅，也叹息，流年逝水。
　　夏日炎炎，高山常在，绿水要长流。
　　忙碌的罅隙，憧憬天空，与爱情。
　　多少长夜，衣服湿透，泪水已流干。
　　烈火寒冰般，饥渴难耐，还要姑娘。
　　没有你的风景，风景仍然是风景。
　　多情自伤，不多情，杨柳却如丝丝。
　　樱花零落，梧桐似雨，春光易泻，
　　朴素的姑娘，你在何方我亦飘零。
　　所谓，人间惆怅事，乌龟不管不顾，
　　虽是，如此煽情泪，却也无风无雨。
