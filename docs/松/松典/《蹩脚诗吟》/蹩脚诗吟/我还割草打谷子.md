# 我还割草打谷子

Created: March 21, 2024 10:34 PM
Text: 　　晚风吹山间的稻田，圆月悄悄现身。
　　雨水伴着电闪雷鸣，稻谷间，蛙声连连----
　　清晨的溪水，流得可是，特别欢快。
　　或许，此刻有种喜欢，便是丰收！
　　丰收？当然，他们还要割草打谷子！
　　打谷子？打谷子就是收割水稻。
　　你可不知道，手捧金灿灿的谷子！
　　他们心里多么欢快！多么美滋滋！
　　乡间的青草，到豪门的满汉全席，
　　有叹息，叹息的，不是他们打谷子。
　　叹息的，其实是我，我叹息前途命运！
　　亲爱的朋友！你可知，生命的沉重：
　　轻快的微风，鲜艳的花朵，消失……
　　年轻的翅膀承载不了，那些儿沉重。
　　还是割草打谷子吧。这不会心累。
