# 我想考研历史系

Created: March 21, 2024 10:34 PM
Text: 　　我想考研历史系，就是历史系！
　　如果三年前，我没有那么的随意
　　现在的我，或即，历史系保研
　　又或，我已在准备考研，历史系
　　如今，我知道，我明白，这条道路
　　已行不通。老实说，我挺后悔，随意
　　我发誓，现在起，我要多读历史书
　　多读历史书，还要多翻看历史教科书
　　等汉语言文学结束后，等我大学毕业后
　　等我安心做完五年的语文工作者后
　　如果对知识的渴求，对历史的热爱
　　对考研历史系的想法都没有改变
　　我就要考研历史系，两次没过
　　我还要考第三次，我不想轻易放弃
　　如果读研，如果成为历史系研究生
　　我渴望把历史学博士学位弄到手
　　这个想法，是在刚刚才诞生的
　　虽然冲动，虽然幻想，虽然离谱
　　可是我，就这么想，也就这么认为
