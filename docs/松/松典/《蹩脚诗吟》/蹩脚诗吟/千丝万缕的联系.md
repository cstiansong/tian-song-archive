# 千丝万缕的联系

Created: March 21, 2024 10:34 PM
Text: 　　读书、经历、思考，视界越为宽广。
　　猛滴发觉，认识什么人，走什么路，
　　遭遇什么事情，懂得什么道理……
　　其实都已注定，尽管纷繁、紊乱
　　尽管？尽管它们八竿子打不着！
　　可那背后，仍有千丝万缕的联系。
　　那是，命运的安排。安排吗？
　　命运的安排是天意，无形的手。
　　无形的手？它像风，像心，像情
　　即使最出色的哲学家，也不能否认
　　天意的存在，因而，安排的命运：
　　注定识人，走路，明理，遭遇事情！
　　朋友啊！千万要善于发现，与感受。
　　你自然明白：即使最寻常的意外，
　　其实，也有着千丝万缕的联系。
　　那就是，我所说的，命运的安排。
　　千丝万缕的联系。换个意思说，
　　即使微不足道的事情，也绝非偶然。
