# 避难所

Created: March 21, 2024 10:34 PM
Text: 　　有些阅读为考试，为技能
　　不是心目中的阅读我以为它
　　不教你谋生、不教你行走
　　不教你世故不教你圆滑
　　心中所指的阅读真正的阅读
　　它是享受、是乐趣而不是负担
　　乐意沉浸其中还把世界遗忘
　　诱惑远胜勾引姑娘，甚至销魂
　　给我们筑造避难所逃离悲哀
　　它是可以随身携带的避难所！
　　那里面有热爱思考的灵魂
　　还有梦幻般享受纯粹的愉悦
　　你是最好的文艺理论批评家
　　哲学激活头脑、散杂文诱惑感官
　　而那些诗歌，让灵魂高涨
　　亲爱的！寻找优秀的作品吧！
　　迟早，你都会有：
　　富饶而轻盈的心灵！
