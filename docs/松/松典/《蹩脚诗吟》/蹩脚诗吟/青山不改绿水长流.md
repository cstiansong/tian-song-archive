# 青山不改绿水长流

Created: March 21, 2024 10:34 PM
Text: 　　微风，露珠，清晨，你遇见的
　　艳阳的笑容，如春风，如春水。
　　或许，他曾泪流，他曾心痛。
　　只是，你不明白，也不曾感受！
　　我想你了，你不明白，不曾感受
　　你永远都不会明白，我想你了。
　　我笑容背后，因想你而满目疮痍……
　　今晚的月色，或许，很美很美。
　　可，再美的月色，终究是冰冷的。
　　你要么，是我的，要么就离开----
　　离开，我难受，或许，再也不见。
　　我在心里，默默祈求千万遍。
　　我想你，我念你，我爱你喜欢你！
　　可是，你没有听见，你也听不见！
　　如今，无法挽回，心爱的姑娘。
　　这时候，我理解了某句话：
　　落花有意，我是那朵有意的落花；
　　流水无情，你是那泓无情的流水。
　　那好，我们就青山不改绿水长流吧
