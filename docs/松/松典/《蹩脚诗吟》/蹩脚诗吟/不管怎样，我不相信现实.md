# 不管怎样，我不相信现实

Created: March 21, 2024 10:34 PM
Text: 　　一湾清水，清水有清水的流向，
　　一剖黄沙，黄沙有黄沙的归属。
　　清水与黄沙，偶然间，遇见再见，
　　问人生，不管怎样，我不相信现实。
　　偶有思念像枯黄的树叶，落在窗台。
　　花儿落，北风吹残，哪怕绿草凋残……
　　总有千百个理由，我不相信现实。
　　发生了的某些事告诉我：人生无常。
　　漫天的花瓣向谁飞扬，又飘扬远方？
　　这就是人生无常。愁云惨淡，茫然。
　　日记泛黄是事实，记忆不觉间模糊。
　　就算纸扉间，我窥见温柔的姑娘~~
　　忙碌的罅隙，似有卑微的野草的孤寂。
　　烦闷的钱馆，我忘记姑娘在何方……
　　总有千百个理由，我不相信现实。
