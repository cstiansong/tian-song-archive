# 荒芜之大地

Created: March 21, 2024 10:34 PM
Text: 　　身边山咔拉就是荒芜之大地
　　迟早孕育出作家浪淘沙
　　他属于我们这个时代这个群体
　　只是，现在的他还缺乏女人
　　没有肥沃福地的滋润
　　这样的青春岁月苍白像苦行
　　苦行的孩儿敏锐觉察到
　　大量田野竟成荒漠
　　大量的土地没有人耕耘
　　那些荒漠那些长满杂草的地
　　有咯哒咯哒的山鸡妈妈
　　领着就像麻雀那样的小鸡
　　偶有弯曲的小蛇兄弟
　　像绳子像鱼那样游弋其中
　　我们这代人生长在农村
　　没有种地，也没有扛锄头
　　更没有面朝黄土背朝天
　　可我们有我们的绝望
　　浪淘沙你为荒芜之大地而生
　　不是浪淘沙选择了大作家
　　而是大作家自己选择了浪淘沙
　　像艺术志向这种事情永远
　　都是老天爷的赋予
　　大地需要思考与记录的人
　　2022.8.5故乡·大堡村
