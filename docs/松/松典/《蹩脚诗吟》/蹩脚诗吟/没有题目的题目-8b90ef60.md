# 没有题目的题目

Created: March 21, 2024 10:34 PM
Text: 　　风儿吹，草儿动，蓝天白云……
　　荒野长满杂草，小蛇在草间游动。
　　这场景为何老在我的梦里出现？
　　最终，我取名为：没有题目的题目。
　　那场景，在南方小镇的荒山野岭，
　　我好像遇见过，有那么点熟悉。
　　或许，我潜意识对她有种想念吧。
　　北方桂花开了，我想起南方的月桂，
　　我不想月桂，而想像月桂的姑娘……
　　她声音轻柔而美丽，像春风像春水。
　　就算离别，我也曾想，我喜欢你。
　　只是生命抓不住风，等不来你。
　　如果你冷，我想抱你，想给温暖，
　　有些话想和你说，有些事想跟你做……
　　当初，有意无意遇见，也没留痕迹。
　　有好感的姑娘，仿佛数不清楚。
　　可是，再也没有你的温柔。
