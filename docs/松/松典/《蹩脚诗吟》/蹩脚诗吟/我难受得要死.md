# 我难受得要死

Created: March 21, 2024 10:34 PM
Text: 　　晚霞把教学楼前的梧桐染红。
　　有颗多情的心，想你，想得难受----
　　你的声音，宛如缭绕的音乐；
　　你的容貌，就像洁净的蓝天白云
　　你在我心里散发出圣洁的光芒，
　　披着神圣的光环，你就是我的女王
　　姑娘，做我女人！让我疼你爱你！
　　照顾你！叫我怎么，我都心甘情愿
　　爱情，就是拿得起，偏偏放不下。
　　你不愿意，我也痛苦地爱你，爱你，
　　就像树根在地里纠缠，就像心与情。
　　我喜欢你，你怎能，如此待我
　　人的感情，可以控制，怎么可能？
　　落叶就像蝴蝶，它们飞舞了起来；
　　你可知道，我有多么多么想念你
　　没有月亮的夜晚，夜凉如水，孤寂
　　没有姑娘的岁月，我煎熬我彷徨……
　　我的善意，我的爱意，被黑心人，
　　狠狠装进了棺材，我难受得要死。
