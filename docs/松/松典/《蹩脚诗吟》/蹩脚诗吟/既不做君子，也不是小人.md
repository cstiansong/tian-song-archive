# 既不做君子，也不是小人

Created: March 21, 2024 10:34 PM
Text: 　　事情该了则了，千万别留恋。
　　留恋对谁都没有好处，只是徒劳
　　岁月就像缓慢流动的清水，
　　点点滴滴终究勾勒出青春的模样
　　这个世界，没有巧合，与偶然
　　就算微不足道的事，也不是意外
　　只想及时享乐，离别以后不再相念
　　我没有那么崇高，也不那么伟大
　　我既不做君子，也不是小人
　　首先，我真实，然后，要坦诚
　　假使你认定我虚伪，请自便
　　因为我既不做君子，也不是小人
　　我蹉跎岁月，岁月，也将蹉跎我
　　回忆若是惨败，那是我自己的选择
　　无须，他人复杂恶意的指指点点
　　尽管，我由衷希望，别人的指点
　　道德是抽象而宽泛的概念，于我
　　貌似没有容纳的地方，只因为：
　　我既不做君子，也不是小人，
　　也不做小人，也没有心思做君子。
