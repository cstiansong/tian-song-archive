# 军训那些事儿

刚进大学，特别是军训那段时间，我总要在校园迷路。迷路后，我总是要问路。那个阶段，跟后来我刚进社会时极其相似，有点儿迷茫，有点儿苍白，有点儿懵懂。而且，每天穿着迷彩服，穿着军训套装，混迹在人群。

头几天，我按部就班地该干啥就干啥，偶尔忙里偷闲，翻看几页书，或者发发朋友圈抒发情感。这些事情不能做，我就搭讪说话，仿佛身边有人，不管男生女生，不管外向内向，我都有话说。那时，我说起话来滔滔不绝源源不断喋喋不休，而且，常常指点江山激扬文字挥斥方遒，还不懂看人眼色。

我性格不好，个性强烈，有点儿任性妄为，有点儿随意冲动，有点儿棱角分明。我天生憧憬自由向往无拘无束，而且，固执己见不服管教。所以，军训，对我来说，其实是折磨，因为它要求服从、配合、执行。同样道理，后来有人劝说我当兵，我没有犹豫，直接拒绝，理由是我个性强烈不服管教。

我们教官是军校学长，他很年轻，他的风格不像普通军人，在北方人里属于文雅随和有耐心的。我走齐步，走正步，老是顺拐，屡教不改，搞得他无语不知道该说我啥，而他只是在其他人休息时让我到旁边单练。

中学时代，军训期间，我分不清左右手老慢半拍，被教官训练得哭。现在，我的心理素质强大，强大到教官说我几句，忍不住要笑出声来。尤其是，教官站在我跟前盯着我，不知道出于什么原因，我就是想笑。

凌晨早起，绕着曲江拉练。“军训就是累狗、热狗、晒狗、淋狗、溜狗、训狗的过程。大太阳天是热狗、晒狗。五点起床，曲江拉练，是溜狗。老天落小雨淋狗，让我们变成湿狗。哎呀，走走总比站军姿，走起步被教官训要好。我希望时间能快些流。毕竟，我不喜欢军训。”长大后，觉得可笑。

我们那级军训，其实很轻松。十来天，仿佛有老天爷帮助，三四天都在下雨，天气完全不像往几年。一遇下雨，教官会领着我们去教室。每次去教室，我要么埋头读书，要么跟身边人窃窃私语，要么跟着学姐学练校歌。

有关这位学姐，当时，我们这个连有教官对她有那个意思。我清晰记得，我们教官劝说那教官，跑去要学姐微信，然后，尝试着约出来吃饭看电影什么的。然而，那位教官忸忸怩怩的。我知道，他们肯定没有后文。

再后来，偶尔遇见这位学姐，这位学姐应该是换了两三次男朋友。爱情对女孩子来说不公平，因为有些女孩恋爱后，容光焕发珠圆玉润，惹得路人都要停留；有些女孩恋爱后，胭脂俗粉油光满面，只教人不想再看。

学姐属于后者，她不认识我，我不认识她，仅是过客。
