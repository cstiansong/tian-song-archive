# 坐着火车去西安

按照日子，我拖着行李箱，到大湾河边乘坐乡村班车到木黄镇，又从木黄镇坐大巴到铜仁。铜仁火车站，售票厅窗口，工作人员说：“没有直达西安的火车票，只能中转。有中转怀化直达西安的，有中转重庆直达西安的。”

姐姐的话没有说完，我抢答：“麻烦您给我选中转重庆到西安的。”她查找信息，告诉我说：“今晚七点四十发车，从这里到重庆没有坐票，还剩站票。你要买票吗？”我点点头，随后在候车厅候车，直到太阳掉进山里头。

很久以前，某次周末，我们骑自行车出去游玩。在大山深处，偶遇绿皮。那时，许是受到书籍的影响，我憧憬着将来有天，我可以乘火车前往大学。按照天真的想法，我来到火车站。混迹在人群中，不懂就问。

车厢末尾，洗手间附近，我倚靠在壁上。透过玻璃窗，我瞧见万家灯火，直觉告诉我未来会很艰辛，甚至会忍受许多痛苦经受许多挫折。

长夜将尽，黎明划破夜空，重庆北站已到。走出站台，走进重庆北站，重庆北站大到让我找不到方向。附近，有位中年大叔，我走上前去问他，他告诉我说：“你跟着我走，我带你去，但你要给我十块钱！”

走进车厢，按照车票，找到自己的位置。我对面是青年大哥，他准备去太原。初生牛犊不怕虎，像我这种随意的人，很快跟他说起话来。

期间，我告诉他：“我从铜仁站到重庆。”他告诉我：“正常情况，每节车厢都有空位，这个空位没有售票。”让我印象深刻的是，他告诉我：“成绩不重要，重要的是能力！”我说：“我到大学会读好多好多书。”

穿越秦岭特长隧道，我感受到，人生也是这样，要忍受漫长的黑暗，才能懂得光明的珍贵。没有经过努力，轻易得到的东西终将失去。

在那瞬间，我想起，有人说：“未经省察的人生，不值得一过。”懵懵懂懂的，我满脸青涩，对未来压根没有明确的目标规划，只是在大量尝试当中隐隐预感到未来的轮廓。虽然有某种想法，但我不懂未来究竟在何方。

西安南站到了，走出火车站，行李被司机装进后备箱，我被安排坐副驾。这司机不厚道，他中途把我转给另外的出租车，那司机还加价。

从彩虹桥走进校园，我拖着行李箱，走到梧桐东道上，拐弯往前。繁茂的梧桐树下，我瞧见不挂女神，瞧见腾飞塔。也许是童心大发，我推着行李箱往前飞奔，像是野狗撒欢，惹得周围行人投以惊奇的目光。

大学末尾，我跟朋友乘火车去延安。这时，我对火车已经没有感觉。窗外，风景还是风景；窗内，我已不是少年。我给自己留下太多遗憾。

虽然这些遗憾是我青春岁月珍贵的回忆，但它们让我伤感。
