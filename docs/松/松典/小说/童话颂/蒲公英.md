# **蒲公英一**

1

这天，晴空万里，风轻云淡，风吹麦田发出波涛滚滚的音乐。小径被野草掩埋，在这样的路边，有株蒲公英，它随着风儿摇曳，顺着自然的旋律唱歌又跳舞。突然，狂风大作，黄沙漫漫，乌云掩盖了月。蒲公英开口说话啦。

我是蒲公英的种子，我们是紫色的蒲公英，我们的头发是紫色的，我们的血液也是紫色的。紫色蒲公英，白色蒲公英……根据蒲公英王国最古老的法典记载，遥远的地方居住着蓝色的蒲公英，我们共同归属于蒲公英界，也是同个家族。

家族里的每个成员都有自己的使命，这种使命神圣、庄严、崇高，不可推卸。这是我们活着的意义。

我是浅紫，别人都尊称我为“浅色小王子”。小时候，也就是暮春三月，我被柔风姐姐带到，这个说不出名字的地方。时间流逝，春天的步伐踏遍大地的每个角落，万物生长欣欣向荣的样子，有阵阵春雨滋润着山川与大地，我们苏醒。

在蒲公英的王国，我的父亲是国王。他法力无边，至高无上，可以号令天下。我是王子，受世人尊敬的王子，一人之下万人之上。我是父亲独有的孩子。从来到这个世界，我便注定要取代父王，成为蒲公英世界，新的王，新的主宰。

我的师傅神树爷爷曾跟我说：“你会是蒲公英王国有史以来最伟大的王！你将引领我们走向繁荣，走向光明！”

# **蒲公英二**

2

我们这个世界，生长着神树，没有人知道神树的实际年龄。只是，人们都觉得他活在蒲公英的历史以前，他见证了蒲公英家族一代又一代族人的生老病死。我曾问神树爷爷是否记得自己的树龄，他微笑着说:“我记不起。”

神树爷爷经常给我们讲故事，他娓娓道来，讲的内容深入浅出环环相扣，动听又诱人。我们喜欢他的故事。

从神树爷爷的故事深处，我了解到家族的过去，也曾听说我父母光辉的事迹。

你知道吗？陪伴我长大的不是我的父母，我的父母在我出身以前已经凋亡。我是个没有家的孩子。

所幸的是，整个成长的岁月，我不孤独，我有许多朋友，柔风姐姐、山神大叔、土地公公、白蒲哥哥、黄蒲妹妹，还有许许多多的臣民。他们对我很关注，而我享受着他们的爱戴。在我背后有我的师傅神树爷爷，他倾尽全力辅佐我。

我们的生命周期最最长久的，有八个月。对其他生灵来说，这跟天边的流星那样短暂。我们倒没觉得有什么遗憾，也没有谁像吉尔伽美什那样追求永生。你知道吗？在我们这里，生命的意义，不在于生命本身的长短，而在于是否完成肩负的使命。我们的生命匆匆，却传承着一种大爱，一种无法停留的大爱，一种无法凝固的大爱。这种爱是我们的使命。

相对我的臣民，我的寿命较为长久，只因为我有着蒲公英王国最为高贵的血统——紫色血液。

作为新一代的王，我肩负着领导蒲公英家族繁衍生息的历史使命。神树爷爷曾告诉我，我的寿命是这一代里最大的，因为我有九个月的生命。痛苦的是我必须见证：伙伴们的死去，连爱戴我的，和我爱戴的子民，也都要看着他们死去。

# **蒲公英三**

3

春天受大地之母滋润的我们抽芽。我们需要阳光，需要水分，需要温暖，有着和其他种子相似的历程。在我们这条路上并不顺利，我们是大自然最普通的种子。我们经受漫无天日的沉睡，烈日的折磨，可我们依然努力生长。

生命从泥土探出小脑袋的那瞬，随着太阳的东升西落抽出嫩芽，默默汲取大地深处的养料，慢慢成长。紫色蒲公英、白色蒲公英……我们的生长是相似的，可我们的颜色不同，颜色背后的等级不同，身份与象征符号也不同。

夏天，阳光明媚的清晨会有蒲公英绽放，花儿摆动，似柳絮那样飘柔。柔风姐姐轻轻吹拂，朵朵花儿像极了跳动的乐符，和柔风姐姐载歌载舞，欢欣鼓舞。绿丛中绽开朵朵紫色的、白色的、黄色的小花。无论从远处看，还是近处看，这都是美丽让人赏心悦目的风景。七八天花开以后，种子上的紫色冠毛，白色冠毛，黄色冠毛会结为一个个相应颜色的小绒球。

我们老了，没有多余的爱恨，没有多余的丑恶，没有多余的伤春悲秋，甚至没有对亲人的挂念。一切的一切都很自然，一种水到渠成的自然。一切的一切，都很平常，一种瓜熟蒂落的平常。我看着他们飘向远方，到最后，我也会远去。

我看着他们走，凄美的笑容，毫无眷恋。我看着他们安详又恬淡的面容，在我的心底说再见，待我完成肩负的使命，我也会来到你们的那个世界。青山仍在，绿水长流。岁月如流，转眼间，我垂垂老矣。如今，我是蒲公英世界的王，我法力无边，至高无上，可以号令天下，还是蒲公英家族史上最伟大的君王。我把我看见的想象的都记录在法典上。

# **蒲公英四**

4

作为蒲公英，尤其是紫色家族的领袖，如今的国王，我没有逃避的理由。因为使命，所以责无旁贷；因为道义，所以义无反顾。秋天已然走过，留下天际那抹黄昏，大雁南向而飞，留给我们绵绵秋雨。冬天来了，我的生命也要划上句号。

柔风姐姐，请把我的孩子们，带向远方。当他们懂事的时候，请你告诉他们，我深深地爱着他们。只是，我无法把我浓浓的爱凝固，装扮成礼物送给他们。你还要告诉他们，我每时每刻都牵挂他们，想念他们，希望他们快快乐乐地成长。

一年又一年，蒲公英代代延续，繁衍生息。风儿姐姐把蒲公英的种子，也就是我们的下一代带到远方。

朵朵紫色的，白色的，黄色的小伞在蔚蓝的天空里飘荡，美极了。看着安详的正在熟睡的宝贝们，有种快乐油然而生，有种自豪在胸腔里荡漾。永别了，我曾守护的蒲公英王国，永别了，我曾深深爱着的王国，还有我的孩子们。

我俯瞰苍茫大地，我看见自己曾生活的山坡，我看见我的神树爷爷、山神大叔、土地公公、柔风姐姐。他们向我招手，示意永别，他们眼睛里的依依不舍像汪汪春水那样温柔美丽。我的眼睛涌现温润的液体，想哭想要留在这里。

一缕青烟，一拱小桥，一弯清水，一曲小路，一头小毛驴，还有一位断肠人都消融在残阳如血的黄昏，消失在眼前漆黑如夜的黑暗。死无痛，无痒，无知，无觉；我无怨，无悔，无憾；死者已逝，爱却永驻人间。安息吧。
