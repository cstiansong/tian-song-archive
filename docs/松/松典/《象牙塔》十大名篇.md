# 《象牙塔》十大名篇

# 好讨厌高数

> 我的心被高数折磨得像老农民丢到路边的包心菜，由里到外都已经烂透。
——田松《[好讨厌高数](%E9%9A%8F%E7%AC%94/%E5%A5%BD%E8%AE%A8%E5%8E%8C%E9%AB%98%E6%95%B0.md)》
> 

# 我在图书馆脱鞋

> 那瞬间，我的脸像蒸熟了的螃蟹甚至尴尬。
——田松《[我在图书馆脱鞋](%E9%9A%8F%E7%AC%94/%E6%88%91%E5%9C%A8%E5%9B%BE%E4%B9%A6%E9%A6%86%E8%84%B1%E9%9E%8B.md)》
> 

# 美丽的女孩子

> 我把她写进我的文章，而她把我像路边的野草，遗忘到岁月深处。
——田松《[美丽的女孩子](%E9%9A%8F%E7%AC%94/%E7%BE%8E%E4%B8%BD%E7%9A%84%E5%A5%B3%E5%AD%A9%E5%AD%90.md)》
> 

# 出走杨凌

> 伪君子发消息告诉我说：“哈哈，你就是个大傻逼！”
——田松《[出走杨凌](%E9%9A%8F%E7%AC%94/%E5%87%BA%E8%B5%B0%E6%9D%A8%E5%87%8C.md)》
> 

# 游泳课

> 遇见她的时候，我想起她穿泳衣的模样，腹部涌起滚烫的热流。
——田松《[游泳课](%E9%9A%8F%E7%AC%94/%E6%B8%B8%E6%B3%B3%E8%AF%BE.md)》
> 

# 她掰弯了眼镜

> 如果韩信注定要遭受胯下之辱，即使给他戴上最高贵的皇冠，他也不能逃脱厄运。
——田松《[她掰弯了眼镜](%E9%9A%8F%E7%AC%94/%E5%A5%B9%E6%8E%B0%E5%BC%AF%E4%BA%86%E6%88%91%E7%9A%84%E7%9C%BC%E9%95%9C.md)》
> 

# 追姑娘太猛

> 今生今世，我死心塌地地喜欢着这个女孩，却不得不把她遗忘，这是在偿还前世我对她的拖欠。
——田松《[追姑娘太猛](%E9%9A%8F%E7%AC%94/%E8%BF%BD%E5%A7%91%E5%A8%98%E5%A4%AA%E7%8C%9B.md)》
> 

# 小包工头

> 有那么瞬间，我就在想，一辈子只跟一个女人，确实有点单调。
——田松《[小包工头](%E9%9A%8F%E7%AC%94/%E5%B0%8F%E5%8C%85%E5%B7%A5%E5%A4%B4.md)》
> 

# 愿意跟我交往吗

> 如果愿意出卖灵魂不择手段，我相信，我可以跟这个女孩有场轰轰烈烈的恋爱。像所有言情小说里常有的桥段，我会跟她牵手，我会跟她接吻，我会跟她走很多地方，我会跟她享受男女最美妙的事情，我会跟她书写书深刻而独特的故事。
——田松《[愿意跟我交往吗](%E9%9A%8F%E7%AC%94/%E6%84%BF%E6%84%8F%E8%B7%9F%E6%88%91%E4%BA%A4%E5%BE%80%E5%90%97.md)》
> 

# 泰发祥

> 我残存的心像海岸边长期以来就被浪涛冲刷的岩石，已经分辨不出曾经的模样，让自己都觉得陌生。
——田松《[泰发祥](%E9%9A%8F%E7%AC%94/%E6%B3%B0%E5%8F%91%E7%A5%A5.md)》
>
