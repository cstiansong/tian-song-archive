# 哥哥啊，我的哥哥

Date: 2020/12/27

> 哥哥啊，你像那个康德
活脱脱的堂·吉诃德
还是一颗痴情的种子鸭
> 

> 你闪闪发光、足够耀眼
哥哥还把银河的星星
捧着手里送给美丽的女郎
> 

> 哥哥是儿童，哥哥是诗人
哥哥我还像蓝色的精灵
啊，哥哥啊，我的哥哥啊
> 

> 哥哥控住不住我的嘴巴
啊，哥哥啊，它们跑来啦
哥哥！你可要听清楚啦
> 

> 哥哥，你是万物的尺度
哥哥，你是世界的中心
哥哥，我是你生命的全部
> 

> 只有你像太阳，哥哥啊
只有你是神灵，哥哥啊
> 

> 哥哥啊，我的哥哥啊
你是老天爷是上帝是神
是我生命的全部是我的主
> 

> 2020.12.27
