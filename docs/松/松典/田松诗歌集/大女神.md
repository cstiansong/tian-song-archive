# 大女神

Date: 2021/05/31

> 我受尽了你的冷漠与折磨
所以，再也不把你当人
而认你为光芒万丈大女神
> 

> 只有你像太阳那样为我
驱逐心尖上寒冷还有孤独
> 

> 只有你像月亮那样为我
隐藏心底儿自卑还有梦想
> 

> 你让我流下忧伤的泪水
却使我写下这闪光的诗句
> 

> 我想要和你依偎在一起
轻轻握着你的手久久不放
> 

> 你是我想要追逐的花萼
还曾是披星戴月的大女神
> 

> 你曾让我在刹那间飞翔
闪耀着灵感与生命的奥秘
> 

> 我为了你，我的大女神
愿意放弃成为作家的梦想
> 

> 仅仅和你相拥，用舌头
像小蜜蜂那样蛰你轻吻你
> 

> 初稿：2021年5月31日12：20往航天城地铁
>
