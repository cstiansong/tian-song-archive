# 献礼

Date: 2021/05/23

> 月亮儿月亮，你别躲进
乌黑的云彩深处，我
还要你为我照亮前行的路
> 

> 姑娘啊姑娘，你别远走
遥远的城市角落，我
还要你为我露出美丽的脸
> 

> 没有你的夜晚，夜晚是
绝望的夜晚仿佛还是永远
> 

> 没有你的日子，我的心
永远在灰色的寂寞中燃烧
> 

> 姑娘啊姑娘，你别走
我想，用我的唇和你诉说
> 

> 你很美，真的很美很美！
忍不住想要你想要我爱你
> 

> 你有少女般鲜嫩的纯
可以燃烧出熊熊般火焰
> 

> 今儿，夏天的风把你带走
而你却带走我的想念
> 

> 什么都不关注我就想和你
说说话聊聊天走走路看看你
抚摸你，甚至把我的舌头伸进
你的喉咙还要停顿片刻儿
> 

> 姑娘啊姑娘亲爱的姑娘
如果有天，我意外地忘记你
请你记住：我曾真心爱你
> 

> 初稿：2021年5月23日14：05钱馆南三082
