# 致宗宗导员

Created: July 16, 2022 4:33 PM

宗宗导员：

你好！今天收到你的微信，我很感动。谢谢你的关心。我们之间有些隔阂，所以写了以下所有的文字。希望尽释前嫌，我把所有的所有都写出来，希望我们共同改进，相互学习。怎么说呢，你是师兄，也没比我大几岁。所以也就不用您了，那样有点见外。首先我保证我的文字是真实想法也是真实的。如有冒犯，敬请见谅。

我对老师同学是很尊敬的，但触及我底线的人。我不会管他是谁，多看几眼都觉得浪费时间。来大学快一年半了。学到挺多的东西，受到的小挫伤也不少。可能是我不会与他人相处，随便吧。

我觉得自己还行就行。那些因为别人几句话就离开的人，迟早都是要离开的。不同的只是离开的理由。再热情的人经历些事情后，也会冷淡很多。多情的人都难以逃脱自作多情。

从助学金就产生了嫌隙，后来我发现那些家境比我好的人得到的其实更多。这种小事也没什么，反正我也就是用那2500来买书。不要那笔钱，也不会怎么样的。如今的我依旧记得，那时你叫我进办公室，询问些后觉得我爸做泥水工的，然后……本来也不是特穷。高中班主任对于那种助学金基本是你申报了都有的。

芥蒂的是你没有做到公平，还说了些挫伤了我的话。然后我决定再也不要那种钱了，反正我不读研。

后来的某天傍晚，你叫我进办公室。我还以为什么事情呢？急匆匆地赶来，还以为什么要紧事呢。结果是以后别在那些群里说话了。然后我就退群了，再也没说什么话了。再后来也就有了我建立的读书交流群。

你就发条信息给我说，然后我就不会说。关于周芷琦那件事，我当时就没想过。

可你说成“骚扰”多难听。然后对我说，你要检点自己的行为。每次见到要么拍肩膀要么叫名字，那次我从图书馆出来遇见她坐在那儿，我便坐到旁边去，佯装要靠肩。也就当开开玩笑。然后你说成“骚扰”。感觉如同在某个不知名的地方突然就被自己认为亲近的人捅了几刀，有着强烈的背叛意味。

像这种事情，当面说不管你怎样说都会适得其反的。以后，还是在微信说吧。只是，我觉得不会有以后了。你的初衷就是让人认错，可我根本就没意识到自己的错。毕竟，像我这么蠢货的学生太稀少了。

我根本就没喜欢她。信不信都这样，我说完我想说的了。

在那以后，我见到周虽然还会招呼，可已经没有以前那种热情了。有时看见了也当没看见，热脸贴冷……感觉认不认识都无所谓。怪自己看错人了。现在也只是无所谓，她失去了也就是个自以为是好朋友的人而已。我无所谓。本来就没喜欢过，那种仿佛没人喜欢，觉得别人多看几眼就觉得别人喜欢自己，不觉得搞笑吗。

最近那件事情，错的地方在于我不应该跟她说我喜欢她，随后写的几份邮件都是将错就错。“大晚上的，你跟踪人家，你自己想想尾随女生，人家是什么感受？”听起来就好像我要犯罪似的，简直就是搞笑。

那天，我在图书馆遇见了。然后等了她准备一起走，见她什么都没说。等了会也就走了。走在路上，我觉得我应该找回去，把事情说清楚。以后不做朋友也行。其实我想了很久很久犹豫可很多很多才表白的。实际也不算是吧。也就是写了些文字，表白的话还是当面说的好。

以后遇见其他喜欢的女生，我就当面说，不愿意就拉倒。“一起走吧？”然后她说“你有病吧？”然后我走了。然后我走了段，便绕道后面去了。然后隔着老远距离，回到崇实。所有的事情也就是这样了。我最开始写了三封情书，说得很清楚。做不了男女朋友，做朋友。然后其他事情都这样了。

我是发了几天微信消息，然后她把我删了。然后我写了几首自认为是情诗的诗发□□。后来，我觉得没意思就把她□□掉了。再后来，我在读书群里说了句“题外话，各种意思文字是傻，可她意思没明白”。

“你说谁傻呢”“你是说呢愿意做我女朋友？”“你脑子是不是进水了”然后那天夜里，我算是写了几段表白的话放在群里了。然后她退群了。那感觉就好像我要离开了，你的真心实意我带走了。

不喜欢就不喜欢，至于在那群里这样的么。我的脸也就被她这么糟蹋了。后来，偶尔遇见我还会不要脸地叫叫名字算是招呼。她每次都说你傻你有病。说到底就是她自己做不到心平气和。完了还找辅导员，说我骚扰说我跟踪。呵呵，真是想好。这些换成你，你想想啊。那天，我就想说加回微信，以后的事情随便你。

她没给我机会。然后有了第二天的事情。都这么大啦，也不是那种听不进道理的人。

在这场感情纠纷中，我是十足地蠢了。没人教我，书里面那些毕竟是书。然后我应该也不会有以后了。人是笨了点，但相同的错误不会犯错误。那天我在读书群里说了段气话，然后还愚蠢地跟她室友吵了几句。

想来也是白痴，跟女生吵不是自讨苦吃吗。还有关于宁静的事，我自始至终什么都没做，是她自己看不惯别人然后就想怼几句。关我屁事，我什么都没做，如今的我什么也没做。现在我也什么都没说，虽然是同班同学，但她要这样，我就当不认识行了。我不缺同学朋友。

最开始我也是要打招呼的，三番两次不见回应也就没了那个兴致，何况我每天都有自己要做的事情，本来就没时间还还拿来的精力去浪费。以后不见了就是，朋友是做不成也无所谓，这种事情做不了男女朋友，就不见面了。你比我大几岁这些事情你也懂的。你是过来人。你喜欢的姑娘不喜欢，你还能怎样了。

从她拒绝那刻起，我就知道了最后的结果。只是不甘心而已。那种仿佛从没有男生喜欢，有了后四处炫耀。我只能怪自己愚蠢。所有的愚蠢只是因为喜欢，不喜欢什么都无所谓。我只是浪费了点时间感情而已，何况，那也不能算浪费，总比我默默等待下去直到某天被辜负。这次真TM地觉得自己受到了侮辱，也无所谓啦。

基本不会遇见了，遇见也和没遇见没区别。本来就没那么好。还要这么傲娇，她最多也就是找到比我高比我帅的人而已，关我屁事。我努力成为优秀的人就行。我要说的都说完了，估计不会有以后了。有了以后，你直接给我发微信。我是那种说不谁来往就不和谁来往的人。我什么都没有损失，而她彻底失去了爱自己的男生。

因为没人瞧得起，我会更加努力。若干年后，我回来演讲，我顶多也就是说自己在大学里没姑娘瞧得起，所以天天泡在图书馆看书，因而多看了几百本。至于当初那个人，我连名字都记不起来了。即使想起来，也不会说了。

对我来说，只是多了次经历。那位以后找到糟糕的人，我会笑的。我不是君子，也没打算做君子。

我呢有自己要追求的东西，我不属于那种为了姑娘就会放弃自己梦想的人，也不属于那种愿意为了无关紧要的人浪费时间。我可能是中文系最优秀的学生，成绩好不好对我来说无所谓。

我已经是推心置腹了，师兄要是觉得我不行，删掉微信不来往就是。我无所谓。

说到底，我觉得自己做的事情问心无愧，所以我没有内疚感，所以我身边多个人少个人其实无所谓。

我有很多朋友，那些要走的人我也没必要牵挂。希望合作愉快。
