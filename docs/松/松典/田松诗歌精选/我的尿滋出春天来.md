# 我的尿滋出春天来

我的尿滋出春天来

刚刚，我合上电脑，写完日记，打算
关掉电炉、熄灯。可，尿意席卷全身。
打开后门出去，我听见，清水哗啦啦～～
随即，走到核桃树跟前撸开裤子，滋滋

被尿滋的核桃树，还是我以前自己种的！
转眼间，它竟然碗口还粗！我想起，
北伐的桓温曾，树犹如此，人何以堪！
滋滋、滋～舒服喽！偶然间，春天来啦。

滋完以后，我居然发现，春天来啦！
春天来啦？不着衣裳、赤裸的紫荆树旁
的，那颗同样是我栽种的万年青萌发。
萌出嫩绿鹅黄的新叶，还生出枝桠！！！

春天来啦……春天来啦……撸起裤子，我
发现春天来啦！春天好像被我尿滋来啦
无聊的人，读到这，我相信，那感觉
像头顶飞过黑乌鸦，留有：。。。。。

我的尿滋出春天来，似乎有点那个……
然而，人类本身的历史，何尝不是如此？
