# 松的经历

## 早年经历

1997年12月27日，田松出生于贵州省铜仁市印江土家族苗族自治县合水镇大堡村。

小学就读于邻村的石万村小学，初中就读于印江木黄中学，高中就读于铜仁一中。

2017年田松参加高考，取得616分，加上20分之后被西安交通大学录取，同年秋入学，专业为文科实验班。大一结束后综合排名为文科实验班倒数第三（后两位是留学生），分流进入汉语言专业。
