# 新松学概论

唐太宗曾经说过：“夫以铜为镜，可以正衣冠；以古为镜，可以知兴替；以人为镜，可以明得失。”

西交田松，正是这样一面镜子。面对田松，我们总能照见自己内心的某个角落。无论在哪，田松总能发出耀眼的光芒，让人无法忽视其存在。

随着田松的影响力越来越大，西交涌现出了一大批“观松者”。他们观松，进而研究松，逐渐发展出了“松学”。观松者逐渐增多，松学也愈发壮大，旧松学也暴露出了一些问题：

其一，重形而轻质。旧松学研究者往往注重田松一些夸张的言论和有趣的故事，较少研究能体现田松思想的田松作品和其生活轨迹。这就导致松学的研究往往流于表面，很难深入。

其二，没有形成体系。旧松学并没有对田松的事迹、言论、作品分门别类进行研究，往往东一榔头西一棒槌，许多重要资料就此遗失。

其三，内容偏废严重。旧松学研究者往往研究不够广泛，只了解极少数松学内容，如“舒服美好”、“醉酒体”等，对于田松作品也往往只知其名，连《郑家姐妹》读过的都很少，田松的其他经典作品如《狗仔游记》《春来魂散》等更是涉猎者寥寥。

基于以上问题，我们提出“新松学”的概念。新松学将科学、系统地研究松，兼顾深度与广度。我们的目标是既能让新观松者迅速对松有全面的了解，又能让老观松者对松的研究更加深入。

新松学主要分为以下几个方向：

**松史**，主要研究松的事迹。对松的趣事和重大事件、人生转折点进行深入研究，以此了解松。

**松典**，主要研究松的作品。以此了解松的文笔和思想。

**松录**，主要研究松的语录。对松各种渠道发布的言论进行研究，主要记录松的有趣言论和高光时刻。

**松论**，主要研究松的思想。通过对松典、松录的研究，提炼出松的人生观、价值观和思想理论体系。

以松为镜，可以帮助我们更好地看清自己。愿我们在人生道路上能时常看一看松，避免走弯路。

新松学万岁！

是为序。
