# 我是大学生

刀哥给田松算卦后，田松写文《我是大学生》

> 我是大学生
> 
> 
> 学生时代末尾，在钱学森图书馆，完成毕业论文致谢部分后，我透过窗户瞧见暗蓝色的夜空，想起许多年以前，自己还是乡村小学的少年。少年的我曾在日记本里写下“我要成为村里第一个考进名牌大学的人。”
> 
> 转眼间，我已经从所谓的“名牌”大学毕业。回想自己为什么能够实现这种在当时谁都觉得不可能实现的梦想，我就会想起在追梦求学的路上遇见的许多人经历的许多事……今天，我不会叙述这些，因为我想说，我是大学生。
> 
> 我是大学生，这在学历普遍、迅速贬值的时代，并不是什么稀奇的事情。即便自己毕业于西安交通大学，依然不值得骄傲、炫耀，或者显摆。
> 
> 只是，每当生活在美丽的山村的人们，他们不喊我的名字，老是喊我“大学生”时，我就会觉得“大学生”不仅是份荣誉，还是份责任。这份沉甸甸的责任随着自己对传统文化逐渐深刻的理解越来越沉重，竟像山那样压着我。
> 
> 中华民族建立在农耕文明基础上，经过儒释道文化共同塑造。在这样的文化生态系统里，出生农村，生活在城市，从顶尖大学毕业，还博览群书，这样的人没有谁会失败！因为他们不是为了证明“读书无用”来到人间的。
> 
> 如果他们失败了，或者干脆说我的人生最后失败了，我们打击的不仅是自己不仅是家人，还有那些千千万万辛辛苦苦劳动挣钱，为了孩子将来能够考大学改变命运的父母，还有那些千千万万辛辛苦苦读书学习，为了自己将来能够考大学，改变命运的学生……他们怎么办？付出所有换来人生失败？
> 
> 别的不说，就是生活在美丽的山村的人们，他们怎样想？他们的希望还能寄托到孩子考进大学回馈家庭吗？我是大学生，我是村里出来比较出色的大学生，我不能损坏我或者说农村出来的大学生形象，不能浇灭他们心中的希望。
>
