# 端居消业论

## step1

2021年3月5日，有同学在田松抱怨之后提出了”消业”这一概念，得到了松的认可。

![Untitled](%E7%AB%AF%E5%B1%85%E6%B6%88%E4%B8%9A%E8%AE%BA/Untitled.png)

2021年3月6日第一次使用消业这一说法解释自己事事不顺。

> 近来，厄运连连遇事不顺，心情低迷颇为烦闷，逢事不遭殃。自觉身无长物，唯读书两两，然学艺不精，前途灰暗，实乃生平之未有，大为发愤，心却轻轻巧巧。此情此景大有悔不当初的失落，却又不知道何处反思。末了，昨夜还有位朋友说，佛语讲你这是消业
>
