# 松论

主要记录田松的理论思想

# 松学三定律

松学第一定律：一切没有固定标准、或者难以短时间验证的东西，都是松松擅长的

松学第二定律：松所擅长的无固定标准的事业，凡是想要给其订立标准，或想提出与松的标准不同的其他标准的，都是松学意义上的弱智。

松学第三定律：弱智的作用是相互的。我见松松多弱智，料松松见我应如是。
