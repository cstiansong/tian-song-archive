# 名称释义

小雷音寺又名小西天，是《西游记》中的地名和唐僧师徒遇难的地点之一。小雷音寺假冒西天雷音寺，但是里面没有如来佛祖，只有黄眉老佛。

小雷音寺qq群，本名“读书小天地”（和田松官方QQ群同名），但是里面没有田松，只有假冒的浪淘沙。因此群友为了和官方读书小天地区别，将假冒的“读书小天地”戏称为小雷音寺。

# 由来

## 恒纪元和乱纪元

由于QQ群有群禁言功能，田松学会这一技能后经常在松Q群使用。群友将群禁言时期称为”乱纪元“，解除禁言的时期称为”恒纪元“。关于恒纪元和乱纪元的名称由来，请阅读小说《三体》。

## 小雷音寺的建立

频繁的乱纪元导致松Q群只有松搬运微信群消息，没有任何其他人能发消息。为了满足观松过程中大家不吐不快的交流欲望，广大观松者建立了小雷音寺。

# 历史沿革

## 小雷音寺1.0

初代小雷音寺，肉松（现小雷音寺转码松）创立，后群主主动解散。

## 小雷音寺2.0

二代小雷音寺，由假松创立。后因为群员键政被封群。

## 小雷音寺3.0

三代小雷音寺，由假松创立，目前群主为甜松，作为第四代小雷音寺备用群。

## 小雷音寺4.0

四代小雷音寺，由假松创立，群主转为甜松。2023.5.5因为群员键政被封群。

## 小雷音寺5.0

五代小雷音寺，由假松创立。

# 重要事件

## 25事件

小雷音寺成员25向田松出卖小雷音寺名单，并在松群大量踢小雷音寺成员，导致松群十分冷清。后田松踢了25，解除禁言，平息了事件。

## 卤门事件

小雷音寺群主甜松利用田松追妹子心切的心理，假扮妹子，离间田松和田松的忠心狗腿子卤门的事件。后卤门澄清误会，并成功上位松V群管理。
