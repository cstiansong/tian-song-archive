# 小红豆子事件

指渣渣辉和风月诗人等人假扮女生“小红豆子”套田松的话、骗田松跑图的事件。

# 缘起

松友**风月诗人**发现田松在小红书上勾搭女生，于是想借机假扮一位小红书上的女生“小红豆子”来骗松。
