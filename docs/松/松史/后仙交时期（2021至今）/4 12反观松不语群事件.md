# 4.12反观松不语群事件

# 起因

田松得知了微信观松群（观松不语真君子），自己的亲信高黄二人也在里面。

![Untitled](4%2012%E5%8F%8D%E8%A7%82%E6%9D%BE%E4%B8%8D%E8%AF%AD%E7%BE%A4%E4%BA%8B%E4%BB%B6/Untitled.png)

# 经过

田松在群里要求退掉观松不语群，在微信和qq群两面开刀，表示宁可错杀三千，不可放过一个。

# 结果

微信观松群多人纷纷退群，从此式微，这是观松史上一次重大挫折。

小雷音寺（qq观松群）吸收部分微信观松群流失成员，逐步壮大。
