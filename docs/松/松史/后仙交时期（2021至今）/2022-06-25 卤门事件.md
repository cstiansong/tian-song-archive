# 2022-06-25 卤门事件

# **背景**

2022年6月，田松在单位认识了一位女孩子。田松对此女展开了热烈的追求，任何企图破坏田松姻缘的人都将被田松视为仇敌并遭到严厉打击。女生即将回北京，田松表达爱意之后，面对即将到来的脆弱的异地恋，内心充满不舍与不安。

![Untitled](2022-06-25%20%E5%8D%A4%E9%97%A8%E4%BA%8B%E4%BB%B6/Untitled.png)

卤门是读书交流群（松群）的一位活跃群友，此前曾积极舔松，为小雷音寺众所不齿。

# **开端**

### **林智浩枪声**

![Untitled](2022-06-25%20%E5%8D%A4%E9%97%A8%E4%BA%8B%E4%BB%B6/Untitled%201.png)

此时，众人对于林智浩的身份并不清楚，也不了解田松动怒的原因。

## 总结

甜松——本次事件的始作俑者，主谋，假扮妹子，戏耍田松。

武松——本次事件的主谋之一，假扮卤门。

# **影响**

### **卤门上位**

卤门从之前只是舔松的狗腿子之一，顺利上位为松群管理，成为副群级干部。卤门在此次事件后受益最大。

### **松与小雷音不共戴天**

~~没有证据证明此次事件与小雷音寺有关，也没有小雷音寺相关人员对此次事件负责。~~根据现有证据，此次事件为小雷音寺群主甜松策划，武松参与实施。此次事件后，田松与小雷音寺不共戴天，此后可能会对小雷音寺展开报复。

### **与妹子的关系**

根据现有信息，并没有人和妹子披露田松的真面目。后续和田松的感情线敬请期待。

### **6·26松群大清洗事件**

此次事件后，6月26日，松清理了约30名活跃成员，松群从此万马齐喑，成为了松和卤门的二人转。
