# 早期田松

1997年12月27日，田松出生于贵州省铜仁市印江土家族苗族自治县合水镇大堡村。

小学就读于邻村的石万村小学，初中就读于印江木黄中学，高中就读于铜仁一中。

2017年高考，田松考上西安交通大学，同年秋入学，就读于汉语言文学专业。
