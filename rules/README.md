# Rules

本目录用于沉淀“新增资料处理”相关的流程规范，规则文档和自动化脚本配套维护。

- `content-ingestion.md`：新增资料导入与合并流程（SOP）
- `image-handling.md`：图片引用、多图并列与压缩规则
- `validation-gate.md`：提交前校验门禁与 CI 对齐策略

配套脚本位于 `scripts/`：

- `scripts/add_entry.py`
- `scripts/auto_gallery_layout.py`
- `scripts/compress_images_in_md.py`
- `scripts/compress_new_images_and_validate.py`
- `scripts/check_gallery_format.py`
